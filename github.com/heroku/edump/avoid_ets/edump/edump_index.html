edump_index

Generates an index for a crashdump.

Generates an index for a crashdump.

Index generation is completely isolated, it is therefore possible and completely safe to generate an index for more than 1 crash_dump file in parallel. The resulting index() term can then be fed into edump_viewer for further processing of individual segments.
