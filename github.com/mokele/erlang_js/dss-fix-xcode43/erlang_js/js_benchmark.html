js_benchmark

Utility module for measuring latencies across the Erlang/Javascript boundary.

Utility module for measuring latencies across the Erlang/Javascript boundary. This module implements a very simple benchmark of defining and executing a simple Javascript function 1000, 10000, and 100000 times.

Functions

run() -> list(int(), int(), int())

Runs the benchmark and returns the average time to process a function call for each set of iterations in microseconds.
