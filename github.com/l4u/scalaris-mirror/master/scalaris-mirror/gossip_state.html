get(Values::values() | values_internal() | state(), Key::avgLoad) -> avg()

Gets information from a values, values_internal or state record. Allowed keys include:

avgLoad = average load,

stddev = standard deviation of the load among the nodes,

size_ldr = leader-based size estimate,

size_kr = size estimate based on average key range calculation,

size = size_kr if not unknown, otherwise size_ldr,

minLoad = minimum load,

maxLoad = maximum load,

triggered = how often the trigger called since the node entered/created the round,

msg_exch = how often messages have been exchanged with other nodes,

avgLoad2 = average of load^2,

size_inv = 1 / size_ldr,

avg_kr = average key range (distance between nodes in the address space),

round = gossip round the node is in,

values = internal values stored in a gossip state,

initialized = load and range information from the local node have been integrated?,

converge_avg_count = how often all values based on averages have changed less than epsilon percent

See type spec for details on which keys are allowed on which records.
