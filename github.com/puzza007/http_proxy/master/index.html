Module Index


http_proxy

http_proxy
This library module implements the public API for the http_proxy application.

http_proxy_app
This application behaviour callback module starts and stops the http_proxy application.

http_proxy_origin_connect_fsm
This gen_fsm behaviour callback module handles outgoing TCP connections to origin servers.

http_proxy_origin_server
This gen_server behaviour callback module receives user agent (UA) requests from http_proxy_ua_connect_fsm processes for resources at origin servers and forwards them to a http_proxy_origin_connect_fsm process handling a TCP connection to the origin server, starting the process if it does not yet exist.

http_proxy_ua_accept_server
This gen_server behaviour callback module accepts connections on a listening socket and starts an http_proxy_ua_connect_fsm process to handle it.

http_proxy_ua_connect_fsm
This gen_fsm behaviour callback module handles incoming TCP connections from user agents (UA).

http_proxy_ua_listen_server
This gen_server behaviour callback module opens a listen socket on a port and starts the accept pool.

http_proxy_util
This library module implements utilities for the http_proxy application.
