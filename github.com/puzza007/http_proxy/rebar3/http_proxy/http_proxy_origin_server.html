http_proxy_origin_server

This gen_server behaviour callback module receives user agent (UA) requests from http_proxy_ua_connect_fsm processes for resources at origin servers and forwards them to a http_proxy_origin_connect_fsm process handling a TCP connection to the origin server, starting the process if it does not yet exist.
