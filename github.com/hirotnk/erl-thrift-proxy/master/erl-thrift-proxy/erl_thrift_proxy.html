handle_function(Func, Args) -> term()

Callback interface for Thrift. handle_function/2 is called from Thrift server

1. Retrieve the original module name from ETS table and call the original handle_function/2 of that module 2. Forward the request to a thrift proxy process 3. Return a reply

handle_error(Type, Reason) -> term()

Callback interface for Thrift for error handling

start_link(Name, Mod, Conf, ThriftConf, UserConf) -> term()

Start up a proxy process that forwards requests to another Thrift server and takes care of callback functions calls

Note: you need to start a thrift server that receives forwarded requests from this proxy process on the same/remote machine
