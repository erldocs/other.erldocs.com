Module Index


ierlang

ierl_code_manager
The code manager handles execution of erlang code and compilation of erlang modules.

ierl_control_server
The Control server handles some requests from IPython.

ierl_heartbeat_server
The heartbeat sever keeps IPython alive by constantly listening for messages on the given socket and replying with a ping message.

ierl_message_builder
Module that contains the functions to create replies to IPython messages.

ierl_message_parser
Module to parse IPython messages and their contents.

ierl_message_sender
This module sends messages to IPython frontend using ZMQ.

ierl_script
This modules defines functions used to starts all the IErlang servers, to handle messages, and to parse the json configuration file.

ierl_shell_server
The Shell Server handles the most important messages received from IPython.

ierl_zmq_manager
Set up all the ZMQ sockets and bindings, also spawns the servers that handle the messaging.
