tt(Fun::function() | {F::atom(), A::list()} | {M::atom(), F::atom(), A::list()}, Duration::non_neg_integer()) -> {Count::pos_integer(), RealDuration::non_neg_integer()}

测试在期望的时间内(单位：毫秒)指定的函数能执行多少次。 返回{实际运行的次数, 实际运行的时间} 注意事项：如果系统时间的最小精度为毫秒，那么只测1毫秒的误差可能很大。

ta(Fun::function() | {F::atom(), A::list()} | {M::atom(), F::atom(), A::list()}, N::pos_integer()) -> {Min::non_neg_integer(), Max::non_neg_integer(), Med::non_neg_integer(), Avg::non_neg_integer()}

运行函数N次，返回{Min, Max, Med, Avg}。 适用于单次运行时间大于1毫秒的情况。

tc(Fun::function() | {F::atom(), A::list()} | {M::atom(), F::atom(), A::list()}, N::pos_integer()) -> Duration::non_neg_integer()

运行函数N次，返回总耗时。这个函数对测试结果的影响相对小。
