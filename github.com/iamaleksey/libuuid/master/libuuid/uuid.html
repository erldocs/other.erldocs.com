uuid

Functions

generate() -> UUID16::binary()

Generates a UUID based on high-quality randomness from /dev/urandom, if available.

generate_random() -> UUID16::binary()

Generates a UUID using the all-random format, even if a high-quality random number generator (i.e., /dev/urandom) is not available.

generate_time() -> UUID16::binary()

Generates a UUID using the current time and the local ethernet MAC address (if available).

parse(UUID36::binary()) -> UUID16::binary()

Converts the UUID from a 36-bytes-binary-string into a 16-bytes-binary.

unparse(UUID16::binary()) -> UUID36::binary()

Converts the UUID from a 16-bytes-binary into a 36-bytes-binary-string. May be upper or lower case depending on the system.

unparse_upper(UUID16::binary()) -> UUID36::binary()

Converts the UUID from a 16-bytes-binary into an uppercase 36-bytes-binary-string.

unparse_lower(UUID16::binary()) -> UUID36::binary()

Converts the UUID from a 16-bytes-binary into a lowercase 36-bytes-binary-string.
