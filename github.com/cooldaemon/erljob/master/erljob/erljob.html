erljob

erljob is a job scheduler.

erljob is a job scheduler.

Here's a quick example illustrating how to use erljob: 

     erljob:start(),
     erljob:add_job(hi, fun (_X) -> io:fwrite("Hi!"), ok end, ok, 1000, 2),
     erljob:stop()
