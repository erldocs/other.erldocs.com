hc_node

Interface module for network node representation.

Interface module for network node representation. A node in the cluster, called a cnode, is known to all other nodes. This means all of its values, including the user-defined attribs, are transmitted to the entire network.

Do not manipulate the cnode record directly nor manually store them in a custom list/dict/etc. Instead, use this module to do so. The empty/0 function can generate an empty nodes container as a starting point.
