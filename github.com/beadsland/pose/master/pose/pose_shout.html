pose_shout

Watch for the creation of a text file, tail it until its associated lock file has been removed, and then exit.

Watch for the creation of a text file, tail it until its associated lock file has been removed, and then exit. Used by pose_shell to receive the stdout of a shell command.

Note: Code is not Unicode safe. Assumes all shell commands produce latin1 output.

Types

filename() = string()

Functions

monitor(File::filename()) -> pid()

Watch a text file, copying each line written to it to calling process as standard output, until an associated lock file is removed (this signaling that writing to the file has finished.)
