qu

quモジュールはN Queensパズルの解を求めるための関数を提供します。 qu_math_permの機能拡張により、オリジナルのN Queensよりもメモリーの 使用量が少なくなっています。.

quモジュールはN Queensパズルの解を求めるための関数を提供します。 qu_math_permの機能拡張により、オリジナルのN Queensよりもメモリーの 使用量が少なくなっています。

Types

cols() = [integer()]

Functions

queens(N::integer()) -> [cols()]

queens関数はN Queensパズルの解を求めます。 整数を引数に取り、解をcols型のリストで返します。

queens(Prefix::[integer()], Rest::[integer()]) -> [cols()]

queens関数はN Queensパズルの解を求めます。 引数に整数をリストを2つ取り、それを元に部分的な駒の配置のみを検証します。 解はcols型のリストで返します。
