erlrcsupnotify

Additional functions a supervisor can (optionally) implement in order to be notified of upgrades and downgrades by erlrcdynamic.

Additional functions a supervisor can (optionally) implement in order to be notified of upgrades and downgrades by erlrcdynamic.

Functions

sup_downgrade_notify(OldVsn::string(), NewVsn::string()) -> void

Notification of downgrade (before downgrade has occurred). Use this to ensure new childspecs are stopped, etc. Return value is ignored.

sup_upgrade_notify(OldVsn::string(), NewVsn::string()) -> void

Notification of upgrade (after upgrade has occurred). Use this to ensure new childspecs are started, etc. Return value is ignored.
