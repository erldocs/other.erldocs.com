load_application(App::atom()) -> ok | {error, Reason}

Load the given application App, obeying an override resource file App.app in the directory $ERLRC_ROOT/applications. for override spec of: { application, my_app [ { modules, [ module, module2] } { env, { keymerge, [ {var, "value"} ] } { applications, { merge, [ kernel, stdlib } } { registered, { override, [ reg, reg2 ] } ] }

replaces modules list with the [ module, module2 ]

merges the new {var, "value"} into the existing env list, replacing any { var, _ }

merges kernel and stdlib into the applications list.

replaces registered list with [ reg, reg2 ]

All other props from the override spec remain.
