prim_file

The bridge between 9p layer and file server.

The bridge between 9p layer and file server. The module translates file operations into 9p requests handling multiple 9p mounts. The module interface is the same as prim_file module of Erlang/OTP. The implementation is completely different.

The Port parameter passed to some API functions is always ignored.
