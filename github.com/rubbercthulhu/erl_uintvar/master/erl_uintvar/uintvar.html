uintvar - enconding and decoding API for variable-length unsigned integer (or simply uintvar).

uintvar - enconding and decoding API for variable-length unsigned integer (or simply uintvar).

Functions

encode(Max, Value) -> {ok, Octets} | {error, badard}

  • Max = pos_integer() | infinity
  • Value = non_neg_integer()
  • Octets = binary()

Encodes unsigned integer Value into binary Octets with uintvar format. The encoded value must be Max bytes size or less. If Max is infinity then entire value is encoded regardless of dimension. Returns {ok, Octets} in success case. If encoded value requires more then Max bytes then {error, badarg} is returned.

encode(Value) -> {ok, Octets} | {error, badarg}

  • Value = non_neg_integer()
  • Octets = binary()

Is equal to encode(infinity, Value).

encode32(Value) -> {ok, Octets} | {error, badarg}

  • Value = non_neg_integer()
  • Octets = binary()

Encodes 32bit unsigned integer into binary with uintvar format. The size of encoded uint32 is 5 bytes or less. So calling encode32(Value) is equal to encode(5, Value).

decode(Max, Octets) -> {ok, Value, Rest} | {error, badarg}

  • Max = pos_integer() | infinity
  • Octets = binary()
  • Value = non_neg_integer()
  • Rest = binary()

Decodes Octets as uintvar value into unsigned integer. Max is maximum number of bytes which is processed by the function to decode single uintvar value. If Max is infinity then decoding continues until entire single uintvar value is extracted or all bytes of Octets is processed. If uintvar value is decoded successfully then returns {ok, Value, Rest} where Value is decoded unsigned integer and Rest is the rest of Octets after encoded uintvar. Otherwise if first Max bytes of Octets do not contain entire uintvar value {error, badarg} is returned.

decode(Octets) -> {ok, Value, Rest} | {error, badarg}

  • Octets = binary()
  • Value = non_neg_integer()
  • Rest = binary()

Is equal to decode(infinity, Octets).

decode32(Octets) -> {ok, Value, Rest} | {error, badarg}

  • Octets = binary()
  • Value = non_neg_integer()
  • Rest = binary()

Decodes uintvar binary Octets into 32bit unsigned integer. The size of encoded uint32 is 5 bytes or less. So calling decode32(Octets) is equal to decode(5, Octets).
