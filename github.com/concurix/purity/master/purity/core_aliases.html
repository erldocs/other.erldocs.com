core_aliases

Locate aliases to variables in core erlang expressions.

Locate aliases to variables in core erlang expressions.

Functions

scan(Fun::c_fun() (see module cerl)) -> dict()

Scan a core erlang function and return a mapping of alias names to the corresponding variables names.

We are only interested in the reverse mapping of variable aliases, which should also be unique, so the expression case <cor1, cor2> of <Pred, List> -> ... <Pred, cor3> -> ... <cor1, cor2> when true -> match_fail will produce a mapping of {Pred, cor1}, {List, cor2} and {cor3, cor2}.
