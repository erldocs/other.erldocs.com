update(Plt::plt(), ExtraFiles0::files(), Table0::dict(), Final0::dict(), Options::options() (see module purity_utils)) -> plt()

Update PLT with a new table and cache, for the extra files provided. The Table is considered to include up-to-date values for both previous files, and ExtraFiles.

It is however possible that not all files are valid, even though they could have been analysed. This could be because they were e.g. deleted between the call to analysis and the call to update. Any module we can't checksum is therefore removed from both tables.

Return a copy of the PLT with specified files removed. Any functions from modules corresponding to those files are removed from the normal as well as the cached tables.
