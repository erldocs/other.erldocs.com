  • Opts = [{device, Device}]

nexax(Serial, Channel, Level) -> ok | {error, Error}

  • Serial = integer()
  • Channel = integer()
  • Level = boolean() | bell | integer()

Sends a nexax protocol request to the device. Serial should be in the range [0 - 16#3fffffff].
Channel should be in the range [1 - 16].
If Level is an integer it should be in the range [0 - 255].
