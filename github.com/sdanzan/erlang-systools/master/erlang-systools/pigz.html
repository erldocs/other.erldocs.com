pigz

A wrapper around the pigz program.

A wrapper around the pigz program. Provide a "compressor" process to send file compresion orders.

Functions

create_compressor(Options::list()) -> pid()

Create a compressor process to wait for compression tasks. Allowed options: { level, Level }: compression level (default 9). { parallel, Parallel }: compression threads. silent: no logging.

create_compressor() -> pid()

Equivalent to create_compressor([]).

delete_compressor(Compressor::pid()) -> ok

Requires the given compressor to stop serving compression tasks.

compress(String::string(), Compressor::pid()) -> string()

Synchronously compress a given file and return the compressed file name.

start_compress(String::string(), Compressor::pid()) -> string()

Send a compression request to a compressor. Once finished the compressor will send back a message: { Compressor, { success, CompressedFileName } } or { Compressor, { error, Reason } }
