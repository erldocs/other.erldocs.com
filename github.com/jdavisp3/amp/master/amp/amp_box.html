amp_box

AMP protocol utilities.

AMP protocol utilities. See: http://twistedmatrix.com/documents/current/api/twisted.protocols.amp.html http://twistedmatrix.com/documents/current/core/howto/amp.html

This module implements low-level encoding and decoding of AMP Boxes. An Amp Box is represented in Erlang as a list of key/value pairs.

Return a new decoder object suitable for unserializing a wire format of an Amp box. Decoders are required arguments for decode_bin_box/2.

decode_bin_box(Decoder::decoder(), New::binary()) -> {not_done, decoder()} | {amp_bin_box() (see module amp), decoder()}

Decode a part (or whole) of a box. If the complete box is decoded, return the box we decoded and a new decoder state. The decoder may contain another box, so a second decode call with an empty binary is needed.
