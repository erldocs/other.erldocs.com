football_events

Functions

new_match() -> term()

Generates a new match with two random teams, to start in 10..60 minutes.

matches() -> term()

returns a list of match ids

match_info(MatchId) -> term()

returns a pair consisting of the pair of teams involed and the current status of the match.

subscribe(MatchId) -> term()

subscribes the calling process to match_info events from the MatchId match. The info has the shape {match_info, match_id()}, status()}.

unsubscribe(MatchId) -> term()

start_ticks() -> term()

start the ticks that drives forward the matches.

stop_ticks() -> term()

tick_interval() -> term()

get the time in milli seconds between the ticks.

set_tick_interval(T) -> term()

set the tick interval to T milli seconds.
