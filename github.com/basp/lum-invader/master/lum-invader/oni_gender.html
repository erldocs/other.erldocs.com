ps(X1::gender()) -> binary()

Returns the subjective pronoun (e.g. he, she, it).

po(X1::gender()) -> binary()

Returns the objective pronoun (e.g. him, her, it).

pp(X1::gender()) -> binary()

Returns the possessive pronoun (e.g. his, her, its).

pq(X1::gender()) -> binary()

Returns the possessive pronoun in noun form (e.g. his, hers, its).

pr(X1::gender()) -> binary()

Returns the reflexive pronoun (e.g. his, hers, its).
