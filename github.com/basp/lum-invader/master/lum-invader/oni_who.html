Functions

init() -> true

Initializes the connections table.

This table is used to connect object ids and sockets.

insert_connection(Id::objid() (see module oni_db), Socket::socket() (see module gen_tcp)) -> true

Inserts a connection into the table.

Any existing connection with the same object id will be blissfully overwritten.

delete_connection(Id::objid() (see module oni_db)) -> true

Deletes the connection associated with given object id.

This will hapilly return with true if there are no connections for given id.

get_connection(Id::objid() (see module oni_db)) -> socket() (see module gen_tcp)

Returns the connection associated with given object id.

get_player(Socket::socket() (see module gen_tcp)) -> objid() (see module oni_db)

Returns the object id corresponding to the given socket.

list() -> [{Id::objid() (see module oni_db), Socket::socket() (see module gen_tcp)}]

Returns a list of all active connection associations.
