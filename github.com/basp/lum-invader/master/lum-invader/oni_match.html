Functions

list(Pred::(binary()) -> boolean(), List::[binary()]) -> match()

Matches the predicate on a list of binary strings.

This might return {ambiguous, [binary()]} when there is more than one match.

list_i(Pred::(binary()) -> boolean(), List::[binary()], Index::integer()) -> nothing | binary()

Like list but returns the nth match or nothing instead.

In contrast with list, this will never return ambiguous.

verb(Str::binary(), Verb::binary()) -> boolean()

Matches verb names including the wildcard asterisk.
