epmd_register() -> ok | {error, Reason::any()}

reregisters an disconnected node back to epmd. Requires the node to be initially started with '-proto_dist imem_inet_tcp' as command line option and there is a running instance of epmd process in the local system. If unsuccessful {error, Reason} is returned. This function never generates any exception.
