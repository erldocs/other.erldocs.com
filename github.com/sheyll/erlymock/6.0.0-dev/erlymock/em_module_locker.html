em_module_locker

Internal server that provides a locking mechanism between em processes so that no two mock processes mock the same modules.

Internal server that provides a locking mechanism between em processes so that no two mock processes mock the same modules. A mock process locks some modules and as soon as the mock process dies, its modules will automatically be unlocked.
