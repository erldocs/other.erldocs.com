mongrel_app

Mongrel application.

Mongrel application. This module implements the application behaviour. On startup the application starts the mongrel supervisor, mongrel_sup.

Functions

start(Type::any(), StartArgs::list()) -> {ok, pid()} | {error, any()}

Starts the application. This function creates an ETS table that is passed to the mongrel supervisor. The table lives the for the lifetime of the application.

stop(State::any()) -> ok

Stops the application.
