pmap(Fun::function(), List::list(), Limit::integer(), Timeout::timeout()) -> {list(), integer(), integer(), integer()}

Parallel pmap() (see Erlang/OTP docs for lists:pmap/2).

Evaluates Fun(Elem), for every element Elem in List, in parallel subject to the specified Limit number of parallel processes. Returns list of return values, in the same order as in List. Each result in the first tuple member appears as:

{ok, Res} for successful calls

{error, Res} for error calls

'timeout' for timed-out calls

For the 2nd-4th elements of the return tuple, the integers are: number of ok/successful calls, number of error calls, and number of timeout calls. .

pmap_kv(Fun, List) -> term()

pmap_kv(Fun, List, Limit) -> term()

pmap_kv(Fun::function(), List::list(), Limit::integer(), Timeout::timeout()) -> {list(), list(), list()}

Variation of parallel pmap() (see Erlang/OTP docs for lists:pmap/2).

The return value of this variation of pmap is a 3-tuple:

list({Argument, SuccessfulResult})

list({Argument, ErrorResult})

list({Argument, timeout})

For each of these three lists, the order of the results is the same relative to the original List argument.
