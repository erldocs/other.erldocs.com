Add the dbg:fun2ms() style match spec to the tracing system.

Examples:

add_match_spec(dbg:fun2ms(fun([_, target_category, _, _, _, _]) -> return_trace() end))

add_match_spec(dbg:fun2ms(fun([Pri, _, _, _, _, _]) when Prio > 0 -> return_trace() end))

add_match_spec(dbg:fun2ms(fun([_, target_category, target_module, _, _, _]) -> return_trace(); ([_, _, other_module, 88, _, _]) -> return_trace() end))

format_file(TraceFileName::string(), OutputFile::string(), MatchStr::[] | string()) -> ok

Format a raw trace file into a human-readable file.

If the MatchStr argument is "", all trace entries will be output. If MatchStr is any other string, then only trace entries which contain that string (exact string match, case sensitive, definitely no regexps!) will be output.
