Types

qplist() = [{term(), term()}]

Functions

get(Key::term(), Proplist::qplist()) -> term()

If {Key, Value} is in Proplist, Value is returned. Otherwise the atom undefined is returned.

get(Key::term(), Proplist::qplist(), Default::term()) -> term()

If {Key, Value} is in Proplist, Value is returned. Otherwise Default is returned.

set(Key::term(), Value::term(), Proplist::qplist()) -> qplist()

{Key, Value} is added to Proplist, replacing the first other occurrence of {Key, _} in Proplist, if any.

set_if(Key::term(), Value::term(), Proplist::qplist(), Cond::boolean()) -> qplist()

Behaves like set/3 if Cond is true; otherwise, Proplist is returned unmodified.

See also: set/3.

set_if_defined(Key::term(), Value::term(), Proplist::qplist()) -> qplist()

Behaves like set/3 unless Value =:= undefined, in which case Proplist is returned unmodified.

See also: set/3.

set_if_nonempty(Key::term(), Value::term(), Proplist::qplist()) -> qplist()

Behaves like set/3 unless Value =:= [] or Value =:= <<>>, in which case Proplist is returned unmodified.

See also: set/3.
