world_ctl_sserv

world_ctl_sserv is a socket based server which provides the interfaces to control the simulated world.

world_ctl_sserv is a socket based server which provides the interfaces to control the simulated world. The socket server is based on the behaviour gen_server and handles tcp sessions which can control the environment.
