The module provides a command line user interface from an Erlang shell by calling the start/0 function.

The module provides a command line user interface from an Erlang shell by calling the start/0 function. The functionalities:

Add trace case

List/Run trace cases

Save trace configuration to file

Load trace configuration from file

Set cookie

Version: 1.0

Functions

start() -> ok

Starts a terminal like configuration client for the onviso wrapper.
