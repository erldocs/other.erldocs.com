child_spec(Ref::ref() (see module ranch), NbAcceptors::non_neg_integer(), Transport::module(), TransOpts::any(), Options::any(), Handler::module(), HandlerOpts::any()) -> child_spec() (see module supervisor)

Return a child spec suitable for embedding.

When you want to embed Bones RPC in another application, you can use this function to create a ChildSpec suitable for use in a supervisor. The parameters are the same as in start_listener/7 but rather than hooking the listener to the Ranch internal supervisor, it just returns the spec.
