create_credentials(Provider::string() | binary(), UserName::string() | binary(), Password::string() | binary()) -> {ok, credentials()} | {error, {not_supported_provider, binary()}}

Return a credentials() term which can be passed to other elibcloud functions to set up the connection with the provider.

The supported providers are the following: EC2, OPENSTACK_HP, OPENSTACK_RACKSPACE, DUMMY.

The OPENSTACK_RACKSPACE provider does not support security groups (on Rackspace instances, all ports are open by default).

list_nodes(Credentials::credentials()) -> elibcloud_func_result(no_predefined_error)

List virtual machine nodes.

In case of success, the result is a JSON list containins JSON objects (once JSON object = one node). The JSON objects contain the following fields:

<<"id">>: Node ID.

<<"name">>: Node Name.

<<"state">> ("RUNNING" | "REBOOTING" | "TERMINATED" | "PENDING" | "UNKNOWN" | "STOPPED"): Node State

<<"public_ips">>: Public IP addresses associated with this node.

<<"private_ips">>: Private IP addresses associated with this node.

<<"image">>: Size of this node. (optional)

<<"size">>: Image of this node. (optional)

<<"extra">>: Optional provider specific attributes associated with this node.
