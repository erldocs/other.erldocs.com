eradius_server_mon

Manager for RADIUS server processes.

Manager for RADIUS server processes. This module manages the RADIUS server registry and validates and applies the server configuration from the application environment. It starts all servers that are configured as part of its initialization, then sends ping requests to all nodes that are part of the configuration in order to keep them connected.
