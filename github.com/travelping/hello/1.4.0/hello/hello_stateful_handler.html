reply(From_context::from_context(), Reply) -> ok

  • Reply = {ok, value() (see module hello_json)} | Error
  • Error = {error, ErrorCode} | {error, ErrorCode, ErrorMessage} | {error, ErrorCode, ErrorMessage, ErrorData}
  • ErrorCode = integer()
  • ErrorMessage = binary()
  • ErrorData = value() (see module hello_json)

Send an asynchronous reply to an RPC request. If you want to use asynchronous replies, your handler module must return {noreply, State} from handle_request/4. reply/2 can be called from any process, not just the handler process.

transport_param(Key::atom(), Context::context() | from_context(), Default::any()) -> any()

Get the value of a transport parameter. The list of supported transport parameters will vary between transport implementations.
If you rely on transport parameters, your RPC handler will no longer be transport-agnostic. Please take this into consideration when designing your system.
