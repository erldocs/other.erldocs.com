couchdoc

Functions

new() -> term()

Creates a new empty document. Uses a random UUID for the ID.

new(Attrs) -> term()

Creates a document with the specified Attrs. Uses a UUID for its ID. ---------------------------------------------------------------------------

new(Id, Attrs) -> term()

Creates a document with the specified Id and Attrs. Id must be a binary or a string. String Ids are converted to binary. ---------------------------------------------------------------------------

get_id(Doc) -> term()

Returns the document's Id. ---------------------------------------------------------------------------

get_rev(Doc) -> term()

Returns the document's current revision or undefined if the document hasn't been added to the database. ---------------------------------------------------------------------------

get_attr_names(Doc) -> term()

Returns the document's attributes as a property list. ---------------------------------------------------------------------------

get_attrs(Doc) -> term()

Returns all of the document's attributes as a property list. ---------------------------------------------------------------------------

set_attr(Name, Value, Doc) -> term()

Adds or updates a single attribute for Doc. ---------------------------------------------------------------------------

set_attrs(T, Doc) -> term()

Sets the specified list of attributes leaving any other attributes umodified. This is equivalent to calling set_attr/3 for each modified attribute. ---------------------------------------------------------------------------

replace_attrs(Attrs, Doc) -> term()

Replaces all of the document's attributes. ---------------------------------------------------------------------------

get_attr(Name, Doc) -> term()

Returns a single attribute value or undefined if the attribute doesn't exist. ---------------------------------------------------------------------------

get_attr(Name, Doc, Default) -> term()

Returns a single attribute value or Default of the attribute doesn't exist. ---------------------------------------------------------------------------

del_attr(Name, Doc) -> term()

Deletes a single document attribute. ---------------------------------------------------------------------------

is_deleted(Doc) -> term()

Returns a boolean indicating whether the document is deleted. ---------------------------------------------------------------------------
