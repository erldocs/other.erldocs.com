validate(Env, Form) -> term()

Validates the form data provided in Env. Returns {errors, Errors, RenderedForm} if there were errors or {ok, Data} if all of the form fields were validated and converted into their native values.

validate_field(Field, Env) -> term()

Validates a field. Returns {ok, Value} if the field is validated. Value is a converted value based on the environ params and is field type specific. If the field cannot be validated, returns {error, Error}, where Error is conventionally a tuple of {ErrorNo, ErrorMsg}.

process(Env, Form) -> term()

Returns either {form, RenderedForm} or (action, Action} where Action is action performed by the user as per the form's actions list.

form_action(Env, Rest) -> term()
