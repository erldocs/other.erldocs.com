pqueue2

Skew Heap Priority Queue. Ulf Wiger suggested pursuing a skew heap as an optimal Erlang priority queue implementation.

Skew Heap Priority Queue. Ulf Wiger suggested pursuing a skew heap as an optimal Erlang priority queue implementation. Unfortunately, testing has shown this solution to be more than 2 times slower than pqueue.
