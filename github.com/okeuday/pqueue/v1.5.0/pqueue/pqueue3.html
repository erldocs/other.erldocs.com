pqueue3

A Large Priority Queue. This priority queue implementation depends on layered tuples, so that tuple access times can be exploited for quick in/out priority queue operations when using 64 or more total priorities.

A Large Priority Queue. This priority queue implementation depends on layered tuples, so that tuple access times can be exploited for quick in/out priority queue operations when using 64 or more total priorities. This implementation was created to avoid the slowness within the priority queue used by both RabbitMQ and Riak (https://github.com/basho/riak_core/blob/master/src/priority_queue.erl).
