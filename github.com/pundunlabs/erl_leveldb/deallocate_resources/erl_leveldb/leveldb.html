Functions

open_db(Options::options(), Path::string()) -> {ok, DB::db()} | {error, Reason::any()}

Open a leveldb database with provided Options on provided Path. Returns {ok, DB} where DB is an NIF Resource, or {error, Reason}.

close_db(DB::db()) -> ok | {error, Reason::any()}

Close a leveldb database that is referenced by provided NIF Resource, DB.

get(DB::db(), ReadOptions::readoptions(), Key::key()) -> {ok, value()} | {error, Reason::any()}

Get the value of the key provided by Key from leveldb database referenced by NIF Resource DB. Operation performed using provided ReadOptions NIF Resource.

put(DB::db(), WriteOptions::writeoptions(), Key::key(), Value::value()) -> ok | {error, Reason::any()}

Put key/value pair provided by Key and Value to leveldb database referenced by NIF Resource DB. Operation performed using provided WriteOptions NIF Resource.

delete(DB::db(), WriteOptions::writeoptions(), Key::key()) -> ok | {error, Reason::any()}

Delete the key/value pair refered by the provided Key from leveldb database referenced by NIF Resource DB. Operation performed using provided WriteOptions NIF Resource.

write(DB::db(), WriteOptions::writeoptions(), DeleteKeys::[key()], PutKeyValuePairs::[{key(), value()}]) -> ok | {error, Reason::any()}

Write a batch of keys to delete and key/value pairs to put provided by DeleteKeys and PutKeyValuePairs to leveldb database referenced by NIF Resource DB. Operation performed using provided WriteOptions NIF Resource.

options(LeveldbOptions::#leveldb_options{comparator=integer(), create_if_missing=boolean(), error_if_exists=boolean(), paranoid_checks=boolean(), env=undefined | binary(), info_log=undefined | binary(), write_buffer_size=pos_integer(), max_open_files=pos_integer(), block_cache=undefined | binary(), block_size=pos_integer(), block_restart_interval=pos_integer(), compression=undefined | binary(), filter_policy=undefined | binary()}) -> {ok, options()} | {error, Reason::any()}

Make and get a NIF resource constructed by the provided LeveldbOptions. This resource can be used for first arity in open_db/2 function.

readoptions(LeveldbReadOptions::#leveldb_readoptions{verify_checksums=boolean(), fill_cache=boolean(), snapshot=undefined | binary()}) -> {ok, readoptions()} | {error, Reason::any()}

Make and get a NIF resource constructed by the provided LeveldbReadOptions. This resource can be used for second arity in get/3 function.

writeoptions(LeveldbWriteOptions::#leveldb_writeoptions{sync=boolean()}) -> {ok, writeoptions()} | {error, Reason::any()}

Make and get a NIF resource constructed by the provided LeveldbWriteOptions. This resource can be used for second arity in put/4. delete/3, write/4 functions.

destroy_db(Path::string(), Options::options()) -> ok | {error, Reason::any()}

Destroy the contents of the specified leveldb database with provided Path and Options. Returns ok or {error, Reason}.

repair_db(Path::string(), Options::options()) -> ok | {error, Reason::any()}

Try to repair the specified leveldb database with provided Path and Options. Returns ok or {error, Reason}.

approximate_sizes(DB::db(), Ranges::[range()]) -> {ok, [Bytes::integer()]} | {error, Reason::any()}

Approximate the file system space used by keys in given Range for a database. Returned size is compressed size and might be smaller than user data. The result might not include the size of recently written data.

approximate_size(DB::db(), ReadOptions::readoptions()) -> {ok, Bytes::integer()} | {error, Reason::any()}

Approximate the file system space used a database. Returned size is compressed size and might be smaller than user data. The result might not include the size of recently written data.

read_range(DB::db(), Options::options(), ReadOptions::readoptions(), Range::range(), Limit::pos_integer()) -> {ok, [{key(), value()}], Cont::complete | key()} | {error, Reason::any()}

Read a set of Key/Value Pairs in between key range that is sopecified by Range. Limit the number of pairs to be read by Limit.

read_range_n(DB::db(), ReadOptions::readoptions(), StartKey::key(), N::pos_integer()) -> {ok, [{key(), value()}]} | {error, Reason::any()}

Read N number of Key/Value Pairs from start key.

iterator(DB::db(), ReadOptions::readoptions()) -> {ok, binary()} | {error, Reason::any()}

Create a LevelDB Iterator and get NIF resource.

delete_iterator(It::it()) -> ok | {error, Reason::any()}

Delete a LevelDB Iterator.

first(It::it()) -> {ok, {key(), value()}} | {error, Reason::any()}

Iterate to first entry in LevelDB database and return first key/value pair.

last(It::it()) -> {ok, {key(), value()}} | {error, Reason::any()}

Iterate to last entry in LevelDB database and return last key/value pair.

seek(It::it(), StartKey::key()) -> {ok, {key(), value()}} | {error, Reason::any()}

Iterate to the entry specified by StartKey in LevelDB database and return seeked key/value pair.

next(It::it()) -> {ok, {key(), value()}} | {error, Reason::any()}

Iterate to next entry in LevelDB database and return the next key/value pair.

prev(It::it()) -> {ok, {key(), value()}} | {error, Reason::any()}

Iterate to previous entry in LevelDB database and return the previous key/value pair.
