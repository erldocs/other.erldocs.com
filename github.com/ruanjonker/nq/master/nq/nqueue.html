Enqueue a message.
Insert a message at the back of the queue.

deq(QName::queue_name()) -> {ok, message()} | {error, empty} | error()

Dequeue a message.
Remove a message from the front of the queue.

deq(QName::queue_name(), Fun::proc_fun(), Args::proc_args()) -> {ok, message()} | {error, empty} | error()

Dequeue a message with a fun.
Remove a message from the front of the queue by evaluating proc_fun().
The message is removed from the queue only if proc_fun() returns ok.

NOTE:The queue will block while proc_fun() is called.

peek(QName::queue_name()) -> {ok, message()} | {error, empty} | error()

purge(QName::queue_name()) -> ok | error()

Clear the queue.

subscribe(QName::queue_name()) -> ok

Subscribe for events.

The calling process will be subscribed to receiving notifications on the queue.

Notifications will be sent from the queue in the following fashion:

pid() ! ready_notification()

unsubscribe(QName::queue_name()) -> ok

Unsubscribe for events.

The calling process will be unsubscribed from receiving notifications on the queue.

subscriber_count(QName::queue_name()) -> non_neg_integer()

Return the number of subscribers on the queue.
