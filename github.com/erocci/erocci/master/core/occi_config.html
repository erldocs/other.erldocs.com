occi_config

Config has 2 sources: - application env - load/1 arg Application env override load/1 env.

Config has 2 sources: - application env - load/1 arg Application env override load/1 env

Config properties: - backends: a list of backends, loaded with occi_store:register/1 - name: base URI of the server. Normally, listeners are responsible for discovering name (e.g.: http://localhost:8080) - listeners: a list of listeners, loaded with occi_listener:register/1. - backend_timeout: timeout after which a backend is considered dead. Default to 5000 (ms) - categories_map: a function for mapping category id (occi_cid()) to an URI. Default to occi_category_mgr:hash/1 - categories_prefix: prefix for collections, for the occi_category_mgr:hash/1 function. Default: /collections

All other properties are stored in the config manager and accessible with get/1 and get/2.
