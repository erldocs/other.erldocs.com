BPar - Yet another worker pool.

BPar - Yet another worker pool

Three modes of operation * synchronous (run) * asynchronous (run_async) * asynchronous, wait for a worker to become ready (run_async_wait)

Start arguments, required * mod - worker callback module * size - number of workers

Start arguments, optional: * queue_size - max number of enqueued tasks * args - arguments to callback

Task arguments, optional: * queue_timeout - timeout for task waiting in queue (default infinity) * caller_alive - caller must be alive for execution (default false)

In addition to the queue_timeout a timeout can be placed for the entire call (defaults to 5 seconds).
