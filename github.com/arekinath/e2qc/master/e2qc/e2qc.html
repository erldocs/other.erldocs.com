cache(Cache::atom(), Key::term(), ValFun::function()) -> term()

Cache an operation using the given key.

ValFun is a zero-argument fun that computes some expensive operation and returns any term. The call to cache/3 will return that term, either by running the fun, or consulting the named Cache for Key.

cache(Cache::atom(), Key::term(), Lifetime::integer(), ValFun::function()) -> term()

Cache an operation using the given key with a timeout.

As for e2qc:cache/3, but the Lifetime argument contains a number of seconds for which this cache entry should remain valid. After Lifetime seconds have elapsed, the entry is automatically evicted and will be recalculated if a miss occurs.
