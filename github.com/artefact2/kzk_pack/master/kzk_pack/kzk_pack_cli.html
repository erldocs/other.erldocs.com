kzk_pack_cli

This script provides the kzkpack tool to manipulate packs from the command-line conviniently.

This script provides the kzkpack tool to manipulate packs from the command-line conviniently. The kzkpack binary should be created by make(1) in the project's root directory. The full documentation of this tool can be found in README.usage.
