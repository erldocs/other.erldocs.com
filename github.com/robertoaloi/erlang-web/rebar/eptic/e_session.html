Functions

new_session(SessionData::term()) -> string()

Initializes the new session with the SessionData and generates the cookie. The randomly generated cookie is returned.

update_session(Cookie::term(), NewData::term()) -> true | {error, no_prev_session}

Updates the session bound to the Cookie with the NewData. If no session with the Cookie has been found, {error, no_prev_session} is returned.

get_session(Cookie::term()) -> {ok, undefined} | {ok, term()}

Retrives the session state from the internal data storage. Updates also the session expire time.
Returns the previous session state or the tuple {ok, undefined} if no previous session has been found.

delete_session(Cookie::term()) -> []

Removes the session from the session table. Returns new, empty cookie.

print_sessions() -> ok

Prints all kept sessions on the Erlang system shell.
