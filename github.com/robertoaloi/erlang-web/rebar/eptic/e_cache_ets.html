e_cache_ets

Module responsible for managing the ets cached XML records.

Module responsible for managing the ets cached XML records.

Functions

read_file(Filename::string(), Expander::atom()) -> term()

Reads the file from ets cache. If the file has not been cached or the original one has changed, the cache will be read once again.
When the specified file is missing the erlang:error({Reason, File}) is called.

flush() -> true

Removes all the cache entries. Simply invalidates all the cached items.
