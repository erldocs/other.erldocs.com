Functions

read_file(Filename::string(), Expander::atom()) -> term()

Reads the file from disk cache. If the file has not been cached or the original one has changed, the cache will be read once again.
When the specified file is missing or the cached file cannot be created, the erlang:error({Reason, File}) is called.
