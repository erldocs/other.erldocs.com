Module responsible for translating the contents of the page.

Module responsible for translating the contents of the page.

The translations are read form the translation files (usually placed in config/languages/ directory (they should be specified in project.conf under language_files key). The language_files configuration tuple should look like: {langauage_files, {Language, PathToTranslationFile}}.
The translation files should contain a two element tuples: {Key, Translation} where Key should be string or a tuple of strings.

Functions

install() -> true

Loads the translation files and creates the proper ets table.

reinstall() -> true

Clears the translation and reloads the translation files.

translate(Language::atom(), Key::string() | tuple()) -> string()

Translates the given Key. If no translation for the given Language is provided and the Key is a tuple, then the Key is returned. Otherwise, "no translation found" sign is returned.

list() -> list()

Returns the list of all possible translations. The list is in format: {{Language, Key}, Translation}.
