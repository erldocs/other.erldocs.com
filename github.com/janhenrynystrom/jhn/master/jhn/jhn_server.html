start(Mod::atom()) -> {ok, pid()} | ignore | {error, term()}

Starts a jhn_server.

start(Mod::atom(), Options::opts()) -> {ok, pid()} | ignore | {error, term()}

Starts a jhn_server with options. Options are: {link, Boolean} -> if the server is linked to the parent, default true {timeout, infinity | Integer} -> Time in ms for the server to start and initialise, after that an exception is generated, default 5000. {name, Atom} -> name that the server is registered under. {arg, Term} -> argument provided to the init/1 callback function, default is 'no_arg'.

A cast is made to the server, allways returns ok.

delayed_cast(Time::non_neg_integer(), Msg::term()) -> reference()

A cast is made to the calling Server after Delay ms, always returns a timer reference that can be canceled using cancel/1.

cancel(Ref::reference()) -> non_neg_integer() | false

A timer a timer started using delayed_cast/2, returns time remaining or false.

abcast(Name::atom(), Msg::term()) -> ok

abcast(Nodes::[node()], Name::atom(), Msg::term()) -> ok

call(Server::server_ref(), Msg::term()) -> term()

A call is made to Server with default timeout 5000ms.

call(Server::server_ref(), Msg::term(), Timeout::timeout()) -> term()

A call is made to Server with Timeout. Will generate a timeout exception if the Server takes more than Timeout time to answer. Generates an exception if the process is dead, dies, or no process registered under that name. If the server returns that is returned from the call.

reply(Msg::term()) -> ok

Called inside call back function to provide reply to a call or cast. If called twice will cause the server to crash.

reply(From::from(), Msg::term()) -> ok

Called by any process will send a reply to the one that sent the request to the server, the From argument is the result from a call to from/1 inside a handle_req/2 or handle_msg/2 callback function.

from() -> from()

Called inside a handle_req/2 or handle_msg/2 callback function it will provide an opaque data data enables a reply to the call or cast outside the scope of the callback function.
