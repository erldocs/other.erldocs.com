start(Mod::atom()) -> {ok, pid()} | ignore | {error, term()}

Starts a jhn_fsm.

start(Mod::atom(), Options::opts()) -> {ok, pid()} | ignore | {error, term()}

Starts a jhn_fsm with options. Options are: {link, Boolean} -> if the fsm is linked to the parent, default true {timeout, infinity | Integer} -> Time in ms for the fsm to start and initialise, after that an exception is generated, default 5000. {name, Atom} -> name that the fsm is registered under. {arg, Term} -> argument provided to the init/1 callback function, default is 'no_arg'.

event(FSM::fsm_ref(), Msg::term()) -> ok

An event is sent to the FSM, always retuns ok.

delayed_event(Time::non_neg_integer(), Msg::term()) -> reference()

An event is sent to the calling FSM after Delay ms, always returns a timer reference that can be canceled using cancel/1.

cancel(Ref::reference()) -> non_neg_integer() | false

A timer a timer started using delayed_event/2, returns time remaining or false.

call(FSM::fsm_ref(), Msg::term()) -> term()

A call is made to FSM with default timeout 5000ms.

call(FSM::fsm_ref(), Msg::term(), Timeout::timeout()) -> term()

A call is made to FSM with Timeout. Will generate a timeout exception if the FSM takes more than Timeout time to answer. Generates an exception if the process is dead, dies, or no process registered under that name. If the FSM returns that is returned from the call.

reply(Msg::term()) -> ok

Called inside call back function to provide reply to a call or event. If called twice will cause the fsm to crash.

reply(From::from(), Msg::term()) -> ok

Called by any process will send a reply to the one that sent the request to the fsm, the From argument is the result from a call to from/1 inside a state function or handle_event/3 callback function.

from() -> from()

Called inside a state function or handle_req/3 callback function it will provide an opaque data data enables a reply to the call or event outside the scope of the callback function.
