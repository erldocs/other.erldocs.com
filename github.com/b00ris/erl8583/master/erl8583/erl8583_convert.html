erl8583_convert

This module provides a number of functions for converting between representations of data common in ISO 8583.

This module provides a number of functions for converting between representations of data common in ISO 8583. Particular data representations are ASCII and EBCDIC strings, BCD encoding and the encoding used for track 2 data.

Functions

string_to_ascii_hex(Str::string()) -> string()

Converts a string of characters to a list containing the ASCII hex character codes.

ascii_hex_to_string(HexStr::string()) -> string()

Converts a string containing ASCII hex characters to an equivalent ASCII string().

integer_to_string(IntValue::integer(), Length::integer()) -> string()

Converts an integer to an ASCII string of fixed length with leading zeroes if necessary.

pad_with_trailing_spaces(Str::string(), Length::integer()) -> string()

Pads an ASCII string with a number of spaces so that the resultant string has specified length.

binary_to_ascii_hex(BinValue::binary()) -> string()

Returns the ASCII hex encoding of a binary value.

binary_list_to_ascii_hex(BinList::binary()) -> string()

Returns the ASCII hex encoding of a list of bytes.

ascii_hex_to_binary(HexStr::string()) -> binary()

Returns the binary value corresponding to an ASCII hex string.

ascii_hex_to_binary_list(HexStr::string()) -> [byte()]

Returns a binary list corresponding to an ASCII hex string.

integer_to_bcd(IntValue::integer(), Length::integer()) -> [byte()]

Converts an integer to a list of specified length of BCD encoded bytes.

ascii_hex_to_bcd(HexStr::string(), PaddingChar::char()) -> [byte()]

Converts an ASCII hex encoded string to a list of BCD encoded bytes ()padded with a specified padding character if the string has odd length).

bcd_to_integer(BcdList::[byte()]) -> integer()

Converts a list of BCD encoded bytes to an integer.

bcd_to_ascii_hex(BcdList::[byte()], Length::integer(), PaddingChar::char()) -> string()

Converts a BCD encoding of a value of specified length (possibly padded with a padding character) to an ASCII hex string.

track2_to_string(Track2Data::[byte()], Length::integer()) -> string()

Converts a list of track 2 nibbles to a string containing an ASCII encoding of the same data.

string_to_track2(Str::string()) -> [byte()]

Converts a string of ASCII characters to a track 2 encoding.

ascii_hex_to_digit(HexDigit::string()) -> integer()

Converts a string containing 1 ASCII hex character to its value. The ASCII hex digits A-F can be encoded as upper-case ("A" - "F") or lower-case ("a" - "f") characters.

digit_to_ascii_hex(IntValue::integer()) -> string()

Converts a value in the range 0-15 to a 1 character ASCII string containing the equivalent hexadecimal digit. Values 10 - 15 are converted to the upper case strings "A" - "F".

strip_trailing_spaces(Str::string()) -> string()

Strips trailing spaces from an ASCII string.

strip_leading_zeroes(Str::string()) -> string()

Strips leading zeroes from an ASCII string.

ascii_to_ebcdic(Str::string()) -> [byte()]

Converts an ASCII string to an EBCDIC string.

ebcdic_to_ascii(EbcdicStr::[byte()]) -> string()

Converts an EBCDIC string to an ASCII string.

list_to_bitmap(FieldIds::[integer()], Offset::integer()) -> binary()

Converts a list of integer IDs to a 64-bit bitmap. Values in the range [Offset+1, Offset+64] are encoded. Values outside the specified range are ignored.

bitmap_to_list(Bitmap::binary(), Offset::integer()) -> [integer()]

Converts a 64-bit bitmap to a list of integers starting at a specified offset.
