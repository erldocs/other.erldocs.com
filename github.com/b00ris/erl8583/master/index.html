Module Index


erl8583

erl8583_convert
This module provides a number of functions for converting between representations of data common in ISO 8583.

erl8583_fields
This module provides a function to get how a field is encoded according to the 1987 version of the ISO 8583 specification.

erl8583_fields_1993
This module provides a function to get how a field is encoded according to the 1993 version of the ISO 8583 specification.

erl8583_fields_2003
This module will provide a function to get how a field is encoded according to the 2003 version of the ISO 8583 specification.

erl8583_marshaller
This module provides functions for marshalling and unmarshalling ISO 8583 messages into various encodings.

erl8583_marshaller_ascii
This module provides functions to marshal an iso8583message() into an ASCII string, to unmarshal an ASCII string into an iso8583message().

erl8583_marshaller_binary
This module provides functions to marshal an iso8583message() into a list of bytes, to unmarshal a list of bytes into an iso8583message().

erl8583_marshaller_ebcdic
This module provides functions to marshal an iso8583message() into an EBCDIC string, to unmarshal an EBCDIC string into an iso8583message().

erl8583_marshaller_json
This module marshals an iso8583message() into a JSON document and can unmarshal a JSIBL document into an iso8583message().

erl8583_marshaller_xml
This module marshals an iso8583message() into an XML document and can unmarshal an XML document into an iso8583message().

erl8583_message
Functions for constructing, reading and updating ISO 8583 messages.
