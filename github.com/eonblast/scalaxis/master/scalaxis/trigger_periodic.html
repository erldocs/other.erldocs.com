trigger_periodic

Periodic trigger for (parameterized) modules.

Periodic trigger for (parameterized) modules.

Can be used by a module Module in order to get a configurable message (by default {trigger}) every BaseIntervalFun() (default: Module:get_base_interval()) milliseconds.

Use this module through the interface provided by the trigger module, initializing it with trigger_periodic!

Types

state()

Functions

init(BaseIntervalFun::interval_fun() (see module trigger), MinIntervalFun::interval_fun() (see module trigger), MaxIntervalFun::interval_fun() (see module trigger), MsgTag::message_tag() (see module comm)) -> state()

Initializes the trigger with the given interval functions and the given message tag used for the trigger message.

now(X1::state()) -> state()

Sets the trigger to send its message immediately, for example after its initialization. Any previous trigger will be canceled!

next(X1::state(), IntervalTag::interval() (see module trigger)) -> state()

Sets the trigger to send its message after BaseIntervalFun() milliseconds. Any previous trigger will be canceled!

stop(State::state()) -> state()

Stops the trigger until next or now are called again.
