read(Key::atom()) -> any() | failed

Reads a config parameter. If it is not found, the application's environment is checked or failed is returned. The result will be cached in the config.

start_link(Files::[name() (see module file)], Options::[tuple()]) -> {ok, pid()}

Starts the config process. If Options contains a {config, [{Key1, Value1},...]} tuple, each Key is set to its Value in the config.

start_link2() -> {ok, pid()}

Short for start_link2/1 with no options.

start_link2(Options::[tuple()]) -> {ok, pid()}

Starts the config process and determines the config files from the application's environment. If there is no application, "scalaris.cfg" and "scalaris.local.cfg" are used. If Options contains a {config, [{Key1, Value1},...]} tuple, each Key is set to its Value in the config.
