handle(Filename::filename(), Reason::any()) -> {fallback, {snapshot, datetime() (see module calendar)}, {reason, any()}} | {error, Reason::any()}

Try explain the error to the user and run configuration fallback if possible.

This functions tries to find human-readable error explanation and seeks for previous configuration snapshot in the mnesia database. If snapshot is found it is used producing a log info message, if not - the function fails with the original reason.
