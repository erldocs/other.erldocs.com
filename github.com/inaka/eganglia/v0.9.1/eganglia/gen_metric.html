gen_metric

Generic metric server for the Ganglia Monitoring System.

Generic metric server for the Ganglia Monitoring System

The user module should export: 
init(Args::term()) -> init_result()

Opens and/or initializes the server.

handle_metric(State::term()) -> metric_result()

Called each time the metric should be collected

handle_call(Msg::term(), State::term()) -> call_result()

Called from gen_metric:call/2

handle_cast(Msg::term(), State::term()) -> cast_result()

Called from gen_metric:cast/2

terminate(Reason :: term(), State) -> _

Let the user module clean up. Always called when server terminates.

 Timeouts (expressed in milliseconds) are used everywhere to determine the next time the metric should be announced

Types

call_result() = {reply, Reply :: term(), State :: term()}
                      | {stop,
                         Reason :: term(),
                         Reply :: term(),
                         State :: term()}

cast_result() = {noreply, State :: term()}
                      | {stop, Reason :: term(), State :: term()}

init_result() = {ok, Title :: binary(), State :: term()}

metric_result() = {ok, Value :: term(), State :: term()}
