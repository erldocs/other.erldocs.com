riakc_counter

Encapsulates a counter data-type.

Encapsulates a counter data-type. Counters are integers that can be incremented or decremented. Like the other eventually-consistent types, the original fetched value is unmodified by increments. Instead, increments are captured for later application in Riak. Use dirty_value/1 to access a local "view" of the updated value.
