hash_ring

This module implements a hash ring.

This module implements a hash ring. Items are hashed into the ring NumReplicas times.

An item is hashed using the following:

sha1(Item + string(N)) where N is (1..NumReplicas)

Note: When N is converted to a string it is not zero padded.

Functions

get_item(Key, X2) -> term()

Finds the item that contains the Key on the Ring

create_ring(Items, NumReplicas) -> term()

Creates a hash ring that places Items in the ring NumReplicas times

A "replica" just means that the Item is placed on the ring in multiple places to make the distribution more even.

add_item(Item, X2) -> term()

Adds an item into the ring. Returns the ring with item added in.

remove_item(Item, X2) -> term()

Removes an item and its replicas from the ring. Returns the ring without the item.
