start_link(ConnList) -> term()

Starts a supervisor that manages pools of Redis connections.

ConnList is a list of {{Host, Port}, PoolSize} tuples. PoolSize connections will be opened to each {Host, Port}.

ConnList = [{{"localhost", 6379}, 5}, {{"localhost", 6380}, 5}].

The above ConnList specifies 5 connections to localhost:6379 and 5 connections to localhost:6380.
