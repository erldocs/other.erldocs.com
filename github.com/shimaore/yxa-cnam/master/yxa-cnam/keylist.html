keylist

Kind of generic key-value list module, but really only meant/used for storing SIP message headers.

fetch(Key, Keylist) -> [string()]

  • Key = string() | atom()
  • Keylist = #keylist{}

Return the contents of the header that matches Key. The result is a list of values, or the empty list if no header matching Key was found in Keylist.

map(Func, Keylist) -> [term()]

  • Func = function()
  • Keylist = #keylist{}

Apply Func to all elements in Keylist. Func must be of arity three, and accept arguments Func(Key, Name, Value) where Key is an atom() or a string(), Name is always a string() and Value is [string()].

normalize(Name::Key) -> string() | atom()

  • Key = string()

Convert header key to a standard format - either an atom for well known headers, or a lowercased string. We don't do list_to_atom() because that way it would be possible to exhaust our atom() space by sending us SIP requests with made-up headers.
