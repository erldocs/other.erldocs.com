cowboy_http_req

HTTP request manipulation API.

HTTP request manipulation API.

Almost all functions in this module return a new Req variable. It should always be used instead of the one used in your function call because it keeps the state of the request. It also allows Cowboy to do some lazy evaluation and cache results where possible.
