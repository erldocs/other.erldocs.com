partial(Ranges::[range()], Boundary::binary(), FileSize::non_neg_integer()) -> [range() | iolist()]

Return a partial response. The total size of the multipart response must be calculated before we begin sending it. We can accomplish this by writing everything except the content of each range to a temprary buffer. The content_lengt/1 function is used to calculate the effective size based on this buffer. Callers are responsible for replacing occurances of range() terms in the response with the actual contents of the range.

content_length(Partial::[iolist() | range()]) -> non_neg_integer()

Return the content length of a multipart response. The input is expected to be a partial response. All range() terms are expected to occur as an element of the top-level list of the response.
