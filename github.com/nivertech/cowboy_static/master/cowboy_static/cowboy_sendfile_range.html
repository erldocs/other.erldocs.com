get_range(T) -> {boolean(), T}

Get the range requested by a client. This function will return the raw value of the Range header. The parse_range function must be used to convert this into a usable form.

parse_range(BinRange::binary(), ContentLength::pos_integer()) -> {Start::pos_integer(), End::pos_integer()}

Return a usable form of a Range header value. The range returned by this function is inclusive.

make_range(Start::uint(), End::uint(), ContentLength::uint()) -> {binary(), binary()}

Make a Content-Range header with a known Content-Length. The content range header is included in 206 (Partial Content) responses and indicates which byte range is sent in the response. The content range header also includes the content length of the resource to aid clients making subsequent requests.

make_range(Start::uint(), End::uint()) -> {binary(), binary()}

Make a Content-Range header with an unknown Content-Length. This is equivalent to make_range/2. The total content range is replaced with an asterix (*) to indicate that the total content is unknown.
