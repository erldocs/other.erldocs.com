a collection of functions to help handling multiple processes.

a collection of functions to help handling multiple processes.

the module also implements the gen_server behavior, bus should only be started by the functions of the module, as they will also take care of ending the server.

wait_for_exit(Pid::pid(), Reason::term()) -> ok | {error, {expected, Exp::term()}, {value, Val::term()}}

wait for the process with the given pid to terminate.

function will ensure that the process terminated with the return value given for Reason. If the process has terminated before this function was called, or has never existed, this function will return immediately with 'ok' indicating success.

wait_for_event(EventMgrPid::pid(), Event::term()) -> {ok, Event::term()} | {error, timeout}

wait for th given event to occour, blocking the function until the event is recieved.

the function will register a handler with the event manager given, and return if the event is seen.

if the event is not seen within 3000 milliseconds, {error, timeout} will be returned
