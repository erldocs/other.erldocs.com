pwm

This is the implementation of the PWM interface module.

This is the implementation of the PWM interface module. There is one process for each pwm device. Each process is liked to the supervisor of the pwm application.

Types

channel() = atom()

devname() = string()

val() = 0..1024

Functions

start_link(X1::{channel(), devname()}) -> {ok, pid()} | {error, reason}

Start the process with the channel name and Initialize the devname device. You can identify the device by a channel name. Each channel drive a devname device.

stop(Channel) -> term()

Stop the process channel and release it.

value(Channel, X) -> term()

Assign a PWM value at the channel.

load_nif() -> ok | {error, error_type}

Load the PWM C library.

pwm_init() -> ok | {error, pwm_initialization_error}

Initialise the PWM peripheral.

pwm_value(Val::val()) -> ok

Set PWM value.

pwm_release() -> ok

Releases the PWM peripheral and unmaps the memory.
