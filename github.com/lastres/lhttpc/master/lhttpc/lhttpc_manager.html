lhttpc_manager

Connection manager for the HTTP client.

Connection manager for the HTTP client. This module provides the pool of connections functionality. It exports funcions intended to provide information and manage the existing pools.

Types

destination() = {host(), integer() >= 1, boolean()} | string()

host() = string()
               | {integer(), integer(), integer(), integer()}

pool_id() = pid() | atom()

Functions

dump_settings(PidOrName::pool_id()) -> list()

list_pools() -> term()

Lists all the pools already started.

Returns the total number of active clients maintained by the specified lhttpc pool (manager).

connection_count(PidOrName::pool_id()) -> non_neg_integer()

Returns the total number of active connections maintained by the specified lhttpc pool (manager).

connection_count(PidOrName::pool_id(), Destination::destination()) -> non_neg_integer()

Returns the number of active connections to the specific Destination maintained by the httpc manager.

update_connection_timeout(PidOrName::pool_id(), Milliseconds::non_neg_integer()) -> ok

Updates the timeout for persistent connections. This will only affect future sockets handed to the manager. The sockets already managed will keep their timers.
