sql_query(Query::iodata(), Connection::pgsql_connection()) -> odbc_result_tuple() | {error, any()}

Perform a query. This function creates a statement and runs step as many times as required. The result is:

{selected, Rows} if the query was a SELECT query.

{updated, NbRows} if the query was not a SELECT query.

(the return types are compatible with ODBC's sql_query function).
