deverl_console_wx

This module initalises the console, which is currently implemented as a wxStyledTextCtrl.

This module initalises the console, which is currently implemented as a wxStyledTextCtrl. In the current implementation we manage the prompt ourselves (deverl_console_parser strips the prompt returned from the port), so that we can write to the console anywhere we like without having to worry about deleting/re-numbering prompts. We settled on this current implementation as a comprimise (in the original implementation everything received from the port was written to the console). The optimal solution would be to use the erl_parse/erl_eval/erl_scan APIs in the same way that shell.erl does from std lib. We may well use that approach in the future.
