contract_manager_tlog

Contract manager transaction logging.

Contract manager transaction logging.

This module can be used by the server to log individual protocol transactions, if desired. This simple default module uses the Erlang/OTP error_logger module; we highly recommend that the sasl application be running to take full advantage of OTP's error and event handling capabilities.

tlog/6 callback API:

-type op() :: rpc | lpc | event_in | event_out. -type now() :: {pos_integer(), pos_integer(), pos_integer()}. -type plugin() :: module().

-spec tlog(op(), now(), plugin(), Q::term(), Reply::term(), Status::term()) -> ok.
