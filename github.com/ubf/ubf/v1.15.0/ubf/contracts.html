checkType(HumanType::contract_type_name_atom(), Term::term(), Mod::contract_module_name_atom()) -> yup | error_hints_term_only_human_readable_sorry()

Given a contract type name, a term to check against that contract type, and a contract module name, verify the term against that contract's type.

Example usage from the irc_plugin.con contract:

contracts:checkType(ok, ok, irc_plugin).

contracts:checkType(bool, true, irc_plugin).

contracts:checkType(nick, {'#S', "foo"}, irc_plugin).

contracts:checkType(joinEvent, {joins, {'#S', "nck"}, {'#S', "grp"}}, irc_plugin).

contracts:checkType(joinEvent, {joins, {'#S', "nck"}, {'#S', bad_atom}}, irc_plugin).

NOTE: This is a brute-force function, but it works, mostly. Don't try to have a computer parse the output in error cases: the failure output is meant only for human eyes.
