Functions

start_link(Name, Port, Opts, Fun) -> {ok, pid()} | ignore | {error, term()}

  • Name = atom()
  • Port = non_neg_integer()
  • Opts = [socket_setopt() (see module inet)]
  • Fun = function()

Starts the listener supervisor

Name - the name of the server Port - the port which is listening on Opts - socket options Fun - call back function.

When there is a new comer, the call back function is invoked in the context of a worker process without any argument, the callback function is expected to return {ok, Pid} where Pid is the worker process id.
