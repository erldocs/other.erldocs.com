Master_slaves

The module contains an example implementation of the Master/slaves exercise from [www.erlang.org/exercises]

Functions

start(N::integer()) -> pid()

Register current process as output and spawn master process

master(N::integer()) -> integer() | nil

Register process as master, spawn N slave processes and listen

to_slave(N::integer(), Msg::list()) -> list()

Send message Msg to master process and tell him to forward it to N-th slave process. Then wait for proper reaction.

slave_listen() -> nil

Listen to messages. If message is DIE, terminate gracefully. Otherwise send ack to master process.
