WorkerSpawner

The module contains an implementation of the Erlang web framework service worker spawner

Functions

spawnWorker(Pid::pid(), Module::atom()) -> ok

Function called after client request. It spawns worker process of suitable type and waits for it to do his work. If he finishes in time, function returns generated respons, otherwise timeout message is returned. If requested worker doesn't exist, error is returned.
