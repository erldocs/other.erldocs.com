just_request_pq

A data structure for the scheduler.

A data structure for the scheduler. Holds all the customers' requests belonging to a particular gateway. There are ten (0..9) possible customer priorities. Requests are ordered by [CustomerPriority desc, AttemptTime asc].

Returns an empty priority queue.

in(R::request(), PQ::request_pq()) -> false | {true, request_pq()}

Inserts a request into the priority queue. Returns 'false' if the customer doesn't exist.

out(TimeNow::precise_time() (see module just_time), PQ::request_pq()) -> empty | {request(), request_pq()}

Takes out next available request from the priority queue. Returns 'empty' if there is no ready request in the queue.

throttle(Customer::binary(), PQ::request_pq()) -> false | {true, request_pq()}

Temporarily deactivates a customer. While the customer is being throttled, its requests are not being dequeued. Returns 'false' if there is no such customer in the queue or if it's already being throttled.

unthrottle(Customer::binary(), PQ::request_pq()) -> false | {true, request_pq()}

Activates a throttled customer. Returns 'false' if there is no such customer in the queue or if it's not being throttled.

update(Customer::binary(), PQ::request_pq()) -> false | {true, request_pq()} | {true, request_pq(), [binary()]}

Updates priority of a customer. Returns 'false' if there is no such customer in the queue or if its priority hasn't changed. Returns {true, request_pq()} if the priority has changed. Returns {true, request_pq(), Requests::binary()} if the customer is no longer available, where Requests is a list of request UUIDs that belonged to the customer.

remove(Customer::binary(), PQ::request_pq()) -> false | {true, request_pq(), [binary()]}

Removes all the customer's requests from the queue. Returns 'false' if there is no such customer in the queue. Returns {true, request_pq(), Requests::binary()} otherwise, where Requests is a list of request UUIDs that belonged to the customer.
