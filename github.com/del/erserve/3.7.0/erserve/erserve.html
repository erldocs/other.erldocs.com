erserve

erserve allows Erlang code to call R code via Rserve.

erserve allows Erlang code to call R code via Rserve. The basic usage is to open a connection and send one or more commands to it, then use the type/1 and parse/1 functions to transfer the data from internal representation to "Erlangy" formats.

Convert an R dataframe to an Erlang proplist. This is a simple helper that runs erserve:parse/1 and performs these transformations: - convert binary strings to lists - convert single-value lists to scalars (e.g. [123] -> 123)
