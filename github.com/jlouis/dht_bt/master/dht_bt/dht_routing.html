Wrap a routing table in timer constructs

This module implements a "wrapper" around the routing table code, by adding timer tables for nodes and ranges. The module provides several functions which are used to manipulate not only the routing table, but also the timers for the routing table. The invariant is, roughly, that any node/range in the table has a timer.

The module also provides a number of query-facilities for the policy module (dht_state) to query the internal timer state when a timer triggers.
