File I/O Request Process.

File I/O Request Process.

All file I/O operations are performed in a process different from the client process requesting a read or write operation from the IO subsystem.

Functions

start_read(TorrentID, Dirpid, Piece, Offset, Length, ClientPid) -> term()

Start a read request process. On success the chunk is sent to the client process.

start_write(TorrentID, Dirpid, Piece, Offset, Length, Chunk, ClientPid) -> term()

Start a write request process. On success an acknowledgement is sent to the client process.
