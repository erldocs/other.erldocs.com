etorrent_memory_logger

Log to a memory table.

Log to a memory table

The gen_event event handler here is used to handle events and their insertion into into an ETS table. It is used mainly by the webui for listing the last 12 hours of events that occurred in the system.

Functions

add_handler() -> ok

Add the handler to etorrent_event.

delete_handler() -> ok

Remove the handler from etorrent_event.

all_entries() -> [{term(), term(), term()}]

Return all entries in the memory logger table
