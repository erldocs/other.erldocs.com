gsplit(N::integer(), L::[term()]) -> {[term()], [term()]}

Graceful split. Works like lists:split, but if N is greater than length(L1), then it will return L1 =:= L2 and L3 = []. This will gracefully make it ignore out-of-items situations.

queue_remove_check(Item::term(), Q::queue()) -> queue()

Remove items from a queue If Item is present in queue(), remove the first occurence and return it as {ok, Q2}. If the item can not be found return false. Note: Inefficient implementation. Converts to/from lists.

queue_remove(Item::term(), Q::queue()) -> queue()

Remove first occurence of Item in queue() if present. Note: This function assumes the representation of queue is opaque and thus the function is quite ineffective. We can build a much much faster version if we create our own queues.
