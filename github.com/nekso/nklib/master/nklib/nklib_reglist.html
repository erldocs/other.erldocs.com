Generic registration data type.

Generic registration data type

This module implements a datatype to manage pids registering to terms. You can register and unregister a pid() to any regterm(). Later on, you can find the regterm()'s registered by a process, or the processes that have registered a regterm().

For each new pid(), a monitor is started. On receiving any {'DOWN', _Ref, process, Pid, _Reason} you must call down/2 to see if this pid belongs to a registered process.
