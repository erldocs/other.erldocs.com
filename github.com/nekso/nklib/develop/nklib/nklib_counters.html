Nekso Process Counters.

Nekso Process Counters.

This module implements a ETS-based process counters manager. Any process can register and update any number of counters, and they can be shared with other processes. When the process exists, all updates made from that process are reverted.

Functions

incr(Name::term()) -> ok

Equivalent to incr(Name, 1, self()).

incr(Name::term(), Value::integer()) -> ok

Equivalent to incr(Name, Value, self()).

incr(Name::term(), Value::integer(), Pid::pid()) -> ok

Creates or increments synchronously a Name counter associated with process having Pid. Value will be the initial value if the counter doesn't exist, or it will be added to the previous value if it exists. Value must be a positive or negative integer. When the process having Pid exists, all updates sent from it will be reverted.

value(Name::term()) -> integer()

Gets a counter's current value.

del(Name::term()) -> ok

Equivalent to del(Name, self()).

del(Name::term(), Pid::pid()) -> ok

Removes all Name counter updates sent from Pid.

async(Op::[Op]) -> ok

  • Op = Name | {Name, Value} | {Name, Value, Pid}
  • Name = term()
  • Value = integer()
  • Pid = pid()

Performs an asynchronous multiple counter update as a batch. Default value for Value is 1, and the current process's pid() for Pid.
