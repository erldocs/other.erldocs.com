enable() -> ok | {error, term()}

Forces enabling of ensemble subsystem. This should only be performed in a single node in the cluster. The recommended way is letting enable the subsystem using the 'enable_consensus' riak_core parameter, and waiting to have three nodes in the cluster.

After the subsystem is enabled, riak_core will take care of add every new cluster member to the ensemble cluster and as a member of the 'root' ensemble, and also removing leaving nodes.
