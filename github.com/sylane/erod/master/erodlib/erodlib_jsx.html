encode_any(Key, Value) -> Jsx

  • Key = atom()
  • Value = term()
  • Jsx = json_term() (see module jsx)

Gives a JSX formated term for the give value.

Throws an error if the given value doesn't match any expectation.

If the value is a record, assume there is a module with the same name as the record with and encode/2 function able to encode it.

If the value is a property list (a list with a tuple as the first element) it will be converted by erodlib_term:encode/2 function. If the list is empty it will eb encoded as a list, not as an object.

The Key argument is the name of the attribute this value is for to be able to add some context to the format error.
