Stage producing data from a TCP socket.

Supported output formats:

- {list, line} - {list, block} - {list, {block, MaxSize}} when is_integer(MaxSize) - {list, {packet, T}} when T =:= 1; T =:= 2; T =:= 4 T =:= asn1; T =:= acdr; T =:= sunrm; T =:= fcgi; T =:= tpkt - {binary, line} - {binary, block} - {binary, {block, MaxSize}} when is_integer(MaxSize) - {binary, {packet, T}} when T =:= 1; T =:= 2; T =:= 4 T =:= asn1; T =:= acdr; T =:= sunrm; T =:= fcgi; T =:= tpkt - {iodata, line} - {iodata, block} - {iodata, {block, MaxSize}} when is_integer(MaxSize) - {iodata, {packet, T}} when T =:= 1; T =:= 2; T =:= 4 T =:= asn1; T =:= acdr; T =:= sunrm; T =:= fcgi; T =:= tpkt - {http, list} - {http, binary}

Supported input format:

- message

Supported pull queries:

- next - {size, {bytes, S}} when is_integer(S) (Only for block format)
