closest_points

Find the closest pair of 2D points from a given set of points.

Find the closest pair of 2D points from a given set of points.

This is an Erlang implementation of a well known recursive divide and conquer algorithm with complexity O(n log n).

Usage

  $ make shell
  1> D = closest_points:random_data(10000).
  2> closest_points:find_closest(D).

Benchmark

  $ make shell
  1> closest_points:benchmark(10000, 20).
