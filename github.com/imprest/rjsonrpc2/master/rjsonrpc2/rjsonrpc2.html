Restricted implementation of the JSONRPC 2.0 spec http://www.jsonrpc.org/specification

Only supports JSONRPC 2.0 requests with named parameters and needs a interface (proplists) of methods and parameter types that can decoded as valid jsonrpc requests. See README for more information.
