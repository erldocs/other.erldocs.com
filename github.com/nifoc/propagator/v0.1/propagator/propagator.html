Types

group() = atom()

tag() = atom()

Functions

start() -> ok

Starts the Propagator application and all of its dependencies. This is really only meant for usage inside the console.

stop() -> ok

Stops the Propagator application and all of its dependencies. This is really only meant for usage inside the console.

create(Group::group()) -> ok

Creates a new group that processes can publish and/or subscribe to.

delete(Group::group()) -> ok

Removes an existing group. If the group does not exist, nothing happens.

publish(Group::group(), Tag::tag(), Data::term()) -> ok | {error, term()}

Publishes Data to Group and broadcasts it to all subscribers. All messages have to be tagged, allowing subscribers to easily pattern-match on the messages they receive.

subscribe(Group::group()) -> ok | {error, term()}

Subscribes the current processes (self()) to a group. Delegates to subscribe/2.

subscribe(Group::group(), Pid::pid()) -> ok | {error, term()}

Subscribes any Pid to a given Group.

unsubscribe(Group::group()) -> ok | {error, term()}

Unsubscribes the current processes (self()) from a group. Delegates to unsubscribe/2.

unsubscribe(Group::group(), Pid::pid()) -> ok | {error, term()}

Unsubscribes any Pid from a given Group.

groups() -> [group()]

Returns a list of all existing groups.

is_group(Group::group()) -> boolean()

Checks wether or not a given group exists.

subscribers(Group::group()) -> [pid()] | {error, term()}

Returns a list of all processes that are subscribed to a Group.

is_subscriber(Group::group(), Pid::pid()) -> boolean()

Checks wether or not Pid is subscribed to Group.

statistics(Group::group()) -> {ok, [{atom(), term()}]} | {error, term()}

Returns a property list with statistics about a given group.

Currently the following keys are returned:
group: The name of the group
subscriber_count: Current number of subscribers
message_count: Number of unique messages this process (group) sent
