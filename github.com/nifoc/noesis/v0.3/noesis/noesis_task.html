async(Fun::function()) -> ref()

Starts a task that can be awaited on.
This function spawns a process that is monitored by the caller process.

Starts a task that can be awaited on.
This function wraps an erlang:apply/3 in a function and delegates to async/1.

await(Task::task()) -> term()

Awaits a task() reply. The Timeout is set to 5000 ms.

await(Ref::task(), Timeout::pos_integer()) -> term()

Awaits a task() reply. This function returns {error, timeout} after Timeout ms.

await_multi(Tasks::proplist(A, task()) (see module noesis_proplists)) -> proplist(A, term()) (see module noesis_proplists)

Awaits multiple task() replies. The Timeout per task is set to 5000 ms.

Awaits multiple task() replies. The Timeout is applied on a per task basis.
