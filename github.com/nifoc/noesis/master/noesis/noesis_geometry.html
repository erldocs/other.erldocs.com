contains_point(Bounds::bounds(), Point::coordinates()) -> boolean()

Checks whether or not a given point is inside the supplied bounds() tuple.

extend(Bounds::bounds(), Point::coordinates()) -> bounds()

Extends the bounds() tuple by the given point, if the bounds don't already contain the point.

crosses_antimeridian(Bounds::bounds()) -> boolean()

Returns whether or not the bounds intersect the antimeridian.

distance(StartCoordinates::coordinates(), DestCoordinates::coordinates()) -> number()

Calculates the great-circle distance between two coordinates, that is the shortest distance between two points on the surface of a sphere.
StartLng, StartLat, EndLng and EndLat are all expected to be in degrees.

rhumb_distance(StartCoordinates::coordinates(), DestCoordinates::coordinates()) -> number()

Given a starting point and a destination point, this will calculate the distance between the two.
StartPoint and DestPoint are both expected to be in degrees.

Partially based on Movable Type Scripts by Chris Veness.

rhumb_destination_point(Point::coordinates(), Bearing::bearing(), Distance::number()) -> coordinates()

Given a starting point, a bearing and a distance, this will calculate the destination point. If you maintain a constant bearing along a rhumb line, you will gradually spiral in towards one of the poles.
Point and Bearing are both expected to be in degrees. Distance is expected to be in kilometers.

Based on Movable Type Scripts by Chris Veness.

rhumb_bearing_to(StartPoint::coordinates(), DestPoint::coordinates()) -> float()

Given a starting point and a destination point, this will calculate the bearing between the two.
StartPoint and DestPoint are both expected to be in degrees.

Partially based on Movable Type Scripts by Chris Veness.

deg2rad(Deg::number() | coordinates()) -> number() | coordinates()

Converts degrees to radians.

rad2deg(Rad::number() | coordinates()) -> number() | coordinates()

Converts radians to degrees.

normalize_lat(Lat::latitude()) -> latitude()

Normalizes a latitude to the [-90, 90] range. Latitudes above 90 or below -90 are capped, not wrapped.

normalize_lng(Lng::longitude()) -> longitude()

Normalizes a longitude to the [-180, 180] range. Longitudes above 180 or below -180 are wrapped.

normalize_bearing(Bearing::bearing()) -> bearing()

Normalizes a bearing to the [0, 360] range. Bearings above 360 or below 0 are wrapped.
