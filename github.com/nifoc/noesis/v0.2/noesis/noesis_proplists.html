The noesis_proplists module includes functions that work on property lists.

Functions

get_keys(List::proplist()) -> [term()]

Returns a list of all keys in the property list, not including duplicates.

get_value(Key::term(), List::proplist()) -> term()

Delegates to get_value/3 and sets the default return value to undefined.

get_value(Key::term(), List::proplist(), Default::term()) -> term()

Returns the value of a key/value property in List. Returns the default value if Key is not found.

delete_keys(Rest::[term()], List::proplist()) -> proplist()

Removes multiple keys from a property list and returns a new one. Duplicate keys are also removed.

keypos(Key::term(), List::proplist()) -> pos_integer() | undefined

Returns the position of a key in a property list. If Key does not exist undefined will be returned.

extract(Keys::[term()], List::proplist()) -> proplist()

Delegates to extract/4. NullValue is set to undefined and Defaults is set to an empty list.

extract(Keys::[term()], List::proplist(), NullValue::term()) -> proplist()

Delegates to extract/4. Allows you to set NullValue and sets Defaults to an empty list.

extract(Keys::[term()], List::proplist(), NullValue::term(), Defaults::proplist()) -> proplist()

Extracts key/value pairs from a property list and returns a new property list containing only extracted pairs.
You can provide a set of default values that will be used as fallback if the key is not found in List. If the key is not found in Defaults either, the NullValue will be used for the value part.

partial_extract(Keys::[term()], List::proplist()) -> proplist()

Delegates to partial_extract/3. NullValue is set to undefined.

partial_extract(Keys::[term()], List::proplist(), NullValue::term()) -> proplist()

Extracts key/value pairs from a property list and returns a new property list containing only extracted pairs.
Pairs not found in List (and all those matching NullValue) will not be included in the property list that is returned.

merge(ListA::proplist(), ListB::proplist()) -> proplist()

Delegates to merge/3. Fun is set to function that always uses the value from ListB.

merge(Fun::(Key, ValueA, ValueB) -> Value, ListA::proplist(Key, ValueA), ListB::proplist(Key, ValueB)) -> proplist(Key, Value)

Merges two property lists, ListA and ListB, to create a new list. All the Key/Value pairs from both lists are included in the new property list. If a key occurs in both lists then Fun is called with the key and both values to return a new value.
