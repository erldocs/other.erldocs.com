Initialize connection with client by sending greeting message and starting two loops. First for doing actual work contained in messages (writing to DB, switching modes) and second for receiving messages from socket and parse them.

Loop that process messages being in transmit mode. Messages are delivered in the form of tuple {message_type, [Opts]}, where message_type denotes the type of message (MODE READER for example) and Opts is the list of tuples with message properties (user credentials for example).
