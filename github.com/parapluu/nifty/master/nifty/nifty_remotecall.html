Functions

start() -> ok

Starts remote node to safely call NIFs

stop() -> ok

Stops remote node

restart() -> ok

Restarts remote node, useful for testing (side-effects are reset)

call_remote(Mod::atom(), Func::atom(), Args::[term()]) -> term()

Works like erlang:apply/3, with the exception that the calls are redirected to a remote node. The node must be started in order to be able to call functions remote. If the node is down, {error, node_down} is thrown. If the node crashes during the call (SIGSEGV or similar) {error, node_crashed} is thrown.
