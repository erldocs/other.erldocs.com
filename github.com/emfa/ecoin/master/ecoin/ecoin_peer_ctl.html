ecoin_peer_ctl

Controlling peer process This is processes initate new incoming and outgoing connection.

Controlling peer process This is processes initate new incoming and outgoing connection. It does the version handshake and validate that the connection should indeed be used. After the handshake is done it spawns a sending and a receiving process.
