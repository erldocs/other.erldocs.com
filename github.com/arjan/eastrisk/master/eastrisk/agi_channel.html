agi_channel

This module represents a channel process.

This module represents a channel process.

When the AGI Server gets a connection, it spawns an AGI Channel using this module. When in initialisation is done this module will use the agi_events to notify event handlers about the new channel. When the program has finished, agi_channel:close/2 must be called.

Functions

close(ChannelPid) -> ok

Closes the channel.
