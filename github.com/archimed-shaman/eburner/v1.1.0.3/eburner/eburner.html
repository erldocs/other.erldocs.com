load(ConfigName, Getter) -> Result

  • ConfigName = binary()
  • Getter = () -> string()
  • Result = startchild_ret() (see module supervisor)

Loads the new the config with the help of Getter and starts the new gen_server config holder, identified by the ConfigName

unload(ConfigName) -> Result

  • ConfigName = binary()
  • Result = ok | {error, Error}
  • Error = running | restarting | not_found | simple_one_for_one

Unloads the config server, identified by the ConfigName

set_logger(Logger) -> Result

  • Logger = (Level, Msg) -> ok
  • Result = ok
  • Level = debug | trace | info | warning | error
  • Msg = string()

Sets the logger function fun(Level, Msg). Level must take one of the following values: - debug - trace - info - warning - error Msg is the string to be logged.

get_config(ConfigName) -> Result

  • ConfigName = binary()
  • Result = {config, CurrentConfig}
  • CurrentConfig = string()

subscribe(ConfigName) -> Result

  • ConfigName = binary()
  • Result = {config, CurrentConfig}
  • CurrentConfig = string()

Returns the current config value, identified by ConfigName and subscribes the calling pid to receive the messages, when config is reloaded. The messages looks like {config, Config :: string()}
