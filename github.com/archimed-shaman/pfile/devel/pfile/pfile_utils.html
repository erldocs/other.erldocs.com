truncate_id(TokenChars, TokenLen) -> Result

  • TokenChars = string()
  • TokenLen = number()
  • Result = string()

When lexer takes a key, it should find the equation symbol to ensure, that it is exactly the key. So, the string looks like "key =". Tabulars, space, line wraps and the equation symbol should be removed from the end of output string. So, if the input is "key_name =", output will be "key_name". It is a dirty hack, but there is no other obvious way.
