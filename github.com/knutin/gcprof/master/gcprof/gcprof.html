trace(Pid, IdentityF) -> term()

Starts tracing garbage collections in the given process. Note: gen_server:cast is used, so at some point in the future, tracing *might* start.

trace_me(IdentityF) -> term()

Starts tracing garbage collections in the calling process.

sample_me(IdentityF, X2) -> term()

Traces the calling process if the second argument is 0 or 'true'. Useful for executing sample operations that return 'true' or 'false' or '0'.
