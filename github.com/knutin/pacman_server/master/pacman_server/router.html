router

Router for connecting a running game with one or more Comet web requests.

Router for connecting a running game with one or more Comet web requests. A game worker will call router:state_update(Token, State) after Pacman and the ghost has moved. The router will then send a message to all processes registered for that token. The process may then decide to emit the new board state to it's client.
