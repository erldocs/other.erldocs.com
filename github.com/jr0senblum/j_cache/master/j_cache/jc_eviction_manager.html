jc_eviction_manager

jc_eviction_manager spys on the cache tables, creating and deleting TTL timers as necessary.

jc_eviction_manager spys on the cache tables, creating and deleting TTL timers as necessary.

Since 'get' opperations extend timers but no cache table change is made, jc.erl must explicitly tell this server to extend a timer.

Types

rec_ref() = reference()

seconds() = integer() >= 0

ttl() = seconds()

Functions

add_timer(TTLSecs::ttl(), RecRef::rec_ref()) -> ok

Create a TTL timer for the cache entry with the supplied reference. A 0 TTL is ignored as it is the equivalent of infinity.

cancel_timer(RecRef::rec_ref()) -> {ttl, non_neg_integer()} | ok

Cancel the timer for the cache entry with the supplied reference. Returns the TTL associated with the cache entry if there was one, else ok.

extend_timer(RecRef::rec_ref()) -> ok

Extend the TTL for a record.

start_link() -> {ok, pid()} | ignore | {error, any()}

Starts the server and links the caller to it.
