sk_tracer

This module contains utility functions for tracing the execution of skel programs.

This module contains utility functions for tracing the execution of skel programs.

Most skel processes will emit events when they send messages, allowing the tracer to view them in a nice way. This is by far the easiest way to debug the parallelism, rather than 'printf' statements.
