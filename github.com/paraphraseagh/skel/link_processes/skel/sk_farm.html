sk_farm

This module contains the initialization logic of a Farm skeleton.

This module contains the initialization logic of a Farm skeleton.

A task farm has the most basic kind of stream parallelism - inputs are sent to one of n replicas of the inner skeleton for processing.

Example 
        skel:run([{farm, [{seq, fun ?MODULE:p1/1}], 10}], Input)
In this simple example, we produce a farm with ten workers to run the sequential, developer-defined function p/1 using the list of inputs Input.
