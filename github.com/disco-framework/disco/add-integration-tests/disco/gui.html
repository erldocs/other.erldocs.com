Handles interaction with the GUI(s).

Handles interaction with the GUI(s).
This module interfaces possibly multiple external GUIs to the rest of the application. Each GUI has its own gui_port process. The functions in this module handle the delivery of updates to all the GUIs and deliver commands from all the GUIs to the dj.
This module starts a single gui_port_sup which then starts and supervises a list of gui_ports.
