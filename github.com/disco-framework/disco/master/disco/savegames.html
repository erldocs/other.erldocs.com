savegames

Utility functions for saving and loading the game state to/from disk.

Utility functions for saving and loading the game state to/from disk.
Savegame files are created with autosave/1 or save_state/2. To load a savegame, the file should be read with file:consult/1, contents validated with validate/3 and applied with apply/2.
