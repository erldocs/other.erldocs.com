barkeeper

Generates the concrete input strings for the workers.

Generates the concrete input strings for the workers. The defined process opens a port for communicating with the barkeeper executable (location configurable in config file).
The barkeeper generates the worker input from a problem specification and a problem state.

Functions

start_link(ExtProg) -> term()

Start the barkeeper process and open the port, starting the external executable
