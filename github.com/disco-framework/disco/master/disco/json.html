process(ReceivedString::string(), Decoder::decoder()) -> {ok, json_value()} | {incomplete, decoder()} | {bad_json, decoder()}

Try to process an incoming string into valid JSON. Can return a stateful decoder if the string did not contain a complete valid JSON term. The decoder can be used with further strings to construct a complete JSON term.

handle(DecodedObj::json_value(), Handler::(json_value()) -> ok | unknown_json) -> ok | unknown_json

Construct a port message as defined by our protocol, a string ending with a newline, from a JSON term.

process_attrs(Fun::function(), Attrs::[attr_spec()], JsonTerm::json_value()) -> ok | unknown_json

Check wether all the specified attributes are contained in the given JSON object and if successful optionally call the given conversion handlers with the attribute values. Then call the given fun() with the processed values.
