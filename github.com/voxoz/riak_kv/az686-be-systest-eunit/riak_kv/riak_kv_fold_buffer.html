Provide operations for creating and using size-limited buffers for use in folds.

Provide operations for creating and using size-limited buffers for use in folds.

Types

buffer() =
            #buffer{acc = [any()],
                    buffer_fun = undefined | function(),
                    max_size = undefined | integer() >= 1,
                    size = integer() >= 0}

Functions

new(MaxSize::pos_integer(), Fun::([any()]) -> any()) -> buffer()

Returns a new buffer with the specified maximum size and buffer function.

add(Item::any(), Buffer::buffer()) -> buffer()

Add an item to the buffer. If the size of the buffer is equal to the maximum size of this buffer then the buffer function is called on the accumlated buffer items and then the buffer is emptied.

flush(Buffer::buffer()) -> buffer()

Call the buffer function on the remaining items and then reset the buffer.

flush(Buffer::buffer(), Fun::function()) -> buffer()

Call the specified buffer function on the remaining items and then reset the buffer.

size(Buffer::buffer()) -> non_neg_integer()

Returns the size of the buffer.
