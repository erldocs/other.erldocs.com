Functions

format_img_output(Input::string()) -> [term()]

Formats the output of get_images() into a proplist. For example, [{55, "Arch Linux"}, {1, "Ubuntu 10.10"}].

format_flavour_output(Input::string()) -> [term()]

Formats the output of get_flavours() into a proplist. For example, [{1, "256 server"}, {2, "512 server"}].

format_create_srv_output(Input::string()) -> dictionary()

Formats the output of create_server into a dictionary.

format_list_srv_output(Input::string()) -> [dictionary()]

Formats the output of list_server into a list of dictionaries.
