gmt_elog_policy

A lightweight event logging policy implementation.

A lightweight event logging policy implementation.

The emphasis of this module is lightweight. The enabled() function may be called tens of thousands of times per second.

The use of the 4 arguments to enabled can be free-form, since this entire module will be replaced whenever tracing is desired. However, it's strongly recommended that the following conventions be used:

Level: error | warning | info | debug

Category: term()

Module: module()

Line: integer()

Types

log_level() = error | warning | info | debug

Functions

enabled(Level::log_level(), Category::term(), Module::module(), Line::integer(), Fmt::string(), Args::list()) -> true | false

Fixture for event logging and event tracing. Returns true if event was reported to event_logger. Otherwise, returns false. --------------------------------------------------------------------
