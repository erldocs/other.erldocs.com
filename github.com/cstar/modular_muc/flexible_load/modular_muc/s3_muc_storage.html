s3_muc_storage

Multi User Chatroom persistent storage on S3.

Multi User Chatroom persistent storage on S3. You need to specify the s3_bucket option in the ejabberd.cfg mod_muc options And set {storage, s3_muc_storage}. You'll need to have the erls3 application installed.

Functions

init(Host, ServerHost, Opts) -> ok | {error, Reason}

called by mod_muc:init/2.

store_room(Host, ServerHost, Name, Type, Opts) -> ok | {error, Reason}

Stores room. Called on configuration change when MUC is persistent.

restore_room(Host, ServerHost, Name) -> {Handler, Config}

restores room from storage

forget_room(Host, ServerHost, Name) -> ok

removes room from storage
