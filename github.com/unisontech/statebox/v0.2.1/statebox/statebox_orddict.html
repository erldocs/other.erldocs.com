statebox_orddict

Statebox wrappers for orddict-based storage.

Statebox wrappers for orddict-based storage.

Types

op() = statebox:op()

orddict() = proplist()

proplist() = [{term(), term()}]

statebox() = statebox:statebox()

Functions

from_values(Vals::[proplist() | statebox()]) -> statebox()

Return a statebox() from a list of statebox() and/or proplist(). proplist() are convered to a new statebox() with f_merge_proplist/2 before merge.

orddict_from_proplist(P::proplist()) -> orddict()

Convert a proplist() to an orddict() (by sorting it). Only [{term(), term()}] proplists are supported.

is_empty(Box::statebox()) -> boolean()

Return true if the statebox's value is [], false otherwise.

f_union(K::term(), New::[term()]) -> op()

Returns an op() that does an ordsets:union(New, Set) on the value at K in orddict (or [] if not present).

f_subtract(K::term(), Del::[term()]) -> op()

Returns an op() that does an ordsets:union(Del, Set) on the value at K in orddict (or [] if not present).

f_merge_proplist(New::term()) -> op()

Returns an op() that merges the proplist New to the orddict.

f_merge(New::term()) -> op()

Returns an op() that merges the orddict New to the orddict.

f_update(Key::term(), Op::op()) -> op()

Returns an op() that updates the value at Key in orddict (or [] if not present) with the given Op.

f_store(Key::term(), Value::term()) -> op()

Returns an op() that stores Value at Key in orddict.

f_delete(Pair::{term(), term()}) -> op()

Returns an op() that deletes the pair if present from orddict.

f_erase(Key::term()) -> op()

Returns an op() that erases the value at K in orddict.
