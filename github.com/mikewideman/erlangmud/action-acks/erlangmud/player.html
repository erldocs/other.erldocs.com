A Player Character (PC).

A Player Character (PC). PCs are the user's avatar in the game. PCs are controlled by the user.

Functions

start(Name::string(), Health::non_neg_integer() | default, Attack::pos_integer() | default, Room::pid()) -> pid()

Spawn a new player process, initializing it with the given name, health, and room (in which it is located). The health and attack may be set to the default values if desired. Returns the player pid.

See also: make_character/4.
