flv_writer

Module to write FLV files.

Module to write FLV files

There are two ways to use it: 1) passive with state 2) active with process, consuming #video_frame{}

On lower level, lies atomic writer: {ok, Writer} = flv_writer:init_file(Filename), {ok, Writer1} = flv_writer:write_frame(Frame, Writer)

Other way is to launch process, that will accept media frames: {ok, Pid} = flv_writer:start_link(Filename), Pid ! Frame

The code above will open 
  Filename

and write there frames. Also it is possible to pass writer function:

{ok, File} = file:open("test.flv", [binary, write]), {ok, Pid} = flv_writer:start_link(fun(Data) -> file:write(File, Data) end), Pid ! Frame
