Types

disable_opt() = temporary | syncronous

enable_opt() = recursive | temporary | syncronous

mark_opt() = immediate | temporary

service() = binary() | string()

service_state() =
            {Servie :: binary(), State :: binary(), StartTime :: binary()}

svcs_opt() = {zone, Zone :: zone()}

zone() = binary() | string()

Lists all SMF services in the current zone.

list(Zone::zone()) -> {ok, [service_state()]} | {error, N::pos_integer()}

Lists all SMF services for a given zone.

This must be executed from the solaris global zone.
