ancestors(Objs0::[any_obj()]) -> [obj()]

Given a list of obj() return a list of all the ancestors. Ancestors are objects that all the other objects in the list have descent from.

children(ObjsIn::[any_obj() | obj()]) -> [obj()]

Given a list of obj() return a list of the children objects. Children are the descendants of all others objects.
