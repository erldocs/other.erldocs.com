compile(Specs::{file, filename()} | proplist()) -> encoder()

Compile Erlang type specs to avro schema and return data serializer function. Compilation inputs are either defined in a spec file or passed in directly as a prop-list. See priv/sample.spec for example.

load(Specs::{file, filename()} | proplist()) -> encoder()

For the same purpose as compile/1 only the types are loaded from the file specified in input spec. See priv/sample.spec for example. This is to enable schema compilation in Erlang source code compile time and fast load in runtime.
