add(Bootstrap_handler::#bootstrap_handler{pid=undefined | pid(), module=undefined | module(), arg=undefined | term()}) -> ok | term()

Add a bootstrap event handler to this event manager. If the passed handler structure contains a process id the handler registration is supervised. If the given module is already registered, adding the handler will fail.

delete(Module::module()) -> ok | term()

Removes an event handler from this event manager, specified by the callback module implementing the bootstrap behaviour.
