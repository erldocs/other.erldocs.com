syslog

Main module of the syslog application.

Main module of the syslog application. This module contains functions to enable or disable logging via syslog as well as two convinience functions to log messages with a certain severity.

This module also defines the behaviour that must be implemented by protocol backends.

Functions

enable() -> ok

Enable SASL report logging via syslog. This will also disable the standard error_logger TTY output, if successful.

disable() -> ok

Disable SASL report logging via syslog. This will also re-enable the standard error_logger TTY output, if successful.

log(Severity::severity(), Msg::string()) -> ok

Log a message with a specific severity.

log(Severity::severity(), Fmt::string(), Args::[term()]) -> ok

Log a format message with a specific severity.
