syslog_monitor

A server keeping track of event handler registrations.

A server keeping track of event handler registrations. It will attach/detach the syslog_h event handler to the error_logger as necessary (e.g. if the handler gets detached accidentially on error) providing it with the needed UDP socket.

Functions

start_link() -> {ok, pid()} | {error, term()}

Starts the locally registered monitor server which in turn will attach the syslog_h event handler at the error_logger event manager.
