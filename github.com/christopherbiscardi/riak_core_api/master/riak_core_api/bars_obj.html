ancestors(Objs0::[bars_obj()]) -> [bars_obj()]

Given a list of bars_obj() return a list of all the ancestors. Ancestors are objects that all the other objects in the list have descent from.

children(Objs) -> term()

Given a list of bars_obj() return a list of the children objects. Children are the descendants of all others objects.

equal(ObjA::bars_obj(), ObjB::bars_obj()) -> boolean()

Predeicate to determine if ObjA and ObjB are equal.

merge(FSM::atom(), Objs::[bars_obj()]) -> bars_obj()

Merge the list of Objs, calling the appropriate reconcile fun if there are siblings.

unique(Objs::[bars_obj()]) -> [bars_obj()]

Given a list of Objs return the list of uniques.

update(Val::val(), Updater::node(), Bars_obj::bars_obj()) -> bars_obj()
