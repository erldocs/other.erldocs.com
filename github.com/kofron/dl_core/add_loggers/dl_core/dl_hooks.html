dl_hooks

dl hooks are like git hooks - they are functions that get applied either before or after data is taken to transform it in some way.

dl hooks are like git hooks - they are functions that get applied either before or after data is taken to transform it in some way. this could be conversion to JSON, or calibration, or whatever. a hook is defined as a function hook :: dl_data() -> dl_data() such that the caller sees a transparent data source.
