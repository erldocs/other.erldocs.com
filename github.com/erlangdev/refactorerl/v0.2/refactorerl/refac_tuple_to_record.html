refac_tuple_to_record

This module implements the tuple to record refactoring.

This module implements the tuple to record refactoring. This version contains just the really basic refactoring. The refactoring works only in one module and in one step deep. It is always create the compensation macros to the record.

Functions

tuple_to_record(File::string(), FromLine::integer(), FromCol::integer(), ToLine::integer(), ToCol::integer(), RecordName::string(), RecordParamsString::string()) -> {ok, done}

Performs the precondition checks and the and the collects all the necessary information for the refactoring. It throws various terms when there is an error, with this structure: {atom(), term()}. If the atom is ok, the refactoring has been preformed without a problem.

Parameter description:

  File : The path of the module.
  FromLine : The begining of the selection(Line).
  FromCol : The begining of the selection(Column).
  ToLine : The end of the selection(Line).
  ToCol : The end of the selection(Column).
  RecordName : The name of the record.
  RecordParamsString : The name of the record fields.
