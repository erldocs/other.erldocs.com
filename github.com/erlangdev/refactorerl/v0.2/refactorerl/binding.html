get_binding_occurrence_candidates(MId::integer(), VarName::string(), NodeId::integer()) -> [integer()]

Returns the list of binding occurrence candidate ids for a variable.

Parameter description:

  MId : Id of the module.
  VarName : The name of the variable.
  NodeId : The root of the search; has to be
                        of type ?CLAUSE. Is usually the scope
                        of VarName.
