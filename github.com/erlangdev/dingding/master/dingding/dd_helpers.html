add_item_to_proplist_value(Item, PropList, Key) -> term()

When you have a proplist that contains {string(), list(any())} items, you can use this function to add items to the list(any()). This uses lists:append to make sure it gets appended to the end of the list. Since we will use this to add modules to a specific channel configuration, I want that module to be run after the original modules.

remove_item_from_proplist_value(Item, PropList, Key) -> term()

This does the reverse of the adding function.
