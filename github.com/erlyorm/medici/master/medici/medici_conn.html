medici_conn

The medici_conn module handles a single principe connection to the Tyrant remote database.

The medici_conn module handles a single principe connection to the Tyrant remote database. If is a simple gen_server that will dispatch requests to the remote database and exit if its connection to the remote database closes so that its supervisor can start another connection handler.

Types

db_type() = hash | tree | fixed | table

endian() = little | big
