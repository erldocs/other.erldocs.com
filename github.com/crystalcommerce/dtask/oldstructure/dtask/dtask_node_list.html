dtask_node_list

DTask Node List is a zipper implementation on a list.

DTask Node List is a zipper implementation on a list. It is in concept a circuarly-linked list with a pointer to a current "focus" element. Calling step advances this pointer through the list.

Elements are ordered but the order of additional elements being inserted is not guaranteed.

Types

node_list() = {list(), list()}

Functions

new(List::list()) -> node_list()

Creates a new node list based off of the given list of elements. The focus will be set to the first element.

is_empty(X1::node_list()) -> boolean()

Returns whether or not the node list is empty.

focus(X1::node_list()) -> any() | undefined

Returns the current element in the node list. Returns undefined if the node list is empty.

step(X1::node_list()) -> node_list()

Advances the focus by one element and returns the resultant node list.

size(X1::node_list()) -> integer()

Returns the number of nodes in the node list.

add(Node::any(), X2::node_list()) -> node_list()

Inserts an element into the node list in random order and returns the resultant node list.

seek(Node::any(), X2::node_list()) -> {ok, node_list()} | {error, term()}

Fast-forward the list until the focus is on Node. Returns {ok, NewList} or {error, Reason} if Node was not found.

drop(X1::node_list()) -> node_list()

Removes the focus element and returns the resultant node list.
