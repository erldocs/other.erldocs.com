dict2rest_dict_client

RFC 2229 client.

RFC 2229 client. The Dictionary Server Protocol (DICT) is an old TCP transaction based query/response protocol that allows a client to access dictionary definitions from a set of natural language dictionary databases.

See RFC 2229 for details. http://tools.ietf.org/html/rfc2229
