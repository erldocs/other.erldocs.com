Cat's Eye Technologies' Erlang Brainf*ck Interpreter.

An implementation of the world's most beautifulest langugage in the world's second most beautifulest language.

This program demonstrates:

sequential programming in Erlang

simulating Brainf*ck's updatable store by passing arguments

Variable names used in this program:

I = Instruction pointer

B = Brainf*ck program

D = Data pointer (tape head position)

M = Memory (Brainf*ck tape)

Functions

interpret(B::[instruction()], StorageSize::integer()) -> {integer(), tuple()}

  • integer()

The main user interface to the Brainf*ck interpreter. The user generally passes a list, which is parsed into a tuple. In this tuple, each Brainf*ck instruction represented by an atom, except for [ and ], which are represented by a nested tuple to make interpretation simpler. The return value is a tuple containing the data pointer and another tuple representing the state of the Brainf*ck tape after all is said and done.
