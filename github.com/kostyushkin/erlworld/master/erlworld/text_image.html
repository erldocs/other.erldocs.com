new(Display::display(), Text::string()) -> text_image()

Creates a new textual image using the default font.

new(Display::display(), Text::string(), WXFont::wxFont() | none) -> text_image()

Creates an image displaying the given text using the given font. The font should be a wxFont, and it's the callers responsibility to destroy it after calling this function. For the default font suplly the atom 'none'.
