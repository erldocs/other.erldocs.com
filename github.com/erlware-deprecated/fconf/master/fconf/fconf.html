fconf

Supports configuration aspects of the build system.

Supports configuration aspects of the build system. Allows applications to query aspects of the data.

Functions

start_config(Name, Override) -> ok

Start a new config with a handler and and override

stop_config(Name) -> ok

Stop the config identified by the name.

add_config(Name, AdditionalConfig) -> ok

Add a config to the system. The config must be in the format that the system can understand and use.

store(Name, Path, Value::any()) -> ok

Add a key to the config.

get_value(Name, Path) -> Value | undefined

Get a value from the config.

get_value(Name, Key, Default) -> Value | Default

Attempts to get the specified key. If the key doesn't exist it returns the requested default instead of just undefined.

delete(Name, Key) -> ok

Delete a value from the config.

exit(Name) -> ok

Tell sin_config to shutdown.
