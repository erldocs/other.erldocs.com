ktt_decode

Does a safe parsing of erlang tuple syntax.

Does a safe parsing of erlang tuple syntax. If it finds an atom it uses list_to_existing atom to translate it. This will only work if that atom is already in the atom table. This means its a bit quirky but it works for what I need.
