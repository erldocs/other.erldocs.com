Functions

split(String::string()) -> [string()]

Return a list of strings of the strings in String separated by any whitespace characters.

split(String::string(), Separators::char() | string() | whitespace) -> [string()]

Return a list of strings of the strings in String, using Separators as the delimiter string.

See also: split/3.

split(String::string(), Separators::char() | string() | whitespace, MaxSplit::integer()) -> [string()]

Return a list of strings of the strings in String, using Separators as the delimiter string. If MaxSplit is > 0, only MaxSplit splits will occur (giving MaxSplit+1 resulting strings).

strip(String::string()) -> string()

Strips both left and right sides of string of any whitespace.

strip(String::string(), Dir::direction()) -> string()

Strips Dir sides of string of any whitespace.

strip(String::string(), Dir::direction(), Chars::[char()]) -> string()

Strips Dir of string of any characters specified.

is_whitespace(Ch::char()) -> boolean()
