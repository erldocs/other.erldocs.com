Fold over middleware components L, processing each one with process/2.

foldr(Ctx::ewgi_context(), L::[ewgi_app()]) -> ewgi_context()

Same as foldl/2 but in the opposite direction.

See also: foldl/2.

process(F::ewgi_app(), Ctx::ewgi_context()) -> ewgi_context()

If the context contains no errors, processes a middleware component. If the middleware generates an error, it is optionally logged and an error is returned.

noop() -> ewgi_app()

Simple 'noop' middleware that returns the EWGI context unchanged.
