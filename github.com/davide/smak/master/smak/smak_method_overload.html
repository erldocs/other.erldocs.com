smak_method_overload

Basic Smak HTTP method override middleware.

Basic Smak HTTP method override middleware. Parses the x-http-method-override header of a POST request or the _method query-string variable.

Example: 

  OverloadApp = smak_method_overload:init([disable_header]),
  EwgiApp = smak_mw:compose(OverloadApp, MyApp)
