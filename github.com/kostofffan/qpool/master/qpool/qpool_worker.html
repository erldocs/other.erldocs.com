qpool_worker

Generic module for worker process.

Generic module for worker process.

It passes calls from queue to the CallbackModule which has qpool_worker behaviour.

Behaviour callbacks are:

init(WorkerArgs :: term()) -> tuple('ok', term()).

run(Task :: term(), From :: pid(), State :: term()) -> tuple(Result :: term(), StateNew :: term()).

terminate(Reason :: term(), State :: term()) -> ok.

code_change(OldVsn :: term(), State :: term(), Extra :: term()) -> tuple(ok, StateNew :: term()).

update_config(Config :: term(), State :: term()) -> tuple(ok, StateNew :: term()).
