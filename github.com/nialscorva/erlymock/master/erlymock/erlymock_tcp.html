open(Options::[Option]) -> {ok, Socket::client_socket()}

  • Option = {port, Port::integer()} | {socket_options, GenTCPOptions}

Opens a socket that will be mocked on the other end. Note that only one socket can be opened in any test. socket options are defined in the gen_tcp page and will be applied to both sides of the connection. The only caveat is that both sides will be set {active,false}. The mock side will be active once erlymock:replay() is called. The port is optional, if one is not provided then the system will assign one.
