base32

Functions

b32(V::integer()) -> [base32char()]

returns base32 character corresponding to V like 1 -> 'B', 31 -> '7'. V is integer from 0 to 31

See also: unb32/1.

encode(Bin::string()) -> base32string()

returns base32 encoded string of String

See also: decode/1.

unb32(String::base32char()) -> integer()

A=unb32(b32(A))

See also: b32/1.

decode(Bin::base32string()) -> string()

returns base32 decoded string of String

See also: encode/1.
