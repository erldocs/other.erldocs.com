start() -> {ok, pid()} | {error, Reason}

Start gen_server

stop() -> ok

Stop gen_server

reply(S::string) -> ok

Get reply from serial port. Function used as callback function for ar_serial, when data received from port. Function should't be used manually!

mode(M::Mode) -> ok | {ok, Id} | {error, daemon_locked}

  • Id = integer()
  • Mode = file | command

Set gen_server internal mode. In "file" mode, "send" function calls allowed only from process that set's "file" mode In "command" mode, "send" function calls allowed from any process
