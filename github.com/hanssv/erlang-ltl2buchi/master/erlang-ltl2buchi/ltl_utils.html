normalize(Phi::ltl_formula()) -> ltl_formula()

Normalize an LTL expression. Normalize (in effect sort nested ands and ors). Used in order to avoid a more costly equivalence check in some algorithms.

subformulas(X::ltl_formula()) -> [ltl_formula()]

The subformulas of an ltl_formula. Example: (X a) U b gives: [(X a) U b, X a, b, a]
