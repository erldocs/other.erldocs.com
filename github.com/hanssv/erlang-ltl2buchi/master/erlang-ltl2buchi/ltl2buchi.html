ltl2buchi

LTL-to-Büchi translation.

LTL-to-Büchi translation. The core translation algorithm implemented in this module is described in the report From States to Transitions: Improving translation of LTL formulae to Büchi automata by Giannakopoulou and Lerda.

Types

buchi_automaton()

Functions

translate(Phi::ltl_formula()) -> buchi_automaton()

Translate an LTL expression to a Büchi automaton.

translate_norew(Phi::ltl_formula()) -> buchi_automaton()

Translate an LTL expression to a Büchi automaton. Does not use the rewrite heuristics in ltl_rewrite.

See also: translate/1.

simplify(Phi::ltl_formula()) -> ltl_formula()

Simplify an LTL expression. Removes always- and eventually-expressions and replace them with their release and until equivalences.
