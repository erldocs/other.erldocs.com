ltl_parse

Parse LTL strings.

Parse LTL strings. A reference implementation of a parser for LTL expressions. Only implements the ltl_parse:string function that parses a string. Uses yecc for the actual parsning, see ltl_parser.yrl. Can be extended and/or replaced to suit special needs for an example of this, see McErlang

Examples 

  8> ltl_parse:string("G p").
  {always,{lprop,p}}
 
  9> ltl_parse:string("[] p").
  {always,{lprop,p}}
 
  10> ltl_parse:string("F (p | q)").
  {eventually,{lor,{lprop,p},{lprop,q}}}
 
  11> ltl_parse:string("<> (p | q)").
  {eventually,{lor,{lprop,p},{lprop,q}}}
 
  12> ltl_parse:string("<> (p | X q)").
  {eventually,{lor,{lprop,p},{next,{lprop,q}}}}
