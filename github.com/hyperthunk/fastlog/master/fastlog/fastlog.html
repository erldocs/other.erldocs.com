start() -> ok | term()

Starts the fastlog application

stop() -> term()

Stops the fastlog application

add_logger(Name) -> term()

Adds a named logger.

configure(AppName::atom()) -> ignored | [logger_spec()]

Configures the application defined by AppName. Relies on application:get_all_env/1 to get back the 'fastlog' configuration.

load_config(AppData::[term()]) -> ignored | [logger_spec()]

add_logger(Name, Config) -> term()

Adds a named logger. If Config is an atom, then it is set as the logging level, otherwise it is assumed to be a proplist containing fastlog config and is passed on verbatim.

remove_logger(Name::atom()) -> term()

Removed the named logger.
