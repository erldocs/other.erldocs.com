valid_propagation_envelope(ReqData::wm_reqdata(), Explain::bool(), AllowNone::bool()) -> {bool(), {string(), integer(), string()}, wm_reqdata()} | {bool(), wm_reqdata()}

Validates a webmachine #wm_reqdata() term for a Platformer query: Ensure that either:

There is a valid X-Platformer-Memo-Token request header, AND

There is a valid X-Platformer-Memo-Priority request header

OR: Neither of these headers is present. Returns true|false if valid and wm_reqdata() object -- if Explain is true, the wm_reqdata() will carry messages about missing/invalid headers, otherwise it is the original object passed in.

valid_propagation_envelope(ReqData) -> term()

Same as valid_propagation_envelope/2, just assumes true for Explain and AllowNone.

support_preflight(AllowedHeaders::list(), ReqData::wm_reqdata()) -> wm_reqdata()

Compares headers in a #wm_reqdata() term with the allowed AllowedHeaders, and inserts a helpful message in the response body. This is specifically for use in an options() implementation by a webmachine resource.
