hammy

This module implements hammy's public API.

This module implements hammy's public API. Note: Database handles obtained via open/1 and create/1 ARE NOT multi-process safe. This will be fixed in a later release.

Functions

open(FileName::list() | binary()) -> {ok, ref()} | {error, open_db}

Opens an existing database

create(FileName::list() | binary()) -> {ok, ref()} | {error, open_db}

Creates a database from scratch. If the database file already exists this will reset the database resulting in complete data loss.

open_or_create(FileName::list() | binary()) -> {ok, ref()} | {error, open_db}

Creates or opens an existing database, as needed. If the filename exists then it assumed to be a valid database file. If the file doesn't exist then a new database is created.

close(DB::ref()) -> ok

Closes the database. Aborts any pending writes.

put(DB::ref(), Key::binary(), Value::binary()) -> ok | {error, insert}

Inserts a new key/value pair into the database. Will overwrite existing entries if they exist.

get(DB::ref(), Key::binary()) -> {ok, binary()} | {error, notfound}

Retrieves an entry from the database.

del(DB::ref(), Key::binary()) -> ok

Deletes an entry from the database.
