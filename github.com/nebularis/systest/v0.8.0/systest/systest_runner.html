systest_runner

This module handles the execution of test runs, and provides a custom OTP behaviour that callback module can implement in order to provide support for different testing frameworks.

This module handles the execution of test runs, and provides a custom OTP behaviour that callback module can implement in order to provide support for different testing frameworks.
