c4_game_master

Coordinates the creation of games between pairs of players.

Coordinates the creation of games between pairs of players. It mostly sits in a loop receiving join requests. When the first join request comes, the player is told to wait for another. When the next one comes, the pair are put into a newly created game and the process repeats over and over.
