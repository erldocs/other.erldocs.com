Functions

format(Rrd_create::RrdRecord) -> RrdCommand::string()

  • RrdRecord = #rrd_create{} | #rrd_ds{} | #rrd_rra{}

Converts the data structure describing the rrd command to a string that can be executed by rrdtool.

create(File::string(), DSName::string(), Type) -> #rrd_create{}

  • Type = guage | counter | derive | absolute

Creates the #rrd_create{} command data structure for a data source called DSName, in a file named File. The data source Type determines how rrdtool will treat updates (see http://oss.oetiker.ch/rrdtool/ for more information).

steps(Unit::atom(), Step::integer()) -> Steps::integer()

  • Unit = day | week

Calculates the number seconds per Step given the number of Steps in a time period.

to_list(S::atom()) -> string()

Converts the given atom to an upper case string.
