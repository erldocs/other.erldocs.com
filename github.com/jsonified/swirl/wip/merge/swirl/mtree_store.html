mtree_store

This module implements storage API for merkle trees.

API for merkle hash tree

Types

bin() = integer() >= 0

hash() = binary()

Functions

init(Table::atom()) -> Table::atom()

Initialize a new ETS table. The table is named and public and elements are sorted as ordered set.

insert(Table::atom(), X2::{bin(), binary(), binary() | atom()}) -> true

Insert the tuple {bin number, hash, chunk} into the table. Note : this will replace any old object with a key same as new one.

insert_new(Table::atom(), X2::{bin(), hash(), binary()}) -> true | false

Insert the tuple {bin number, hash, chunk} if the bin number is not already in the tree.

lookup(Table::atom(), Bin::bin()) -> {ok, hash(), binary()} | {error, term()}

searches for a given Bin in the table.

get_first(Table::atom()) -> term() | '$end_of_table'

gets the first bin from the tree

is_member(Table::atom(), Bin::bin()) -> true | false

searches for a given Bin in the table.

highest_bin(Table::atom()) -> term() | '$end_of_table'

returns the highest bin number in the ets table, which will always be a leaf node.

delete(Table::atom(), Bin::bin()) -> true

Deletes an object with the given Bin from the table

table_to_file(Table::atom()) -> true | {error, term()}

Write the ets table to the disk and deletes the table from the current running erlang shell.

file_to_table(File_Name::string()) -> {ok, atom()} | {error, term()}

Loads the ets table from the given file.
