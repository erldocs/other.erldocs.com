gen_tcpd_handler_example_echo

Example echo client for gen_tcpd.

Example echo client for gen_tcpd. Module starts with start/1`. To start module from REPL (`12345 is an example port):

gen_tcpd_handler_example_echo:start(12345).

Functions

start(Port) -> term()

init(ConnPid::pid()) -> {ok, pid()}

Starts module.

ConnPid is a pid of gen_tcpd_conn instance.

See also: gen_tcpd_conn.

recv(State::pid(), Data::list()) -> {pid()}

Will be called when some data received by socket.

stop(State::any()) -> ok

Will be called on socket close or system shutdown.
