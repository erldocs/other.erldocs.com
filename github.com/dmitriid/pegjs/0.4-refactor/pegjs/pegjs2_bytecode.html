Conditions and Loops --------------------

[10] IF_ERROR t, f

[11] IF_NOT_ERROR t, f

[12] WHILE_NOT_ERROR b

Matching --------

[13] MATCH_ANY a, f, ...

[14] MATCH_STRING s, a, f, ...

[15] MATCH_STRING_IC s, a, f, ...

[16] MATCH_REGEXP r, a, f, ...

[17] ACCEPT_N n

[18] ACCEPT_STRING s
