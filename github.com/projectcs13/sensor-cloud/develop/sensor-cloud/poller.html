Function: code_change/3 Purpose: this funciton will called when the gen_server should update its internal state during a release upgrade/downgrade. Parameter: _OldVsn -- In the case of upgrade, the _OldVsn is the Vsn, in the case of downgrade, the _OldVsn is {down, Vsn}. Vsn is defined by the vsn attribute(s) of the old version of the callback module Module. If no such attribute is defined, the version is the checksum of the BEAM file. State -- The internal state of the gen_server. _Extra -- is passed as-is from the {advanced,Extra} part of the update instruction. Returns: {ok, State}
