parser this module implements functionalities to extract values from the data body, and stores them into datapoints. In the current design, we only care about the values in the key-value pairs, not the keys. The parsers in the parseJson and parseText functions are regular expressions. for instance, /sensor/temp/value indicates the json object: {'sensor': {'temp': {'value': 123}}}
