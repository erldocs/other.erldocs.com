create_converter(Record, CsvHeaders) -> Converter

  • Record = RecordName | FieldName
  • RecordName = atom()
  • FieldName = binary()
  • Converter = tuple()
  • CsvHeaders = [FieldName]

Converters allows convert data cvs file records to Erlang records. Record is the wanted format: #product{name = <<"Name">>, price = <<"Price">>}. CsvHeaders is usually a first record in a CSV file: [<<"Name">>, <<"Count">>, <<"Price">>].

Fields in the Record can be skipped or in the different with the file order.
