xapian_record

It contains helpers for extracting information from a document.

It contains helpers for extracting information from a document. It is used with MSet.

record(TupleName, TupleFields) -> Meta

  • TupleName = atom()
  • TupleFields = [atom()]
  • Meta = x_document_meta()

Create a term, that contains information about document fields. This information is used for building a record term for each document in the MSet. TupleFields is a list of special names for the record fields:

docid

data

all_values

all_terms

weight

rank

percent

collapse_key

collapse_count
