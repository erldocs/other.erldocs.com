open(PoolParams::[pool_param()], Path::iolist(), Params::[term()]) -> {ok, pid()} | {error, term}

Create a pool. For PoolParams see https://github.com/devinus/poolboy

name: the pool name

worker_module: the module that represents the workers

size: maximum pool size

max_overflow: maximum number of workers created if pool is empty

See also: xapian_server:open/3.

close(PoolName::atom() | pid()) -> term()

Destroy the pool asynchronously.

  • PoolNames = [atom() | pid()]
