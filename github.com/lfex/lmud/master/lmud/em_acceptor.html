em_acceptor

Accept new TCP connections.

Accept new TCP connections. This gen_server will wait in a gen_tcp:accept() call until someone makes a connection, then it will immediately launch a new em_conn server and hand over the new socket for further processing, before it returns with a timeout so that it'll go back into accept mode again.
