em_parser

In-game request handler for user sessions.

In-game request handler for user sessions. When a request is spawned with an MFA during in-game play, it points here, to the parse/2 function.

Types

cmd_ok() = {ok, req()}

req() =
            #req{conn = undefined | em_conn:conn_pid(),
                 user = undefined | em_user:user_pid(),
                 living = undefined | em_living:living_pid(),
                 queue = [init | {input, string()}],
                 handlers = [req_handler()]}

req_any() = req_done() | req_ok()

req_done() = {done, req()}

req_handler() = {atom(), atom(), list()}

req_ok() = {ok, req_handler(), req()}
