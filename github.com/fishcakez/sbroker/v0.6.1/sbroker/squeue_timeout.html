squeue_timeout

Implements a basic queue management algorithm where items are dropped once their sojourn time is greater than a timeout value.

Implements a basic queue management algorithm where items are dropped once their sojourn time is greater than a timeout value.

squeue_timeout can be used as the active queue management module in a squeue queue. It's argument is a non_neg_integer(), which is the timeout value, i.e. the minimum sojourn time at which items are dropped from the queue.
