send(Session::session(), Msg, Timeout) -> ok

  • Msg = term()
  • Timeout = timeout() | async | infinity

Delivers a message to a subscribed handler or stores until such handler subscribes. If no handler appears until session is terminated due to inactivity ALL SUCH MESSAGES ARE LOST! If Timeout is async then function returns immediately after the message is stored for the delivery. Otherwise function returns after the message is delivered to the handler or fails if it is not delivered in that amount of time.

recv(Session::session()) -> Tag

  • Tag = reference()

Subscribes calling process for a message delivery. The message will be asynchronously delivered to the caller message queue as {Tag, Msg}.
