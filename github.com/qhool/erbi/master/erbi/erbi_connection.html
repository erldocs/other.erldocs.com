prepare(Connection::erbi_connection(), Query::any()) -> {ok, erbi_statement()} | {error, any()}

Prepare query/statement for execution.

Parses statement and returns a handle which can be used to execute and retrieve rows.

prepare_cached(Connection::erbi_connection(), Query::any()) -> {ok, erbi_statement()} | {error, any()}

Prepare with cacheing.

This function is the same as prepare/2 except if the statement has been previously prepared on this connection, a cached statement handle may be returned.

selectall_list(Connection::erbi_connection(), Query::any(), BindValues::erbi_bind_values()) -> {ok, [[any()]]} | {error, any()}

Execute query with bind values and return all records as lists.

Results are returned as a list of lists. Each list contains the values, in order for that row. This is a convenience method, equivalent to: 

  Stmt = Connection:prepare( Query ),
  Result = Stmt:fetchall_list( BindValues ),
  Stmt:finish(),
  Result

driver_call(Connection::erbi_connection(), Function::atom(), Args::[any()]) -> any()

Call arbitrary driver function

This function calls an arbitrary function in the driver module. Similar to erbi:call_driver/3 and erbi_statement:call_driver/3, this version prepends the driver's connection object to the argument list you supply, and allows the driver to update its state information.

See the documentation for your driver for what functions are available and how to use them.
