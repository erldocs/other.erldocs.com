driver_call(Statement::erbi_statement(), Function::atom(), Args::[any()]) -> any()

Call arbitrary driver function

This function calls an arbitrary function in the driver module. Similar to erbi:call_driver/3 and erbi_connection:call_driver/3, this version prepends the driver's connection object and the statement handle to the argument list you supply, and allows the driver to update its state information.

See the documentation for your driver for what functions are available and how to use them.
