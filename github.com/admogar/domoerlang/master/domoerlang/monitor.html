monitor

Monitor of an automation component.

Monitor of an automation component. Checks its state and, if that's the case, sends orders to it.

Functions

start_link(GroupPid, SensorId) -> term()

Starts the monitor.

init(GroupPid, SensorId) -> term()

Inits or continues the monitor's execution.

configurar_padre(PidMonitor::pid(), PidGrupo::pid()) -> ok

Configures a monitor in order to send notifications to a group. Synchronous.

start(PidMonitor) -> term()

Changes the monitor state to started.

pause(PidMonitor) -> term()

Changes the monitor state to stopped.
