router

Este módulo implementa el router del juego erlwars.

Este módulo implementa el router del juego erlwars. El servidor web establecerá conexiones con el router para obtener la lista de partidas del juego, crear partidas y asociar clientes con partidas.

Types

unirse_a_partida_return_type() = {atom(), pid()}
                                       | {atom(), atom()}

Functions

router:start() -> atom()

Esta función inicia y registra el proceso router de erlwars. Valor de retorno

ok: Retorna siempre ?OK (predefinido al átomo ok).

router:unirse_a_partida(Nick::string(), NombrePartida::string()) -> unirse_a_partida_return_type() | atom()

Esta función permite a un cliente unirse a una partida enviando un mensaje al router. Si no existe ninguna partida con el nombre especificado se creará. Parámetros

Nick: Cadena de caracteres que contiene el nombre que utilizará el jugador en la partida.

NombrePartida: Nombre de la partida que se desea crear o a la que se desea unirse.

Valor de retorno El pid asociado a la partida o ?PARTIDACERRADA (en caso de que el estado de la partida no permita unirse en ese momento).

router:obtener_partidas() -> [string()]

Esta función obtiene el nombre y estado de las partidas existentes actualmente. Valor de retorno Una lista cuyos elementos son cadenas de caracteres. Cada elemento es de la forma "nombrePartida@Estado". El estado de una partida es: partidaNoDisponible | partidaDisponible.
