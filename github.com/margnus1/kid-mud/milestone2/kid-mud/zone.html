go(Zone::pid(), PlayerPID::pid(), Direction::north | south | east | west | login) -> ok

Looks if the direction is an valid exit. If the exit is invalid, tell the player that he cannot move that way. Otherwise return the exits ID and remove the player from the zone.

validate_target(Zone::pid(), PlayerPID::pid(), Target::string()) -> ok

Checks if Target is a valid target in the zone.

look(Zone::pid(), PlayerPID::pid()) -> ok

Return the look information of the zone to the player.

enter(Zone::pid(), PlayerPID::pid(), Name::string(), Direction::north | south | east | west | login) -> ok

Add the player to the zones player list and send the information of the zone to the player. Inform the other players in the zone of this change.

logout(Zone::pid(), PlayerPID::pid()) -> ok

Remove the player from the zone. If that was the last player in the zone the zone will shutdown, otherwise inform the other players in the zone of this change.

exits(Zone::pid(), PlayerPID::pid()) -> ok

Return the exits of the zone to the player.

kick(Zone::pid(), Name::string()) -> ok

Remove the player from the zone and send a kick message to that player. If that was the last player in the zone the zone will shutdown, otherwise inform the other players in the zone of this change.

attack(Zone::pid(), PlayerPID::pid(), Target::string(), Damage::integer()) -> ok

If Target is a valid target make an attack on it, otherwise tell player to stop attacking that Target. Inform the target of this attack. Inform all the players in the zone of this attack.

death(Zone::pid(), PlayerPID::pid()) -> ok

Remove the player from the zone. If that was the last player in the zone the zone will shutdown, otherwise inform the other players in the zone of this change.

say(Zone::pid(), PlayerPID::pid(), Message::string()) -> ok

Broadcasts the say Message to all the players in the zone.
