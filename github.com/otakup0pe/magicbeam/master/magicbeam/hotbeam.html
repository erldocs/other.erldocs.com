hotbeam

Monitors a subset of source/beam files for changes.

Monitors a subset of source/beam files for changes.

The hotbeam component of magicbeam will monitor a subset of modules depending on configuration. It will extract the source and beam file locations and pay attemtion to when these files are modified. If a file is modified and the requisite configuration option is set then hotbeam will recompile and or reload the file.
