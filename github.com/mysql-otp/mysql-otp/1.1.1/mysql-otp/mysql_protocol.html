mysql_protocol

This module implements parts of the MySQL client/server protocol.

This module implements parts of the MySQL client/server protocol.

The protocol is described in the document "MySQL Internals" which can be found under "MySQL Documentation: Expert Guides" on http://dev.mysql.com/.

TCP communication is not handled in this module. Most of the public functions take funs for data communitaction as parameters.
