mysql_encode

Functions for encoding a term as an SQL literal.

Functions for encoding a term as an SQL literal. This is not really part of the protocol; thus the separate module.

Functions

encode(Int::term()) -> iodata()

Encodes a term as an ANSI SQL literal so that it can be used to inside a query. In strings only single quotes (') are escaped. If backslash escapes are enabled for the connection, you should first use backslash_escape/1 to escape backslashes in strings.

backslash_escape(String) -> term()

Escapes backslashes with an extra backslash. This is necessary if backslash escapes are enabled in the session.
