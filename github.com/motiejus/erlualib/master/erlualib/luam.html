LuaM, extended function calling

This module is responsible for high-level function invocations in Lua.

## Erlang->Lua type conversion table >-------->------> +------------+----------------+ | Erlang | Lua | +------------+----------------+ | 'nil' | nil | boolean | boolean | | binary | string | | atom | string | | string | string | | number | number | | proplist | tagged table | | tuple | indexed table | +------------+----------------+

Whereas >--------->-----> +-----------------+------------+ | Lua | Erlang | +-----------------+------------+ | nil | nil | | boolean | boolean | | light_user_data | --NA-- | | number | float | | string | binary | | table | proplist | | function | --NA-- | | user_data | --NA-- | | thread. | --NA-- | +-----------------+------------+

## Table to proplist conversion rules Table key conversion rules are same like for any other data type.

## Function return value handling When function returns with one value, give it plain. When function returns more values, wrap them to tuple.

Functions

call(L::lua() (see module lua), FunName::string(), Args::[arg() (see module lua)]) -> ret() (see module lua)

Call FunName with Args. Return all arguments in tuple.

one_call(File::name() (see module file), FunName::string(), Args::[arg() (see module lua)]) -> ret() (see module lua) | no_return()

Call a function once. Initializes Lua, does luam:call, closes Lua

Steps: 1. Create a new Lua state 2. Open and load the file 3. luam:call/4 4. Close Lua state 5. Return the result of luam:call

pushterm(L::lua() (see module lua), Arg::arg() (see module lua)) -> ok

Push arbitrary variable on stack

fold(Fun, Acc0, L::lua() (see module lua), N::abs_index() (see module lua)) -> Acc1

  • Fun = (ret() (see module lua), ret() (see module lua), AccIn) -> AccOut
  • Acc0 = term()
  • Acc1 = term()
  • AccIn = term()
  • AccOut = term()

Call Fun over table on absolute index N. [-0, +0].

multipcall(L::lua() (see module lua), N::non_neg_integer()) -> non_neg_integer()

Call function and retrieve its all arguments. [-(N+1), +M]

Equivalent to lua:call(L, N, LUA_MULTRET), but returns how many arguments function returned. Pops function from the stack.

maybe_atom(L::lua() (see module lua), N::index() (see module lua)) -> {ok, atom()} | false

Return atom() if value on N is atom; false otherwise
