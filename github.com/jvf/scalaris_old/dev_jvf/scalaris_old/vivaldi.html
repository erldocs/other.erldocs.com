Functions

activate() -> ok

Activates the vivaldi process. If not activated, the vivaldi process will queue most messages without processing them.

deactivate() -> ok

Deactivates the cyclon process.

get_coordinate() -> ok

Sends a (local) message to the vivaldi process of the requesting process' group asking for the current coordinate and confidence. see on_active({get_coordinate, Pid}, State) and msg_get_coordinate_response/3
