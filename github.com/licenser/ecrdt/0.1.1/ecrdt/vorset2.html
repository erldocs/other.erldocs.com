vorset2

An Implementation of the OR Set (Observe Remove Set) CvRDT.

An Implementation of the OR Set (Observe Remove Set) CvRDT. It is optimized over the default implementation. Details about the optimization are based on the paper: http://www.cmi.ac.in/~madhavan/papers/mss-tr-2013.html
