vorsetg

This is an extention of the standard OR Set implementation that adds the GC interface and functionality as descrived here: http://blog.licenser.net/blog/2013/06/11/asyncronous-garbage-collection-with-crdts/.

Garbage collects a hash bucket from the OR set by removing it's values from both the removes and adds.

If the hash bucket was negotiated correctly this opperation can be performed asyncronously without problems.
