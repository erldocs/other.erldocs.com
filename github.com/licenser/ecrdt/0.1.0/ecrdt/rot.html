rot

This is not a general purpose data structure, it imeplments a set that is supposed to take mostly ordered data, in the form of {ID, Element}, where the ID ideally is some kind of timestamp.

This is not a general purpose data structure, it imeplments a set that is supposed to take mostly ordered data, in the form of {ID, Element}, where the ID ideally is some kind of timestamp.

The datatype is supposed to work as a tool for asyncronous garbage collection, with older entries slowly timing out with an agreement about what to delete.
