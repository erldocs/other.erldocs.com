start() -> ok

Starts eplugin.

wait_for_init() -> ok

Waits until eplugin has finished starting up and has loaded all plugins whose dependencies can be filled at startup.

Use this if you need to call a plugin hook during your application startup for one-off configuration or other similar uses.

callbacks(Plugin::atom()) -> {Module::atom(), Function::atom()}

Lists all registered and enabled callbacks for a given Plugin. Sorted by the priority they were given.

apply(Plugin::atom(), Args::[any()]) -> [any()]

An equivalten to erlang:apply/2 for callbacks. All enabled callbacks as listed by callbacks/1. Results are retunred in an list.

call(Plugin::atom()) -> [any()]

fold(Plugin::atom(), Acc0::any()) -> any()

This function folds Acc0 through all callbacks for Plugin as returned by callbacks/1. The result of function N is passed as argument to function N+1, N0 is passed Acc0.

config(Plugin::atom()) -> undefined | any()

Returns the configuration of a plugin or undefined when the plugin is not known.

disable(Plugin::atom()) -> {error, not_found} | ok

Disables a plugin by removing all callbacks it has registered. The callback eplugin:disable(Config) will be called on the Plugin after it was disabled to allow cleanup. And eplugin:disable_plugin(Plugin) will be called after the plugin was disabled to inform any other interested plugins.

enable(Plugin::atom()) -> {error, not_found} | ok

Enables a plugin by adding all callbacks it has registered. The callback eplugin:enable(Config) will be called on the Plugin befire it is enabled to allow initialisation. And eplugin:disable_plugin(Plugin) will be called after the plugin is enabled to inform any other interested plugins.

register(Plugin::atom(), Callback::atom(), Module::atom(), Function::atom(), Options::callback_config()) -> ok

Registers Module:Function of the plugin Plugin for Callback. This can be used to programatically hook into callbacks from outside of a plugin.

register(Plugin::atom(), Callback::atom(), Module::atom(), Function::atom()) -> ok

Calls register/5 with an default empty option set.
