add(Target::module(), Intercept::module(), Mapping::mapping()) -> ok

Add intercepts against the Target module.

Target - The module on which to intercept calls. E.g. hashtree.

Intercept - The module containing intercept definitions. E.g. hashtree_intercepts

Mapping - The mapping from target functions to intercept functions.

E.g. [{ {update_perform,2}, sleep_update_perform}]
