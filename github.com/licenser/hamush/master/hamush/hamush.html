hamush

This module contains the central interface functions for hamush.

This module contains the central interface functions for hamush. All modules/extensions or other code wanting to interact with objects in the virtual world should use the functions provided here to do so.

Functions

text(ID::string()) -> string()

Retreives a text from the text server.

content(ObjID::integer()) -> [] | [integer()]

Lists all objects that are contained ObjID.

exits(ObjID::integer()) -> [] | [ExitID::integer()]

Lists all exits in a location.

move_to(ObjID::integer(), RoomID::integer()) -> []

moves ObjID to RoomID.

location(ObjID::integer()) -> LocationID::integer()

Returns the location of ObjID

set(ObjID::integer(), Attr, Value) -> []

  • Attr = string() | atom()
  • Value = string() | integer()

Sets an Attr on an ObjID to Value. Attr is converted to uppercase, if Attr is an atom it is untouched otherwise it will converted to upercase.

fset(ObjID::integer(), Attr, Value) -> []

  • Attr = string() | atom()
  • Value = string() | integer()

Sets an Attr on an ObjID to Value, without touching Attr.

get(ObjID::integer(), Attr) -> string() | integer()

  • Attr = string() | atom()

Gets the Attr from ObjID. Unless Attr is atom it will be turned uppercase.

fget(ObjID::integer(), Attr) -> string()

  • Attr = string() | atom()

Gets on Attr from ObjID without modifying Attr.

name(ObjID::integer()) -> string()

Returns the name of ObjID or an empty string of none is exitsant.

create() -> term()

Equivalent to create(Type, Name, Location).

create(Name) -> term()

Equivalent to create(Type, Name, Location).

create(Type, Name) -> term()

Equivalent to create(Type, Name, Location).

create(Type, Name::string(), Location::integer()) -> ID::integer()

  • Type = thing | room | exit

creates a new object of Type with Name in Location

pemit(ObjID::integer(), Message::string()) -> []

Emits Message string to ObjID.

pemit(ObjID::integer(), Format::string(), Arguments::list()) -> []

Emits a message to ObjID using Format and Arguments as in io_lib:format/2.

remit_but(ObjID::integer(), But::integer(), Format::string(), Args::list()) -> []

Emits a message to all objects in ObjID but the object named But using Format and Args as in io_lib:format/2.

remit_but(ObjID::integer(), But::integer(), Message::string()) -> []

Emits a message to all objects in ObjID but the object named But.

remit(ObjID::integer(), Format::string(), Args::list()) -> []

Emits a message to all objects in ObjID using Format and Args as in io_lib:format/2.

remit(ObjID::integer(), Message::string()) -> []

Emits a Message to all objects in ObjID.

resolve(Observer::integer(), Lookup::string()) -> {ok, Pid::pid()}

Resolves the an string to the object Pid of the Object.

"me" returns the Observer.

"here" returns the location of the resolver.

"#<id>" returns the oject with id .

"<name>" returns the object with name in the Observers location.

"*<name>" a global lookup for name.

desc(ObjID::integer(), Observer::integer()) -> string()

Returns the desc of an object by evaling it's DESCRIPTION attribute.

eval_get(ObjID::integer(), Attribute::Attr, Actor::integer()) -> string()

  • Attr = string() | atom()

Gets Attribute from Object and evaluates it from the POV of Actor.

eval(Code::string(), Actor::integer()) -> string()

Evaluates Code from the point of view of Actor.
