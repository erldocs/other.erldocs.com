stateful_supervisor

A supervisor that also let's you handle an associated state.

A supervisor that also let's you handle an associated state. Hacky acknowledgements: 1. It uses the process dictionary to store the state 2. It uses the fact that supervisor is implemented over gen_server 3. It uses the fact that supervisor doesn't reply with hibernate or timeouts 4. Module::init_state/1 function will be called before Module::init/1 5. Module::terminate/1 function will be called before supervisor:terminate/1
