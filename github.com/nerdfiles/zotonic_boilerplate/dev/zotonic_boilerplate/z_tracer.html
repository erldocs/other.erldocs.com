z_tracer

Simple tool for ad-hoc tracing of Zotonic.

Simple tool for ad-hoc tracing of Zotonic. Since the module z_notifier, serves as an internal message router, tracing on specific calls of the module is enough to monitor the internals of zotonic. Trace messages are logged to stdout by default, but a custom callback function can be specified.
