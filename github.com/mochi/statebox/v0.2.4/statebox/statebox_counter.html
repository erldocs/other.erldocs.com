statebox_counter

Integer counter based on an ordered list of counter events.

Integer counter based on an ordered list of counter events.

A counter is stored as an orddict of counter events. Each counter event has a unique key based on the timestamp and some entropy, and it stores the delta from the inc operation. The value of a counter is the sum of all these deltas.

As an optimization, counter events older than a given age are coalesced to a single counter event with a key in the form of {timestamp(), 'acc'}.

Types

counter() = [counter_op()]

counter_id() = statebox_identity:entropy() | acc

counter_key() = {timestamp(), counter_id()}

counter_op() = {counter_key(), integer()}

op() = statebox:op()

timedelta() = statebox:timedelta()

timestamp() = statebox_clock:timestamp()

Functions

value(Rest::counter()) -> integer()

Return the value of the counter (the sum of all counter event deltas).

merge(Counters::[counter()]) -> counter()

Merge the given list of counters and return a new counter with the union of that history.

accumulate(Timestamp::timestamp(), Counter::counter()) -> counter()

Accumulate all counter events older than Timestamp to the key {Timestamp, acc}. If there is already an acc at or before Timestamp this is a no-op.

inc(Key::counter_key(), Value::integer(), Counter::counter()) -> counter()

Return a new counter with the given counter event. If there is an acc at or before the timestamp of the given key then this is a no-op.

f_inc_acc(Value::integer(), Age::timedelta()) -> op()

Equivalent to f_inc_acc(Value, Age, {statebox_clock:timestamp(), statebox_identity:entropy()}).

f_inc_acc(Value::integer(), Age::timedelta(), Key::counter_key()) -> op()

Return a statebox event to increment and accumulate the counter. Value is the delta, Age is the maximum age of counter events in milliseconds (this should be longer than the amount of time you expect your cluster to reach a consistent state), Key is the counter event key.
