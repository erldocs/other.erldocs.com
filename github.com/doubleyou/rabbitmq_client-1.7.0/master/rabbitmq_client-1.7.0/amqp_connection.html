amqp_connection

This module is responsible for maintaining a connection to an AMQP broker and manages channels within the connection.

This module is responsible for maintaining a connection to an AMQP broker and manages channels within the connection. This module is used to open and close connections to the broker as well as creating new channels within a connection. Each amqp_connection process maintains a mapping of the channels that were created by that connection process. Each resulting amqp_channel process is linked to the parent connection process.
