encrypt(Password) -> {HashedPassword, Salt, Type}

  • Password = str_or_bin()
  • HashedPassword = binary()
  • Salt = binary()
  • Type = binary()

Return hashed password, salt, and hash type. Uses bcrypt with the number of rounds as configured in the bcrypt application key default_log_rounds. It is an error to attempt to hash an empty password, but no other restrictions are placed on the password.

migrate_sha1(SHA, OrigSalt) -> term()

Upgrade stored hashed password data that was already hashed using OrigSalt and a SHA1 scheme. The resulting data will be assigned a hash type of <<"sha1+bcrypt">>. The returned salt value will be a compount salt delimited by \t with the first part being a newly assigned bcrypt generated salt and the second containing the original salt.
