route(Type, Req, Args) -> term()

Generate a search URL. Expects Args to be a proplist with a search_index key (the value of which can be either a binary or string). The organization in the URL will be determined from the Webmachine request. If organization does not exist in the request information, it will call the OSC route/3 function to generate URLs without an organization
