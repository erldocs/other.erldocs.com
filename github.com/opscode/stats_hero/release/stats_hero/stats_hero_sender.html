send(Pid::pid(), Payload::iolist()) -> ok | {error, term()}

Send Payload as a Stats Hero Protocol (SHP) version 1 UDP packet to the estatsd server configured for this sender. The input Should be the bare metrics without SHP header. This function will compute the length and add the SHP version and length header to the payload.
