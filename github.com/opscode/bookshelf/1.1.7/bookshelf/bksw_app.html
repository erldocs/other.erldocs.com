remsh_welcome() -> term()

Print an informative message about how to use a remote shell attached to a live bookshelf node. The idea is to call this from a wrapper script used to start a remote shell, like: erl -name user1@127.0.0.7 -setcookie bookshelf \ -remsh bookshelf@127.0.0.1 -s bksw_app remsh_welcome.
