Equivalent to load(YamlStream, []).

load(YamlStream::binary(), Opts::proplist() (see module proplist)) -> {ok, [doc() (see module yaml_compose)]} | {error, Reason::term()}

Convert a yaml binary to a list of documents. Options:

{schema, SchemaModule} - Select schema module. Default is yaml_schema_erlang.

implicit_atoms - For scalar values that look atom-ish, construct as atoms. (yaml_schema_erlang only)
