cb_apps_hndlr

A Cowboy REST handler that handles operations relating to a collection of ERTS 'app' resources.

A Cowboy REST handler that handles operations relating to a collection of ERTS 'app' resources.

When Erlang/OTP applications are considered as RESTful resources, the REST verbs associated with the collection are mapped as follows: 
   GET    : Get the configuration details of a specified application.
   POST   : Start the specified application.
Notes The handler exposes the erts_apps module. The current implementation uses the Cowboy http server. libraries.
