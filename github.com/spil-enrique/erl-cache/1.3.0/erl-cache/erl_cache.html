start_cache(Name::name(), Opts::cache_opts()) -> ok | {error, invalid_opt_error()}

Starts a cache server with the given name and default options. Returns a name in case the given name corresponds to a running cache server or any other process in the system

stop_cache(Name::name()) -> ok | {error, invalid_opt_error()}

Stops a cache server. Returs an error if the given name does not correspond to a running cache server
