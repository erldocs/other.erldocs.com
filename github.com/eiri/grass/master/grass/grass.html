Deletes the graph G. This function not only drops all the verticies on the graph, but also removes graph's directory and shutting down graph's server.

Returns a list of all verticies of the graph G. If the graph is empty, returns an empty list.

Returns a list of the verticies of graph G connected to the vertex V. If the vertex doesn't have the connections returns an empty list.

Creates an exact copy of the vertex V1 with name V2 in graph G. Duplicates all the edges connecting to V1.

modify_vertex(G::graph(), From::vertex(), To::vertex()) -> ok | {error, error()}

Convinience funtion. Creates an exact copy of the vertex V1 with name V2 in graph G. and deletes the vertex V1.

edge_exists(G::graph(), From::vertex(), To::vertex()) -> boolean()

Checks if an edge between verticies 'From' and 'To' exists in graph G.

add_edge(G::graph(), From::vertex(), To::vertex()) -> ok | {error, error()}

Adds edge between verticies 'From' and 'To' in graph G. Returns 'ok' even if the edge already exists.

del_edge(G::graph(), From::vertex(), To::vertex()) -> ok | {error, error()}

Deletes the edge between the verticies 'From' and 'To' in graph G. Returns 'ok' even if the edge doesn't exist.
