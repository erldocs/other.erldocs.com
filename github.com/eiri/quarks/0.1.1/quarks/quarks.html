make() -> [quark()] | [binary()]

Returns a list with a single quark in proplist format.

make(Num::integer()) -> [quark(), ...] | [binary(), ...]

Returns a list with a requested number of the quarks in proplist format.

make(Num::integer(), Opts::options()) -> [quark(), ...] | [preon(), ...] | [binary(), ...]

Returns a list with a requested number of the quarks generated with specified options. Options

Options is a proplist with following attributes:

format

In which form output the quarks.

Available formats: proplist, records, json, protobuf, msgpack, thrift

Default: proplist

preon

Return not a complete quark, but just one of its attributes

Available preons: signature, mark, side, dual, spin, weight, color, special, quote

callback

Function which will be applied to each generated quark

If no callback provided function returns a list of quarks in specified format.

stream() -> ok

Creates a stream of quarks in proplist format, printed to stdout with 1 seconds interval.

stream(Opts::options() | stop) -> ok

Creates a stream of quarks, where provided callback applied to each of the generated quarks. Atom 'stop' passed to the stream stops existing stream if it is in progress. Options

Options is a proplist with following attributes

format

In which form generate the quarks.

Available formats: proplist, records, json, protobuf, msgpack, thrift

Default: proplist

callback

Function which will be applied to each generated quark

Default: fun(Quark) -> io:format("~p", [Quark] end.

delay

Timeout between quarks generation.

If passed value is a tuple, delay will be a random number of milliseconds between given numbers.

Default: 200 milliseconds
