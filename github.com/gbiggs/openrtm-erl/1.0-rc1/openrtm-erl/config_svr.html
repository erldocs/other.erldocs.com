  • FileName = string()
  • Defaults = config()
  • Config = config()
  • Reason = any()

Initialise the configuration file server. Loads the specified configuration file, if possible. After loading a file, the defaults merged in. Any values not specified in the configuration will be filled in from the defaults, the rest will be left untouched. If FileName is empty, the defaults only will be used.
