  • Reason = any()

Deactivate the event handler that publishes to the terminal.

  • File = string()
  • Level = log_level()
  • Reason = any()

Activate an event handler that publishes to a file. Duplicate file publishers are possible; they are identified by their file name.

del_file_output(File) -> ok | {error, Reason}

  • File = string()
  • Reason = any()

Deactivate an event handler that publishes to a file. The file name must be provided to identify the handler.

  • HandlerRef = [atom() | {atom(), any()}]

Get the list of active event handlers.

log(Level, ModLine::{Module, Line, Pid}, String) -> ok

  • Level = log_level()
  • Module = atom()
  • Line = pos_integer()
  • Pid = pid()
  • String = string()

Log an event to the currently-active event handlers. The event is a tuple of the priority level, the module/line in which the event occured, and the event data. If the log manager is not active, the event will be dumped to standard error.

log(Level, ModLine::{Module, Line, Pid}, String, Data) -> ok

  • Level = log_level()
  • Module = atom()
  • Line = pos_integer()
  • Pid = pid()
  • String = string()
  • Data = [any()]

Log an event to the currently-active event handlers. The event is a tuple of the priority level, the module/line in which the event occured, a format string and data to be formatted. If the log manager is not active, the event will be dumped to standard error.
