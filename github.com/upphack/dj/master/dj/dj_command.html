run(Command, Dir, Options) -> Result

  • Command = command()
  • Dir = directory()
  • Options = command_options()
  • Result = run_command_return()

run runs Command in Dir, ignoring Options.

run is equivalent to in (immediate succession) calling run_async(C, D, O), gather_output(P) and erlang:port_close(P) with the obtained port.

returns: see gather_output/1

run_async(Command, Dir, Options) -> port()

  • Command = command()
  • Dir = directory()
  • Options = command_options()

run_async runs Command in Dir, ignoring Options.

returns: the port opened for Command.
