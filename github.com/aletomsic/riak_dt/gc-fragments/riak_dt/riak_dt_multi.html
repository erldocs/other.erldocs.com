riak_dt_multi

a multi CRDT holder.

a multi CRDT holder. A Document-ish thing. Consists of two elements, a Schema and a value list. The schema is an OR-Set of {name, type} tuples that identify a field and it's type (type must be a CRDT module, yes, even this one.) The value list is a dict of {name, type} -> CRDT value mappings. The value list may have it's CRDT value removed as soon as the key is removed from the keyset. The Keyset requries consensus for GC.
