Stateless module that given the needed settings opens a port or starts a server based on the os environmental variable "EMBEDDED_ENV". When EMBEDDED_ENV is set to the value of "sim" the simulated environment will be used otherwise if it is omitted or the value "hard" is set the real hardware will be used.
