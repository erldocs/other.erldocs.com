This module formats erlang dates in the form {{Year, Month, Day}, {Hour, Minute, Second}} to printable strings, using (almost) equivalent formatting rules as http://uk.php.net/date

erlang has no concept of timezone so the following formats are not implemented: B e I O P T Z formats c and r will also differ slightly

See tests at bottom for examples

Types

date() = {year(), month(), day()}

datetime() = {date(), time()}

day() = 1..31

hour() = 0..23

minute() = 0..59

month() = 1..12

now() = {integer(), integer(), integer()}

second() = 0..59

time() = {hour(), minute(), second()}

year() = integer() >= 0
