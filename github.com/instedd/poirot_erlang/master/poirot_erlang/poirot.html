proxied_log(Source, LogEntry) -> term()

Adds a single log entry as if it was sent by another source. This is useful for cases in which the source of the events is not able to connect directly with the poirot receiver.

LogEntry is a proplist with the following keys:

level message timestamp

[optional] pid [optional] tags [optional] fields (additional metadata)
