delete_schema_and_stop() -> term()

Delete the schema, but stay stopped.

prepare_rename() -> term()

Back up the database in preparation for a node rename.

rename_node(From, To) -> term()

Rename a node. Assumes there is only one node. Leaves Mnesia stopped with no schema and a backup installed as a fallback. Finish renaming the node and start Mnesia back up and you should have all your data back. If for some reason you need to go back, you could install the pre_rename backup as fallback and start Mnesia.

start() -> term()

Start Mnesia, creating a new schema if we don't already have one.
