ekafka

Main interface to the EKafka module.

Main interface to the EKafka module.

Functions

start() -> ok | {error, term()}

Start the EKafka application. This function is provided as a convenience so that ekafka may be started on the command line using the -s flag: erl -s ekafka

new_connection() -> {ok, pid()} | {error, term()}

Create a new connection to a Kafka broker.

See also: ekafka_connection.

get_connections() -> [pid()]

Returns a list of connections to the Kafka brokers. A connection should remain valid as long as the ekafka application is running.

See also: ekafka_connection.
