gen_smtp_server

The gen_smtp_server is given a list of tcp listener configurations. You'll typically only want to listen on one port so your options will be a single-item list containing a proplist. e.g.: [ [{port,25},{protocol,tcp},{domain,"myserver.com"},{address,,{0,0,0,0}}] ] by providing additional configurations the server will listen on multiple ports over either tcp or ssl on any given port. e.g.: [ [{port,25},{protocol,tcp},{domain,"myserver.com"},{address,{0,0,0,0}}], [{port,465},{protocol,ssl},{domain,"secure.myserver.com"},{address,{0.0.0.0}}] ]
