etrie

etrie implements a trie, for storing strings with associated values.

etrie implements a trie, for storing strings with associated values.

Functions

find(String::string(), Trie::trie()) -> {ok, Value::any()} | error

Returns {ok, Value} if String was found in Trie, and error otherwise.

new() -> trie()

Returns a new trie.

similar(String::string(), MaxEdits::integer(), Trie::trie()) -> [match()]

Returns a list of matches whose strings require at most MaxEdits edits to equal String. Valid edits are delete, insert, substitute, and transpose. All edits have equal weight.

store(String::string(), Value::any(), Trie::trie()) -> trie()

Returns a new trie after storing String and its associated Value into Trie. If String already exists in Trie, then Value will overwrite the previous value associated with String.
