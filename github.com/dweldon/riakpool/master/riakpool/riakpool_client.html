riakpool_client

riakpool_client is a collection of convenience functions for using riakpool.

riakpool_client is a collection of convenience functions for using riakpool.

Functions

delete(Bucket::binary(), Key::binary()) -> ok

Delete Key from Bucket.

get(Bucket::binary(), Key::binary()) -> {ok, binary()} | {error, any()}

Returns the value associated with Key in Bucket as {ok, binary()}. If an error was encountered or the value was not present, returns {error, any()}.

list_keys(Bucket::binary()) -> {ok, list()} | {error, any()}

Returns the list of keys in Bucket as {ok, list()}. If an error was encountered, returns {error, any()}.

post(Bucket::binary(), Key::binary(), Value::binary()) -> ok

Associates Key with Value in Bucket without checking if Key already exists in Bucket.

put(Bucket::binary(), Key::binary(), Value::binary()) -> ok

Associates Key with Value in Bucket. If Key already exists in Bucket, an update will be preformed.
