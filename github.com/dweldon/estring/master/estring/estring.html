estring is a string manipulation library.

estring is a string manipulation library.

Functions

begins_with(String::string(), SubString::string()) -> bool()

Returns true if String begins with SubString, and false otherwise. 

  > estring:begins_with("fancy pants", "fancy").
  true

contains(String::string(), SubString::string()) -> bool()

Returns true if String contains SubString, and false otherwise. 

  > estring:contains("los angeles", "angel").
  true

edit_distance(String1::string(), String2::string()) -> integer()

Returns the damerau-levenshtein edit distance between String1 and String2. Note the comparison is case sensitive. 

  > estring:edit_distance("theater", "theatre").
  1

edit_distance(String1::string(), String2::string(), IgnoreCase::bool()) -> integer()

Returns the damerau-levenshtein edit distance between String1 and String2. The comparison is case insensitive if IgnoreCase is true. 

  > estring:edit_distance("receive", "RECIEVE", true).
  1
  > estring:edit_distance("Cats", "cast", false).
  2

ends_with(String::string(), SubString::string()) -> bool()

Returns true if String ends with SubString, and false otherwise. 

  > estring:ends_with("fancy pants", "pants").
  true

format(Format::string(), Data::list()) -> string()

Shortcut for lists:flatten(io_lib:format(Format, Data)). 

  > estring:format("~w bottles of ~s on the wall", [99, "beer"]).
  "99 bottles of beer on the wall"

is_integer(String::string()) -> bool()

Returns true if String is a string representation of an integer, and false otherwise. 

  > estring:is_integer("35").
  true
  > estring:is_integer("35.4").
  false

random(N::integer()) -> string()

Returns a random alphanumeric string of length N. 

  > estring:random(32).
  "LzahJub1KOMS0U66mdXHtHyMMXIdxv1t"

rot13(String::string()) -> string()

Applies the rot13 substitution cipher to String. 

  > estring:rot13("The Quick Brown Fox Jumps Over The Lazy Dog.").
  "Gur Dhvpx Oebja Sbk Whzcf Bire Gur Ynml Qbt."

similarity(Source::string(), Target::string()) -> float()

Returns a score between 0 and 1, representing how similar Source is to Target based on the edit distance and normalized by the length of Target. Note the order of Source and Target matters, and the comparison is case sensitive. 

  > estring:similarity("yahoo", "boohoo").
  0.5
  > estring:similarity("boohoo", "yahoo").
  0.4

similarity(Source::string(), Target::string(), IgnoreCase::bool()) -> float()

Returns a score between 0 and 1, representing how similar Source is to Target based on the edit distance and normalized by the length of Target. Note the order of Source and Target matters. The comparison is case insensitive if IgnoreCase is true. 

  > estring:similarity("linux", "Linux", true).
  1.0

similarity(Source::string(), Target::string(), IgnoreCase::bool(), LowerLimit::float()) -> {ok, float()} | {error, limit_reached}

Returns a score between 0 and 1, representing how similar Source is to Target based on the edit distance and normalized by the length of Target. Note the order of Source and Target matters. The comparison is case insensitive if IgnoreCase is true. A simple heuristic is used to estimate the upper bound for similarity between Source and Target. If the estimate is less than LowerLimit, then {error, limit_reached} is returned immediately. otherwise {ok, float()} or {error, limit_reached} is returned based on a call to similarity/3. 

  > estring:similarity("linux", "microsoft", false, 0.5).
  {error,limit_reached}

squeeze(String::string()) -> string()

Shortcut for estring:squeeze(String, " "). 

  > estring:squeeze("i need   a  squeeze!").
  "i need a squeeze!"

squeeze(String::string(), Char::char()) -> string()

Returns a string where runs of Char are replaced with a single Char. 

  > estring:squeeze("the cow says moooo", $o).
  "the cow says mo"
  > estring:squeeze("the cow says moooo", "o").
  "the cow says mo"

strip(String::string()) -> string()

Returns a string where leading and trailing whitespace (" ",\n\t\f\r) has been removed. Note that string:strip/1 only removes spaces. 

  > estring:strip("\t  clean me   \r\n").
  "clean me"

strip_split(String::string(), SeparatorString::string()) -> list()

Shortcut for re:split(estring:strip(String), SeparatorString, [{return, list}]). This is intended for parsing input like csv files. 

  > estring:strip_split("first>,<second>,<third\r\n", ">,<").
  ["first","second","third"]
