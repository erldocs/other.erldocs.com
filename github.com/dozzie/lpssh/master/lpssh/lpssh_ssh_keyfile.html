lpssh_ssh_keyfile

Callback module for SSH key handling.

Callback module for SSH key handling. This module is responsible for loading private RSA/DSA key, finding host's public key and adding host's key to known hosts list.

Functions

private_host_dsa_key(Type::system | user | {remoteuser, User}, Opts::list()) -> {ok, SSHKey} | {error, Reason}

Load host's private DSA key (not used for user authentication).

private_host_rsa_key(Type::system | user | {remoteuser, User}, Opts::list()) -> {ok, SSHKey} | {error, Reason}

Load host's private RSA key (not used for user authentication).

lookup_host_key(Host::string(), Alg::string(), Opts::list()) -> {ok, SSHKey::term()} | {error, Reason}

Find and read host's public key from known_hosts file.

Alg is either "ssh-dss" or "ssh-rsa".

Function should return {error,not_found} when no public key for the host was found.

add_host_key(Host::string(), Key::term(), Opts::list()) -> ok | {error, Reason}

Add host's public key to known_hosts file.
