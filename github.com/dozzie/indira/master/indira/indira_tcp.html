indira_tcp

TCP listener entry point.

TCP listener entry point. This listener is implemented as gen_indira_sock_stream module (listener and connection handler separated). It should be simple enough to serve as an example for your own code, if needed.

Indira parameter This module expects a tuple {Host,Port} as a parameter (see indira). The Host part can be:

string()

inet:ip_address() (i.e. {N1,N2,N3,N4} for IPv4)

'any' to indicate no binding to any particular interface
