Syslog log handler for error_logger.

Syslog log handler for error_logger.

Usage 
     error_logger:add_report_handler(indira_log_syslog_h, [Ident, Facility]).
     gen_event:add_handler(error_logger, indira_log_syslog_h, [Ident, Facility]).
Arguments to the module (as in gen_event:add_handler/3):

[Ident, Facility], where Ident is atom or string and Facility is indira_syslog:facility()

[Ident, Facility, {Host, Port}], as above, with Host being inet:hostname() | inet:ip_address() and Port being integer() | default (as in indira_syslog:open_remote/2
