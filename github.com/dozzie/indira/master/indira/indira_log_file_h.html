indira_log_file_h

Flat file log handler for error_logger.

Flat file log handler for error_logger.

Usage 
     error_logger:add_report_handler(indira_log_file_h, [Ident, File]).
     gen_event:add_handler(error_logger, indira_log_file_h, [Ident, File]).
Arguments to the module (as in gen_event:add_handler/3):

[Ident, File], where Ident is atom or string and File is path (string)

 NOTE: This module is a clone of indira_log_syslog_h. Any changes to code in communication section should be made first there.
