xmerlrpc_xml

Parsing and forming XML-RPC messages for xmerlrpc.

Parsing and forming XML-RPC messages for xmerlrpc.

Module uses data structures after jsx application, which in turn uses EEP-18 structures.

Types

jsx_array() = [jsx_value()]

jsx_hash() = [{}] | [{jsx_hash_key(), jsx_value()}, ...]

jsx_hash_key() = string() | binary() | atom()

jsx_scalar() = binary() | number() | null | true | false

jsx_value() = jsx_hash() | jsx_array() | jsx_scalar()

optlist() = [{atom(), term()} | atom()]

proc_arg() = jsx_value()

proc_name() = string() | binary() | atom()

xml_node() = #xmlElement{} | #xmlText{}

xmlrpc_exception() = {Code :: integer(), Message :: string()}

xmlrpc_request() = {proc_name(), [proc_arg()]}

xmlrpc_result() = jsx_value()
