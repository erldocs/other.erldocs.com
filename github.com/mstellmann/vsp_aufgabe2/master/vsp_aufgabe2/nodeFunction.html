wakeup(EdgeOrddict, OwnNodeName) -> term()

This function is called from the sleeping state of the node. It searches for the akmg and sends the connect msg to the node on the opposite side of the akmg. Afterwards the node changes into the sleeping state and waits for a response from the other node.

test(OwnEdgeOrddict, OwnLevel, OwnNodeState, OwnFragname, OwnNodeName, FindCount, InBranch, BestWT) -> term()

This function search for a new edge and sends a test-message if a new basic edge is found.

returns: {ok, TestEdge, OwnNodeState} if no new edge is found {ok, NewTestEdge, OwnNodeState} if a test-edge is found

report(TestEdge, FindCount, OwnNodeState, InBranch, BestWT) -> term()

This function sends a report-message on the InBranch. It reports the best-weight found so far.

returns: {ok, TestEdge, OwnNodeState} if nothing is there to report {ok, TestEdge, found} if a report was send, to switch into the found state
