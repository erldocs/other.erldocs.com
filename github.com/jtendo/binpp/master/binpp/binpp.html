pprint(Bin::binary() | bitstring(), Opts::opts()) -> ok | any()

Print a binary using custom function or return the formatted result. Valid option is one of:

{return, binary}

{return, iolist}

{printer, CustomFunction}

Custom printers should be of arity 1, accepting the prettified result as an iolist() input. Returned value can be any().
