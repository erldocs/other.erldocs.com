start() -> ok | {error, Reason::any()}

Starts the whole applications

start_link() -> {ok, pid()}

Starts the main guard server. The server is signleton, locally registered under enreconn_srv name.

server() -> {ok, pid()}

Returns pid of the guard server process or rises no_process error if no such process.
