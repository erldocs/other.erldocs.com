gpio

GPIO interface.

GPIO interface

Created: 2012 by Magnus Feuer

Types

direction() = in | out | high | low

level() = none
                | info
                | notice
                | warning
                | error
                | critical
                | alert
                | emergency

posix() = atom()

style() = none | rising | falling | both

uint1() = 0..1

unsigned() = integer() >= 0
