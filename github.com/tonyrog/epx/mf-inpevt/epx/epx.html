pixmap_copy_area(Src::epx_pixmap(), Dst::epx_pixmap(), XSrc::integer(), YSrc::integer(), XDst::integer(), YDst::integer(), Width::unsigned(), Height::unsigned()) -> void()

Copy pixels from the area (XSrc,YSrc,Width,Height) in Src pixmap to the area (XDst,YDst,Width,Height) in the Dst pixmap. The pixels are clipped according to the Dst clip rectangle.

pixmap_copy_area(Src::epx_pixmap(), Dst::epx_pixmap(), XSrc::integer(), YSrc::integer(), XDst::integer(), YDst::integer(), Width::unsigned(), Height::unsigned(), Flags::epx_flags()) -> void()

Copy pixels from the area (XSrc,YSrc,Width,Height) in Src pixmap to the area (XDst,YDst,Width,Height) in the Dst pixmap. The pixels are clipped according to the Dst clip rectangle. The pixels in Src are mixed with Dst according to the flags in Flags.

pixmap_alpha_area(Src::epx_pixmap(), Dst::epx_pixmap(), Alpha::fix8(), XSrc::integer(), YSrc::integer(), XDst::integer(), YDst::integer(), Width::unsigned(), Height::unsigned()) -> void()

Blend Src rectangle (XSrc,YSrc,Width,Height) with Dst rectangle (XDst,YDst,Width,Height) using a fixed alpha value of Alpha. If Alpha is 1.0 it means Src pixels only and an Alpha of 0.0 means using Dst pixels only.
The blending formula used is: (Alpha*(Src-Dst)+(Dst bsl 8)) bsr 8.

pixmap_fade_area(Src::epx_pixmap(), Dst::epx_pixmap(), Fade::fix8(), XSrc::integer(), YSrc::integer(), XDst::integer(), YDst::integer(), Width::unsigned(), Height::unsigned()) -> void()

Blend Src rectangle (XSrc,YSrc,Width,Heght) to Dst rectangle (XDst,YDst,Width,Height) fade using Fade as a blending scale factor.
