Example CANopen application.

Example CANopen application.

Required minimum API:

handle_call - get

handle_call - set

handle_info - notify

Functions

start(CoSerial) -> {ok, Pid} | ignore | {error, Error}

Starts the server.

stop() -> ok | {error, Error}

Stops the server.

set(Pid, X2, NewValue) -> term()

get(Pid, X2) -> term()

write_begin(Pid, X2, Ref) -> term()

write(Pid, Ref, Data) -> term()

write_end(Pid, Ref, N) -> term()

read_begin(Pid, X2, Ref) -> term()

read(Pid, Bytes, Ref) -> term()

abort(Pid, Ref) -> term()

handle_call(Request, From, LoopData) -> {reply, Reply, LoopData} | {reply, Reply, LoopData, Timeout} | {noreply, LoopData} | {noreply, LoopData, Timeout} | {stop, Reason, Reply, LoopData} | {stop, Reason, LoopData}

  • Request = {get, Index, SubIndex} | {set, Index, SubInd, Value}
  • LoopData = term()
  • Index = integer()
  • SubInd = integer()
  • Value = term()

Handling call messages. Required to at least handle get and set requests as specified above. Handling all non call/cast messages. Required to at least handle a notify msg as specified above.
RemoteId = Id of remote CANnode initiating the msg.
Index = Index in Object Dictionary
SubInd = Sub index in Object Disctionary
Value = Any value the node chooses to send.

handle_info(Info, LoopData) -> {noreply, LoopData} | {noreply, LoopData, Timeout} | {stop, Reason, LoopData}

  • Info = {notify, RemoteId, Index, SubInd, Value}
  • LoopData = term()
  • RemoteId = integer()
  • Index = integer()
  • SubInd = integer()
  • Value = term()

Handling all non call/cast messages. Required to at least handle a notify msg as specified above.
RemoteId = Id of remote CANnode initiating the msg.
Index = Index in Object Dictionary
SubInd = Sub index in Object Disctionary
Value = Any value the node chooses to send.

get_entry(Pid, Index::{Index, SubInd}) -> {entry, Entry::record()} | false

Returns the data structure for {Index, SubInd}. Should if possible be implemented without process context switch.
