depsolver_culprit

This app does the culprit search for a failed solve.

This app does the culprit search for a failed solve. It searches throw the goals provided by the user trying to find the first one that fails. It then returns that as the culprit along with the unknown apps from the goal, the version constrained apps from the goal, and the good apps (those not immediately constrained from the goals).
