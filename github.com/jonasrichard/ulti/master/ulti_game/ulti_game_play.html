Functions

beat(Card::card(), OtherCard::card()) -> boolean()

It is true if the first card beats the second card.

beat_low10(Card::card(), OtherCard::card()) -> boolean()

It is true if the first card beats the second one and the 10 is low (betli, szintelen durchmars).
