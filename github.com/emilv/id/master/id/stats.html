stats

En nyckel-värde-struktur.

En nyckel-värde-struktur. Implementerad som en länkad lista. Strukturen är omuterbar så ett nytt stats-objekt returneras av varje funktion.

Types

stats()

Functions

new() -> stats()

Skapa ett tomt stats-objekt

set(Key::term(), Value::term(), Stats::stats()) -> stats()

Sätt nyckeln Key till värdet Value i Stats

set(Values::[tuple(term(), term())], Stats::stats()) -> stats()

För varje nyckel-värde-par i Values, sätt nyckeln till motsvarande värde i Stats.

get(Key, Stats) -> term()

Tupeln {Key, Value} där Value är värdet för Key i Stats. false om Key saknas i Stats.
