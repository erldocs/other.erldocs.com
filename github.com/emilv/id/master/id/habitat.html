habitat

Huvudmodulen för en simulation.

Huvudmodulen för en simulation. Genom att köra start/3 påbörjas en simulation. En simulation består av en world med data om miljön (temperatur, mat, ...) samt ett antal platypus-objekt (kallade djur). Det är här nya djur i simulationen skapas och det är den här modulen som håller koll på alla levande djur.

Functions

start(N::integer(), FoodGrowth::integer(), Temperature::integer()) -> pid()

Startar en simulation med N individer, mattillväxten FoodGrowth samt temperaturen Temperature Returnerar pid till skapad simulation

create_animal(Name) -> term()

Skapar nytt djur i en befintlig simulation

create_animal(Stats, Name) -> term()

Som create_animal/1 men med definierade egenskaper i form av ett stats-objekt.

remove_animal(Pid, Name) -> term()

Tar bort djuret Pid ur simulationen Name.

random_animal(Name) -> term()

Välj slumpmässigt ett av djuren i habitet

list(Name) -> term()

Returnerar en lista med stats från djuren i habitatet Name

world(Name) -> term()

Returnerar stats från simulationens värld

change(Name, Temperature, FoodGrowth) -> term()

Ändra omgivningens temperatur och mattillväxt

step(Name) -> term()

Kör simulationen ett steg.

step(Name, N) -> term()

Kör simulationen N steg.
