sgte_parse

.

Parses a template file or string and returns the compiled template.

This module is not meant to be used directly. It's called through the interface of the sgte module.

Types

compiled() = [char() | token()]

encoding() = latin1 | utf8 | {utf16, little} | {utf16, big} | {utf32, little} | {utf32, big}

gettext_tuple() = {Key, LineNo}

parsed() = {ok, token(), Line::int(), Remaining::template()}

predicate() = function(template())

rules() = [rule()]

template() = string() | binary()

Functions

gettext_strings(T::template()) -> [gettext_tuple()]

Extracts from template T the list of gettext keys with associated line numbers. This is a utility function to use in cojunction with gettext to create initial .po files.
