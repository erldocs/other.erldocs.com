Functions

encode(IntToBcd::integer(), Size::integer()) -> binary()

Pack the digits of an integer as BCD in a given size of binary. The pad with leading zeros. example: 47> bcd:encode(19,1). <<25>> 48>

decode(BcdBinary::binary(), Size::integer()) -> integer()

Unpack the given size of BCD binary into an integer strip leading zeros. example: 48> bcd:decode(<<25>>,1). 19 49>
