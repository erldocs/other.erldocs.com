bit_operations

This module contains functions for bit manipulations.

This module contains functions for bit manipulations.

Types

bit() = 0 | 1

bitfield_mask() = 0..255

bitfield_value() = 0..255

data() = 0..255

Functions

bit_test(Byte::data(), Bit::bit()) -> 0 | 1

Test a bit bitfield_value on a specified position inside the byte.

bit_list_get(Byte::data()) -> [bit()]

Give the list of bits of the given byte.

bit_set(Byte::data(), Bit::bit()) -> data()

Set the bit on the specified position in the byte.

bit_set(Byte::data(), Value::bitfield_value(), Mask::bitfield_mask()) -> data()

Set or clear number of bits in a byte. The number of bits are specified by the Value, the bit positions specified by the Mask. The bit=1 in the Mask means what bit must set/clear.

bit_set(Byte::data(), BitFieldValue::bitfield_value(), BitFieldMask::bitfield_mask(), X4::doShiftValueBeforeSet) -> data()

This is the same function what bit_set/3 is, but here the input Value does not shifted to the right bit positions, the must do before set the required valus of the bits.

bit_clear(Byte::data(), Bit::bit()) -> data()

Clear the bit on the specified position in the byte.

bit_get(Byte::data(), Mask::bitfield_mask()) -> bitfield_value()

Filter out bits in a byte by the bitfield_mask.

bit_get(Byte::data(), BitFieldMask::bitfield_mask(), X3::doShiftValueAfterGet) -> bitfield_value()

This is the same function what bit_get/2 is, but here the Value given by bit_get/2 must shift left till the first bit in the byte.

bit_toggle(Byte::data(), Bit::bit()) -> data()

This function will toggle the specified bit in the byte.
