vec3

vec3 can be used to represent 3D points and vectors.

vec3 can be used to represent 3D points and vectors. ----------------------------------------------------------------------------

Types

vec3() = #vec3{x=undefined | float(), y=undefined | float(), z=undefined | float()}

Functions

length(ThisVec::vec3()) -> float()

Returns the length of the vector.

length_squared(ThisVec::vec3()) -> float()

Returns the length of the vector, squared.

distance(OtherVec::vec3(), ThisVec::vec3()) -> float()

Return the distance between the ends of this and another vector. This is symantically treating the vector like a point.

distance_squared(OtherVec::vec3(), ThisVec::vec3()) -> float()

Return the distance between the ends of this and another vector, squared. This is symantically treating the vector like a point.

fuzzy_zero(ThisVec::vec3()) -> bool()

Returns true if every element in the vector is approximately zero, +/- SIMD_EPSILON. This is more useful than comparing to zero due to float precision errors.

add(OtherVec::vec3(), ThisVec::vec3()) -> vec3()

Add a vector to this one.

subtract(OtherVec::vec3(), ThisVec::vec3()) -> vec3()

Subtract OtherVec from ThisVec.

multiply(Scalar::float()) -> vec3()

Scale the vector.

divide(Scalar::float()) -> vec3()

Inversely scale the vector.

equals(OtherVec::vec3(), ThisVec::vec3()) -> bool()

Returns true if OtherVec equals ThisVec.

dot(OtherVec::vec3(), ThisVec::vec3()) -> float()

Return the dot product.

normalized(ThisVec::vec3()) -> vec3()

Return a normalized version of this vector.

rotate(Axis::vec3(), Angle::float(), ThisVec::vec3()) -> vec3()

Return a rotated version of this vector.

angle(OtherVec::vec3(), ThisVec::vec3()) -> float()

Return the angle between this and another vector.

absolute(ThisVec::vec3()) -> vec3()

Return the angle between this and another vector.

cross(OtherVec::vec3(), ThisVec::vec3()) -> vec3()

Return the cross product between ThisVec and OtherVec.

lerp(OtherVec::vec3(), Ratio::float(), ThisVec::vec3()) -> vec3()

Return the linear interpolation between this and another vector. Ratio is the ratio between ThisVec and OtherVec (i.e. When Ratio is 0, return ThisVec, when Ration is 1 return OtherVec).

set_max(OtherVec::vec3(), ThisVec::vec3()) -> vec3()

Set each element to the max of the values of ThisVec and the values of OtherVec.

set_min(OtherVec::vec3(), ThisVec::vec3()) -> vec3()

Set each element to the min of the values of ThisVec and the values of OtherVec.
