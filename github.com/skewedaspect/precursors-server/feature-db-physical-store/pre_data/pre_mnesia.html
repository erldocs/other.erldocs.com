The bridge between pre_data and an mnesia database.

The bridge between pre_data and an mnesia database. Mnesia should already be started because it's an app and not up to this module to do it.

check_setup(Persist::boolean()) -> ok | {error, any()}

Ensure the mnesia schema and tables exist. This may stop mnesia to create a schema on disk if 'true' is passed. Any other value and mnesia does not perist the data, meaning once mnesia is stopped, it will be lost
