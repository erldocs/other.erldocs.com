pre_account

Convenience wrapper for working with accounts.

Convenience wrapper for working with accounts. Should become more useful once we add things like player mail and other account management features. For now, it's just wrappers around pre_data calls.

-------------------------------------------------------------------------------------------------------------------

Functions

authenticate(AccountName::binary(), Password::binary()) -> ok | {error, term()}

Authenticates an account, given the account name, and password. Account name, in this case, will be the email address.

get_by_email(EmailAddress::binary()) -> {ok, tuple()}

Gets an account by email address. Returns an account.

get_by_id(AccountID::binary()) -> {ok, tuple()}

Gets an account by id. Returns an account.

create(Email::binary(), RealName::binary(), NickName::binary(), Password::binary()) -> {ok, tuple()} | {error, term()}

Creates a new account. Returns the newly created account.

delete(AccountID::any()) -> ok

Removed an account.
