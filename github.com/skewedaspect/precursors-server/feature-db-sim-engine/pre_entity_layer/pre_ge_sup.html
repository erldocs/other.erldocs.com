pre_ge_sup

Supervisor for gen_event managers.

Supervisor for gen_event managers. This is a named supervisor, so only one can be running at a time. --------------------------------------------------------------------------------------------------------------------

Functions

start_link() -> {ok, pid()}

Same as start_link(5).

See also: start_link/1.

start_link(N::non_neg_integer()) -> {ok, pid()}

Start the named supervisor pre_ge_sup linked to the calling process. N is the number of gen_event managers to run. This is a simple_one_for_one supervisor with a restart strategy of transient. This means that all of the children are basically the same, and if they exit with reason normal or shutdown they are not restarted.
