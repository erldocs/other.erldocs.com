start_link(N::non_neg_integer()) -> {ok, pid()}

Start the named supervisor pre_sim_worker_sup linked to the calling process. N is the number of worker processes to run. This is a simple_one_for_one supervisor with a restart strategy of transient. This means that all of the children are basically the same, and if they exit with reason normal or shutdown they are not restarted.
