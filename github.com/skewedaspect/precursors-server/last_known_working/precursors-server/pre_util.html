Miscellaneous utilities, intended for use by included apps, others.

Miscellaneous utilities, intended for use by included apps, others.

Functions

start_app(App) -> term()

Start an application and all of it's required dependencies. This simply tries to start the application, and if it fails, it catches the error and if the error is an unstarted dependency, it attempts to start that application.

start_apps(Rest) -> term()

Starts a list of applications (generally plugins). (Calls start_app under the hood.)
