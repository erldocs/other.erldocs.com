pre_entity_balancer

Load balances pre_gen_entity assignment among gen_event managers.

Load balances pre_gen_entity assignment among gen_event managers. Uses pre_ge_sup to start and supervisor the managers. This will supervisor the entities and recover them if they fail. It will also monitor the gen_event managers and attempt to move the entities it was handling to a new one.

The load strategy is to always add an entity to the gen_event manager with the fewest entities assigned to it. If there is a tie, then one is chosen arbitrarily.

Also allows sending the same event to all gen_event managers.
