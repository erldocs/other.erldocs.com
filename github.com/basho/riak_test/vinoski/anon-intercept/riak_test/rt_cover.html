rt_cover

Code to interact with cover, Erlang's code coverage tool.

Code to interact with cover, Erlang's code coverage tool. Cover needs to be started in the main node once and on each remote node every time it is started. It should be stopped on remote nodes before stopping them to make sure they have flushed all coverage data. In the end, we want to write coverage analysis to the file system. Here we do it as html files for easy browsing.
