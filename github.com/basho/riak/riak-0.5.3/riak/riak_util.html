Functions

moment() -> integer()

Get the current "moment". Current implementation is the number of seconds from year 0 to now, universal time, in the gregorian calendar.

compare_dates(A::string(), B::string()) -> boolean()

Compare two RFC1123 date strings. Return true if date A is later than date B.

make_tmp_dir() -> string()

Create a unique directory in /tmp. Returns the path to the new directory.

is_x_deleted(Obj::riak_object() (see module riak_object)) -> boolean()

'true' if all contents of the input object are marked as deleted; 'false' otherwise

obj_not_deleted(Obj::riak_object() (see module riak_object)) -> undefined | riak_object() (see module riak_object)

Determine whether all contents of an object are marked as deleted. Return is the atom 'undefined' if all contents are marked deleted, or the input Obj if any of them are not.

integer_to_list(Integer::integer(), Base::integer()) -> string()

Convert an integer to its string representation in the given base. Bases 2-62 are supported.

unique_id_62() -> string()

Create a random identifying integer, returning its string representation in base 62.

reload_all(Module::atom()) -> [{purge_response(), load_file_response()}]

Ask each member node of the riak ring to reload the given Module. Return is a list of the results of code:purge/1 and code:load_file/1 on each node.

try_cast(Cmd::term(), Msg::term(), Targets::[{Index::term(), Node::node()}]) -> {[{Index::term(), Node::node(), Node::node()}], [{Index::term(), Node::node()}]}

Cast {Cmd, {Index,Node}, Msg} at riak_vnode_master on Node if Node responds 'pong' to a net_adm:ping. The list of successful casts is the first element of the return tuple, and the list of pang-responding nodes is the second element. Used in riak_put_fsm and riak_get_fsm.

fallback(Cmd::term(), Msg::term(), Pangs::[{Index::term(), Node::node()}], Fallbacks::[{any(), Fallback::node()}]) -> [{Index::term(), Node::node(), Fallback::node()}]

Cast {Cmd, {Index,Node}, Msg} at a node in the Fallbacks list for each node in the Pangs list. Pangs should have come from the second element of the response tuple of a call to try_cast/3. Used in riak_put_fsm and riak_get_fsm

mkclientid(RemoteNode::term()) -> ClientID::list()

Create a unique-enough id for vclock clients.
