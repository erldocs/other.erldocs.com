riak_claim

The default functions used for claiming partition ownership.

The default functions used for claiming partition ownership. Generally, a wants_claim function should return either {yes, Integer} or 'no' where Integer is the number of additional partitions wanted by this node. A choose_claim function should return a riak_ring with one more partition claimed by this node than in the input ring.
