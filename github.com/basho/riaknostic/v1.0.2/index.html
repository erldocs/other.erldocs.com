riaknostic_check
Enforces a common API among all diagnostic modules and provides some automation around their execution.

Diagnostic that checks permissions on data directories and whether noatime is set.

riaknostic_check_dumps
Diagnostic that detects the existence of Erlang-generated crash dumps.

riaknostic_check_memory_use
Diagnostic that checks Riak's current memory usage.

riaknostic_check_nodes_connected
Diagnostic check that detects cluster members that are down.

riaknostic_check_ring_membership
Diagnostic that checks whether the local node is a member of the ring.

riaknostic_check_ring_preflists
Diagnostic that checks the local ring for any preflists that do not satisfy n_val.

riaknostic_check_ring_size
Diagnostic that compares the configured ring_creation_size to the actual size of the ring.

riaknostic_check_search
Diagnostic that checks if riak_search is enabled on every node.

riaknostic_config
Provides convenient access to Riak configuration values.

riaknostic_node
Functions that help diagnostics interact with the local Riak node or other members of the cluster.

riaknostic_util
