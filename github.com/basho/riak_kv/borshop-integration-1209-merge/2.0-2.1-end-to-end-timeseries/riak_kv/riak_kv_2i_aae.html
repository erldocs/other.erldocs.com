start(Partitions::[integer()], Speed::integer()) -> {ok, pid()} | {error, term()}

Starts a process that will issue a repair for each partition in the list, sequentially. The Speed parameter controls how fast we go. 100 means full speed, 50 means to do a unit of work and pause for the duration of that unit to achieve a 50% duty cycle, etc. If a single partition is requested, this function will block until that partition has been blocked or that fails to notify the user immediately.
