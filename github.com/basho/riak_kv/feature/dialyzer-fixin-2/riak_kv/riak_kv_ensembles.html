riak_kv_ensembles

This gen_server is reponsible for bootstrapping consensus ensembles used by riak_kv to provide strong consistency.

This gen_server is reponsible for bootstrapping consensus ensembles used by riak_kv to provide strong consistency. The server polls the ring periodically and registers any missing ensembles with the riak_ensemble_manager.
