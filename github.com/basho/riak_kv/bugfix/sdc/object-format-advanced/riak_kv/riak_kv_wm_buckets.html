riak_kv_wm_buckets

Resource for listing Riak buckets over HTTP.

Resource for listing Riak buckets over HTTP.

Available operations:

GET /types/Type/buckets?buckets=true (with bucket-type) GET /types/Type/buckets?buckets=stream GET /buckets?buckets=true (NEW) GET /buckets?buckets=stream GET /Prefix?buckets=true Get information about available buckets. Note that generating the bucket list is expensive, so we require the "buckets=true" or "buckets=stream" arg.
