riak_cs_copy_object

Object copying Note that Riak CS does not verify checksum in case of copy specified with range copy with "x-amz-copy-source-range".

Object copying Note that Riak CS does not verify checksum in case of copy specified with range copy with "x-amz-copy-source-range". Nor in the case where the source object was uploaded via multipart upload.
