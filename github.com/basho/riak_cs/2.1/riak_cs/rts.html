find_samples(Puller::function(), Start::datetime(), End::datetime(), Period::integer()) -> {Samples::[mochijson2()], Errors::[{slice(), Reason::term()}]}

Fetch all of the samples from riak that overlap the specified time period for the given key postfix.

This implementation reads each slice object from riak, and does the extraction/etc. on the client side. It would be a a trivial modification to do this via MapReduce instead.
