status(Timeout) -> term()

Find out what the archiver is up to. Should return {ok, State, Props} where State is idle or archiving if the archiver had time to respond, or busy if the request timed out. Props will include additional details about what the archiver is up to. May also return {error, Reason} if something else went wrong.
