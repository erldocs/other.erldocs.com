sum_user(RcPid::riak_client(), User::string()) -> {ok, [{string(), integer()}]} | {error, term()}

Sum the number of bytes stored in active files in all of the given user's directories. The result is a list of pairs of {BucketName, Bytes}.

sum_bucket(BucketName::binary()) -> {struct, [{binary(), integer()}]} | {error, term()}

Sum the number of bytes stored in active files in the named bucket. This assumes that the bucket exists; there will be no difference in output between a non-existent bucket and an empty one.

The result is a mochijson structure with two fields: Objects, which is the number of objects that were counted in the bucket, and Bytes, which is the total size of all of those objects.

archive_period() -> {ok, integer()} | {error, term()}

Retreive the number of seconds that should elapse between archivings of storage stats. This setting is controlled by the storage_archive_period environment variable of the riak_cs application.
