http_child_spec(Ref::any(), NbAcceptors::non_neg_integer(), TransOpts::any(), ProtoOpts::any()) -> child_spec() (see module supervisor)

Return an http child spec suitable for embedding.

When you want to embed cowboy in another application, you can use this function to create an http ChildSpec suitable for use in a supervisor. The parameters are the same as in start_listener/6 but rather than hooking the listener to the Ranch internal supervisor, it just returns the spec.
