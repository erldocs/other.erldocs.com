accept(LSock::socket()) -> {ok, socket()} | {error, any()}

Accepts an incoming connection request on a listen socket. Socket must be a socket returned from listen/3. This function blocks indefinitely until a client connection is established.

accept(LSock::socket(), Timeout::integer() | infinity) -> {ok, socket()} | {error, any()}

Accepts an incoming connection request on a listen socket. Socket must be a socket returned from listen/3. Timeout specifies a timeout value in ms.

async_accept(LSock::socket()) -> {ok, Ref::integer()} | {error, any()}

An asynchronous version of accept/1. The caller will receive {inet_async, LSock, Ref::integer(), {ok, CliSocket::socket()}} or {inet_async, ListSock, Ref, {error, Reason}} upon successful/unsuccessful client connection. The function needs to be called again in order to begin asynchronous accept of the next client socket. When a successful socket message {ok, CliSocket::socket()} is received, the server must call set_sockopt/2 function in order to register the socket with the inet driver.

set_sockopt(LSock::socket(), Sock::socket()) -> ok | {error, any()}
