Functions

init(OTS_ACCESS_ID::string(), OTS_ACCESS_KEY::string()) -> ok

Initialize OTS access information.

The provided access id and access key are stored in an ets table named as "erlaly_ots" which is the module name. Make sure this name is not used elsewhere. As the ets table is global, erlaly_ots does not support requests from different access ids simultaneously.
