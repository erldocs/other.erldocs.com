new(CacheName::atom(), Options::proplist() (see module proplists)) -> {ok, Pid::pid()}

Start a new cache

Available options:

max_keys - a positive integer or infinity

eviction_fun - a {M, F, A} which accepts a #trove_entry record and returns a bool

eviction_type - per_key or per_table

lookup_fun - a {M, F, A} that takes a key and returns the value associated with it.

per\_key eviction is checked every time a key is accessed per\_table eviction runs off a timer (every 15 minutes by default) See trove\_eviction.erl for some "out of the box" eviction functions

See below for the default values.

new(CacheName::atom()) -> {ok, Pid::pid()}

Start a new cache with default options.

Equivalent to trove:new(CacheName, [{max_keys, infinity}, {eviction_fun, {trove_eviction, timeout, []}}, {eviction_type, per_key}, {lookup_fun, {trove, default_lookup, []}]).
