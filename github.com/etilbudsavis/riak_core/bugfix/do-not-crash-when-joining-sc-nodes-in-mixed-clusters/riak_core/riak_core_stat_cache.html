riak_core_stat_cache

fetches stats for registered modules and stores them in an ets backed cache.

fetches stats for registered modules and stores them in an ets backed cache. Only ever allows one process at a time to calculate stats. Will always serve the stats that are in the cache. Adds a stat {stat_mod_ts, timestamp()} to the stats returned from get_stats/1 which is the time those stats were calculated.
