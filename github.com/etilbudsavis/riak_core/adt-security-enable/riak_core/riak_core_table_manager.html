riak_core_table_manager

A gen_server process that creates and serves as heir to a ETS table; and coordinates with matching user processes.

A gen_server process that creates and serves as heir to a ETS table; and coordinates with matching user processes. If a user process exits, this server will inherit the ETS table and hand it back to the user process when it restarts.

For theory of operation, please see the web page: http://steve.vinoski.net/blog/2011/03/23/dont-lose-your-ets-tables/
