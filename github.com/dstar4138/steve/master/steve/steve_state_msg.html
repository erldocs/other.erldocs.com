Functions

broadcast(Msg) -> term()

Broadcast a message to ALL friends/peers. To limit who to not send to use the forward/2 function.

forward(ExcludedFriend, Msg) -> term()

Similar to broadcast, but will forward to all friends/peers except the excluded friend. This is for requests that we can't personally fill.

handle_query(X1, State) -> term()

Handle a query and respond.

handle_note(Type, Cnt, Steve_state) -> term()

Handle a notification message and update state.

handle_papim(Friend, Msg, State) -> term()

Checks if the message is a repeat and discards if yes, otherwise it will verify it's not a forwarded message coming back. If it was, then we should forward it back to the person in our forward table. If its a normal message process it.
