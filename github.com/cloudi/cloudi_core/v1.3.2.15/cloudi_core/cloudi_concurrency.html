count(I::pos_integer() | float()) -> pos_integer()

A count to be used for the number of service processes used. An integer is an absolute count while a floating point number is a multiplier on the number of CPU cores available. A floating point number is rounded (or floor-ed) to be close to the CPU core count, to avoid extremes.
