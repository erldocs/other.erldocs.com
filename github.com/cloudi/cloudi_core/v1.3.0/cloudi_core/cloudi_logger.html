error(Process::atom(), Module::atom(), Line::integer(), Format::string(), Args::list()) -> ok

Error log message. which indicates a subsystem has failed but the failure is not fatal. Called with ?LOG_ERROR(Format, []).

warn(Process::atom(), Module::atom(), Line::integer(), Format::string(), Args::list()) -> ok

Warning log message. which indicates an unexpected occurance was found in a subsystem. Called with ?LOG_WARNING(Format, []).

info(Process::atom(), Module::atom(), Line::integer(), Format::string(), Args::list()) -> ok

debug(Process::atom(), Module::atom(), Line::integer(), Format::string(), Args::list()) -> ok
