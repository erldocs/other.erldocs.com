cloudi_trans_id

CloudI Transaction ID Usage Convenience functions for using the v1 UUID used as the transaction id ("TransId" or "trans_id") that follows a CloudI service request for the lifetime (defined by the service request timeout) of the request and its response result.

CloudI Transaction ID Usage Convenience functions for using the v1 UUID used as the transaction id ("TransId" or "trans_id") that follows a CloudI service request for the lifetime (defined by the service request timeout) of the request and its response result.

Functions

datetime(TransId::uuid() (see module uuid)) -> string()

Provide an ISO8601 datetime in UTC based on the time stored in the transaction id.

datetime(TransId::uuid() (see module uuid), MicroSecondsOffset::integer()) -> string()

Provide an ISO8601 datetime in UTC based on the time stored in the transaction id with an offset in microseconds.

from_string(String::string() | binary()) -> uuid() (see module uuid)

Return a transaction id as a binary UUID (16 bytes).

increment(TransId::uuid() (see module uuid)) -> uuid() (see module uuid)

Increment the transaction id while preserving uniqueness. Increment the v1 UUID's clock_seq (14bit) integer for 16384 possible v1 UUID values based on a single transaction id.

to_string(TransId::uuid() (see module uuid)) -> string()

Return the transaction id in a standard string format.

to_string(TransId::uuid() (see module uuid), Format::standard | nodash | list_standard | list_nodash | binary_standard | binary_nodash) -> string() | binary()

Return the transaction id in a string format.
