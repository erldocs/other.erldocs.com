Root supervisor of deck36_amqp_client application.

Root supervisor of deck36_amqp_client application.

start_link/1 starts all setups, consumers and producers that are defined in the application configuration using their supervisors. Setups will be started before consumers and producers are started so all needed entities are set up before trying to use them.

stop/0 stops all setups, consumers and producers. Stop them actively or you might end up getting errors from the amqp_client and none of the setups will tear down the changes they made.
