expect_ets(Tab, K::key(), Value, Tries) -> ok | {error, timeout} | {unexpected, term()}

  • Tab = tid() (see module ets)
  • Value = term()
  • Tries = non_neg_integer()

Expect a certain term to be written to the given ets Tab.

This function will do a sleep-spin-wait Tries times. It will sleep for 1 ms between tries, so you could interpret it as a rough timeout in ms.

You can either specify V to be a predicate function or a term that has to match the written value.
