k6_bytea

Types

bytea()

Functions

count() -> integer()

Returns the number of currently allocated byte arrays.

new(Size::integer()) -> bytea()

Creates a new byte array and returns it.

delete(Bytea::bytea()) -> ok

Frees a byte array immediately. It is no longer valid for use by this module; any attempts will result in badarg.

size(Bytea::bytea()) -> integer()

Returns the size of the byte array in bytes.

get(Bytea::bytea(), From::integer(), Len::integer()) -> binary()

Gets Len bytes from the byte array, starting at From. If the slice specified exceeds any boundaries, badarg results.

set(Bytea::bytea(), From::integer(), Value::binary()) -> ok

Replaces the substring of byte array, starting at From, with Value. If the slice so specified exceeds boundaries of the byte array, badarg results.
