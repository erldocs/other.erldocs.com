enb_basic_filters

This module contains functions used to filter messages.

This module contains functions used to filter messages. Every function's first argument must be #message{}. Other arguments are appended in an {M, F, A} style call.

Functions

all_filter(_Message::#message{}) -> boolean()

Returns true for any message, therefore allowing it through the filter. Used to subscribe to all messages.

command_filter(Message::#message{}, Command::atom()) -> boolean()

Used to filter a specific IRC command. The command should be in atom form compatible with enb_message_parser.

bot_command_filter(Message::#message{}) -> boolean()

Allows through messages that are a command for the bot.
