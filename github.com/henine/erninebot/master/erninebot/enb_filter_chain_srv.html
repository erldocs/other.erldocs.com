enb_filter_chain_srv

Filter chain server is a gen_server, that can be called with a message and returns either the message or the atom 'filtered', if one of the filters in the chain returns false.

Filter chain server is a gen_server, that can be called with a message and returns either the message or the atom 'filtered', if one of the filters in the chain returns false.

Functions

start_link(Name::atom(), ChainSpec::[mfa()]) -> {ok, Pid::pid()} | ignore | {error, Reason::term()}

Starts the server and returns its pid.

The Name argument is the name of the server. The ChainSpec argument is a list of {M, F, A} tuples used to call filter functions. The message being filtered is prepended to A.
