register_element(TargetName::string(), TargetConf::[]) -> ok | {error, Reason::term()}

Register a new element.

TargetName is an arbitrary string() wich identify the target.

TargetConf is a list of property:

MANDATORY:

{ip_address, Ip} when Ip is a string representation of an ip,

{snmp_version, V} when V = "1" | "2c" | "3",

OPTIONAL:

{ip_version, IpV} when IpV = "v4" | "v6", default to "v4"

{security_level, SecLevel} when SecLevel = "authPriv" | "authNoPriv" | "noAuthNoPriv", default to "noAuthNoPriv",

{port, Port} when Port is an integer. Default to 161.

{retries, Retries} when Retries is an integer. Number of retries when timeout occur. Default to 1.

{timeout, Timeout} when Timeout is an integer. Set the time in milliseconds after wich the manager will return {error, timeout}. Default is 2000 (2 seconds).

{community, Community} when community is a string. Default to public.

{priv_proto, PrivProto} when PrivProto = "AES" | "AES192" | "AES356" | "DES" | "3DES" (default to "AES"),

{auto_proto, AuthProto} when AuthProto = "MD5" | "SHA" (default to "SHA")

{priv_key, Key} and {auth_key, Key} when key must be a string of minimum 8 characters. Defaults are the string "undefined".

MANDATORY if version = "3":

{engine_id, EngineId} An hex string representing the engine Id of the target It can be found using {ok, EngineId} = snmpman:discovery/3

{security_name, Name} when Name is non empty string. Name is the name of the USM user to use for the agent.

walk_table(Target::string(), Oids::[Oid::string()]) -> {ok, Reply::term()} | {error, Reason::term()} | {report, Report::string}

Walk a table and return the rows specified in the Oids in the form of a tuple.

Example returning table rows from the mib2 ifTable. We want to get ifName, ifTypes and ifSpeed for all interfaces: 
 walk_table("myTarget", [
   "1.3.6.1.2.1.2.2.1.2",
   "1.3.6.1.2.1.2.2.1.3",
   "1.3.6.1.2.1.2.2.1.5"
   ]).
 {ok,[{varbinds,[{table_row,"sis0",6,100000000},
               {table_row,"lo0",24,0},
               {table_row,"pflog0",1,0},
               {table_row,"enc0",1,0},
               {table_row,"bridge0",209,0},
               {table_row,"sis2",6,0},
               {table_row,"sis1",6,100000000}]}]}
