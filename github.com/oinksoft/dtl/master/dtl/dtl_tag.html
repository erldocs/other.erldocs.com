dtl_tag

Tags are functions that return callbacks which in turn return nodelists (see dtl_node).

Tags are functions that return callbacks which in turn return nodelists (see dtl_node). They are used to transform templates in ways that filters cannot because they can consume tokens, whereas filters only replace the explicitly filtered content (see dtl_filter).
