macaddr

Cross platform MAC address determination.

Cross platform MAC address determination.

Inspired by the macaddr gem for ruby (http://rubyforge.org/projects/codeforpeople)

Tested on Linux (x86_64), Windows Vista, and MacOSX(10.5.4)

Works for:
* /sbin/ifconfig
* /bin/ifconfig
* ifconfig
* ipconfig /all 

   To return the first MAC address on the system:
     macaddr:address()
   
   To return an array of all MAC addresses:
      macaddr:address_list
