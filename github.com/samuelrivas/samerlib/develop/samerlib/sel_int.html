extended_euclid(A::integer(), B::integer()) -> {integer(), integer()}

Returns {X, Y} such that X*A + Y*B == gcd(A, B)

This function is used by gcd/2, and by mod_inv/2 to efficiently find the greatest common divisor and the modulus inverse of integer numbers

mod_inv(N::integer(), Mod::pos_integer()) -> pos_integer()

sqrt(N::pos_integer()) -> {Low::pos_integer(), High::pos_integer()}

Returns the closest integers to the square root of N

Low is the highest number such that Low * Low =< N

High is the lowest number such that High * High >= N

Note that if Low is equal to High then they are the exact square root of N
