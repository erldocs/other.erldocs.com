format(Fmt::string(), Args::[term()]) -> string()

Like io_lib:format/2, but flattening the result

Favour this function over io_lib's only when you really need a flat result, in most situations io_lib's deep strings are perfectly fine and is not worth paying the performance penalty of flattening.

parse_hex(String::string()) -> binary()

Read a binary from its hexadecimal representation

For example: 

  parse_hex("00FF") -> <<0,255>>

Hexadecimal characters can be either in upper case or lower case

format_byte(Byte::byte()) -> string()

Equivalent to format_byte(Byte, upper).

format_byte(Byte::byte(), LetterCase::letter_case()) -> string()

Create a string with the hexadecimal representation of a byte

Characters from a to f can be either in upper case or lower case depending on the value of LetterCase

format_hex(B::binary()) -> [string()]

Equivalent to format_hex(B, upper).

format_hex(B::binary(), Case::letter_case()) -> [string()]

Output the hexadecimal formatting of a binary

For example: 

  > io:format("~s~n", [sel_string:format_hex(<<"Hello world">>)]).
