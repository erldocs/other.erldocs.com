sel_async_queue

Simple asynchronous FIFO queue.

Simple asynchronous FIFO queue

Pushing is non-blocking, but popping will block if the queue is empty.

Types

async_queue()

invalid_queue_error() = {invalid_queue, async_queue()}

Functions

new() -> async_queue()

Returns an empty queue

push(Q::async_queue(), Item::any()) -> ok

Adds a new element to the queue

pop(Q::async_queue()) -> any()

Removes the oldest element from the queue and returns it

This function blocks if the queue is empty until a new element is pushed into it. At that moment, that element will be returned.

destroy(Q::async_queue()) -> ok

Stops the process that holds the resources for the queue

In the current implementation, this call will not release processes waiting for the results of a pop/1. This behaviour is subject to change in the future if there is a need for it
