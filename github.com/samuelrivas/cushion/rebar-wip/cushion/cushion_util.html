cushion_util

Common, useful functions.

Common, useful functions

Don't use this module out of Cushion! Most of these functions are here just temporarily. If they prove they usefulness,they'll graduate to a more universal utilities module.

Functions

start_app(App::atom()) -> [atom()]

Starts an application and all its dependencies.

Returns the list of actually started applications, in starting order.

stop_apps(Apps::[atom()]) -> ok

Stops all the applications in Apps in reverse order

Useful to stop the applications started by start_app/1

format(Format::string(), Args::[term()]) -> string()

The same as io_lib:format/2 but flattening the result.

Don't use this if you can use the deep list returned by io_lib:format, it is more efficient than flattening each intermediate result this function does.

untuple(What::Tuple | term()) -> term()

  • Tuple = {ok, Value} | {error, Reason}
  • Value = term()
  • Reason = term()

Unwraps ok-tuples and throws error-tuples. It lets through any other term.

get_value(Key, List::[{key(), value()}]) -> value()

Looks for the value associated to a key in a tuple list.

get_value(Key::any(), List::[{key(), value()}], Default::term()) -> value()

Looks for the value associated to a key in a tuple list, returning a default value if key is not found.

app_modules(App::atom()) -> [atom()]

Return the list of modules of an application.

The application app file must be in the code path and have the complete list of modules.

unicode_to_binary(L::[int()]) -> binary()

Converts a list of unicode characters into a utf-8 binary.

The main difference with the unicode module is that this function raises an error when L contains non-unicode characters (either values greater than 0x10FFFF, values in the interval 0xD800--0xD8FF, or 0xFFFE and 0xFFFF.

binary_to_unicode(B::binary()) -> [int()]

Converts a utf-8 binary into a list of unicode characters

The main difference with the unicode module is that this function raises an error when B is not an utf-8 encoded binary
