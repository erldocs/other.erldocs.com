is_alpha(Char::char()) -> boolean()

Determines if a character is alphabetic.

is_alnum(Char::char()) -> boolean()

Determines if a character is alphanumeric.

is_lower(Char::char()) -> boolean()

Determines if a character is lower-case.

is_upper(Char::char()) -> boolean()

Determines if a character is upper-case.

is_digit(Char::char()) -> boolean()

Determines if a character is a decimal digit.

is_xdigit(Char::char()) -> boolean()

Determines if a character is an hexadecimal digit.

is_blank(Char::char()) -> boolean()

Determines if a character is blank (\n, \r, \t, \f, \v).

is_space(Char::char()) -> boolean()

Determines if a character is a space or a tab.

is_atom(Char::char()) -> boolean()

Determines if a character is allowed in an unquoted atom (i.e. lower case, numeric, '_' or '@').

lower(Char::char()) -> char()

Convert a character to lower-case.

upper(Char::char()) -> char()

Convert a character to upper-case.

integer_to_hex(N::integer()) -> char()

Convert an integer between 0 and 15 to an hexadecimal character.

integer_to_hex(N::integer(), X2::lower | upper) -> char()

Convert an integer between 0 and 15 to an hexadecimal character choosing the case (lower, upper) for the hexadecimal digits.

hex_to_integer(Char::char()) -> integer()

Convert an hexadecimal character to an integer. If the character is not an hexadecimal character we return a 'badarg' exception.

must_urlencode(Char::integer()) -> boolean()

must_xmlencode(Char::char()) -> boolean()

Determine whether a character has to be XML-encoded. See http://en.wikipedia.org/wiki/UTF-8#Description and http://www.asciitable.com/.
