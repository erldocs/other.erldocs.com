Time helper functions.

Types

Functions

datetime_to_iso8601(X1::datetime() (see module calendar)) -> binary()

Convert a date and time in the format returned by calendar:universal_time/0 to a binary string in the ISO 8601 format (e.g. "2012-02-15T14:39:15Z"; "2012-02-15T14:39:15.671Z").

iso8601_to_datetime(X1::binary()) -> datetime()

Convert a datetime in the ISO 8601 format to a date and time in the format returned by calendar:universal_time/0.

gregorian_seconds_to_iso8601(Seconds::non_neg_integer()) -> binary()

Convert a date and time as the number of seconds since Jan 1, 0001, 00:00:00 to a binary string in the ISO 8601 format (e.g. "2012-02-15T14:39:15Z").

iso8601_to_gregorian_seconds(Iso8601::binary()) -> non_neg_integer()

Convert a date and time as binary string in the ISO 8601 format to the number of seconds since the Jan 1, 0001, 00:00:00. e.g. "2012-02-15T14:39:15.345Z"; "2012-02-15T14:39:15Z"; "2012-02-15T14:39:15.345-0300".

datetime_to_epoch(Datetime::datetime()) -> epoch()

Convert a datetime in the format returned by the calendar:universal_time/0 function into a timestamp as a floating-point with the number of seconds since the Unix Epoch (Jan 1, 1970, 00:00:00) and a precision of microseconds.

universal_time_to_epoch() -> epoch()

Returns the date and time as a floating-point timestamp with the number of seconds since the Unix Epoch (Jan 1, 1970, 00:00:00).

epoch_to_datetime(Epoch::epoch()) -> datetime()

Convert a timestamp as a floating-point with the number of seconds since the Unix Epoch (Jan 1, 1970, 00:00:00) and a precision of microseconds into a datetime in the format returned by the calendar:universal_time/0 function.

epoch_to_iso8601(Epoch::epoch()) -> binary()

Convert a date and time as the number of seconds since the Unix epoch (Jan 1, 1970, 00:00:00) with millisecond precision to a binary string in the ISO 8601 format (e.g. "2012-02-15T14:39:15.539Z").

iso8601_to_epoch(Iso8601::binary()) -> epoch()

Convert a date and time as binary string in the ISO 8601 format to the number of seconds since the Unix epoch (Jan 1, 1970, 00:00:00) with millisecond precision.

timestamp_to_epoch() -> epoch()

Returns the current date and time as a floating-point timestamp with the number of seconds since the Unix Epoch (Jan 1, 1970, 00:00:00) and a precision of microseconds, as returned by erlang:now/0.

timestamp_to_epoch(X1::timestamp() (see module erlang)) -> epoch()

Converts the date and time inf the format returned by os:timestamp/0 and erlang:now/0 ({Megasecs, Secs. Microsecs}) to a floating-point timestamp with the number of seconds since the Unix Epoch (Jan 1, 1970, 00:00:00) and a precision of microseconds.

timestamp_to_datetime(Timestamp::timestamp() (see module erlang)) -> datetime()

Converts the date and time in the format returned by os:timestamp/0 and erlang:now/0 ({Megasecs, Secs. Microsecs}) to a datetime in the format returned by the calendar:universal_time/0 function.
