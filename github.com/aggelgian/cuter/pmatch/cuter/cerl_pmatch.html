clauses(Clauses::[Clause], Env) -> {Expr, Vars}

  • Clause = cerl()
  • Expr = cerl()
  • Vars = [cerl()]
  • Env = environment() (see module rec_env)

Rewrites a sequence of clauses to an equivalent expression, removing as much repeated testing as possible. Returns a pair {Expr, Vars}, where Expr is the resulting expression, and Vars is a list of new variables (i.e., not already in the given environment) to be bound to the arguments to the switch. The following is a typical example (assuming E is a Core Erlang case expression): 

    handle_case(E, Env) ->
        Cs = case_clauses(E),
        {E1, Vs} = cerl_pmatch(Cs, Env),
        c_let(Vs, case_arg(E), E1).

The environment is used for generating new variables which do not shadow existing bindings.

See also: rec_env, expr/2, transform/2.
