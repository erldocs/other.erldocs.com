Min-Heap, Max-Heap for Priority Queues

This module implements min-heaps and max-heaps for use in priority queues. Each value in the heap is assosiated with a reference so that the user can change its priority in O(log n).

The implementation is based on ETS tables for the O(1) lookup time. It supports all the basic heap operations:

min/1, max/1 in O(1)

take_min/1, take_max/1 in O(log n)

insert/2 in O(log n)

update/3 in O(log n)

from_list/2 in O(n)

In order to achieve the above complexities the heap needs to store an extra tuple {Key, Reference} for every Key stored. In addition, the size of the heap is stored as a tuple {size, Size}.

For examples you can check the heap_demo module.

Types

heap() =
            #heap{mode = undefined | mode(),
                  htab = undefined | ets:tab()}

mode() = max | min

refterm() = {term(), reference()}

Functions

to_list(H::heap()) -> [term()]

Returns a list of the terms in a heap.

new(M::mode()) -> heap()

Creates an empty heap.

If M is max then it will be a max heap, else if M is min it will be a min heap.

delete(H::heap()) -> true

Deletes a heap.

heap_size(H::heap()) -> non_neg_integer()

Returns the number of elements contained in a heap.

is_empty(H::heap()) -> boolean()

Checks whether a heap is empty or not.

max(H::heap()) -> term() | {error, min_heap | empty_heap}

Returns the element of a max heap with the maximum priority.

If it is a min heap, it returns {error, min_heap}.

min(H::heap()) -> term() | {error, max_heap | empty_heap}

Returns the element of a min heap with the minimum priority.

If it is a max heap, it returns {error, max_heap}.

insert(H::heap(), X::term()) -> refterm()

Add a new element to a heap.

It returns a tuple with the element added and a reference so that one can change its priority.

take_max(H::heap()) -> term() | {error, min_heap | empty_heap}

Removes and returns the maximum priority element of a max heap.

take_min(H::heap()) -> term() | {error, max_heap | empty_heap}

Removes and returns the minimum priority element of a min heap.

update(H::heap(), Ref::reference(), X::term()) -> true

Change the priority of an element.

It changes the priority of the element referenced with Ref to Value and then re-arranges the heap.

from_list(M::mode(), L::[term()]) -> {heap(), [refterm()]}

Create a heap from a list of terms.

It returns the heap and a list of tuples {Key, Ref} where Key is the term that was added and Ref is its reference (used to change its priority).
