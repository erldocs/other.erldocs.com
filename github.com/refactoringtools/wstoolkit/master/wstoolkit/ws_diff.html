ws_diff(X1::{OldWsdl::filename() (see module file), Oldxsd::filename() (see module file)}, X2::{NewWsdl::filename() (see module file), NewXsd::filename() (see module file)}) -> {ok, [term()], [term()]}

This funtions tries to infer the API changes between two versions of the web service specification. It takes the WSDL and XSD specification of both versions, and reports what has been changed from the first version to the second version.So far, the changes this tool is able to report include: the adding/removing of WS APIs, renaming of APIs, adding/removing of API parameters, renaming of API parameters, as well as parameter type changes. The accuracy of this tool is still to be evaluated!
