wrangler_slice

Forward and backward slicing of Erlang programs.

Forward and backward slicing of Erlang programs. The current implementation does not handle message passing, and is not fully tested yet.

Functions

forward_slice(Files::[filename()], AnnAST::syntaxTree(), ModName::atom(), FunDef::syntaxTree(), Expr::syntaxTree()) -> [syntaxTree()]

Forward slice the program with expression Expr, which in contained in function FunDef, as the slicing criterion.

backward_slice(Files::[filename()], AnnAST::syntaxTree(), ModName::atom(), FunDef::syntaxTree(), Expr::syntaxTree()) -> term()

Backward slice the program with expression Expr, which is contained in function FunDef, as the slicing criterion.
