droplast(List::[T]) -> [T]

Drops the last element of a List. The list should be non-empty, otherwise the function will crash with a function_clause

foreach_except(Fun::(Elem::T) -> term(), List::[T]) -> ok

Calls Fun(Elem) for each element Elem in List except the last. This function is used for its side effects and the evaluation order is defined to be the same as the order of the elements in the list. The list should be non-empty, otherwise the function will crash with a function_clause

foreach_except2(Fun::(Elem::T) -> term(), List::[T]) -> ok

Calls Fun(Elem) for each element Elem in List except the two lasts elements. This function is used for its side effects and the evaluation order is defined to be the same as the order of the elements in the list. The list should be non-empty, otherwise the function will crash with a function_clause
