twilio

Twilio REST API library.

Twilio REST API library. Uses API version "2010-04-01".

Types

twilio_param() = {string(), string()}

twilio_response() = any()

Functions

make_call(AccountSID::string(), AuthToken::string(), From::string(), To::string(), Params::[twilio_param()]) -> twilio_response()

Makes a call. Opts is a list of parameters to send to twilio. The list of accepted parameters can be found at http://www.twilio.com/docs/api/rest/making_calls. One of "Url" or "ApplicationSid" must be provided.

send_sms(AccountSID::string(), AuthToken::string(), From::string(), To::string(), Body::[twilio_param()]) -> twilio_response()

Sends an SMS. Opts is a list of parameters to send to twilio. The list of accepted parameters can be found at http://www.twilio.com/docs/api/rest/making_calls. One of "Url" or "ApplicationSid" must be provided.

request(AccountSID::string(), AuthToken::string(), X3::atom(), Path::string(), Params::[{string(), string()}]) -> twilio_response()

Makes a Twilio API request.
