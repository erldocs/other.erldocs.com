get_cseq() -> cseq() (see module nksip)

Gets a new CSeq. After booting, CSeq's counter is set using nksip_util:cseq/0. Then each call to this function increments the counter by one.

initial_cseq() -> non_neg_integer()

Generates an incrementing-each-second 31 bit integer. It will not wrap around until until {{2080,1,19},{3,14,7}} GMT.
