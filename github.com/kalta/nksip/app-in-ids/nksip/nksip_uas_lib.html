preprocess(Req::request() (see module nksip)) -> request() (see module nksip) | own_ack

Preprocess an incoming request. Returns own_ack if the request is an ACK to a [3456]xx response generated by NkSip, or the preprocessed request in any other case.

If performs the following actions:

Adds rport, received and transport options to Via.

Generates a To Tag candidate.

Performs strict routing processing.

Updates the request if maddr is present.

Removes first route if it is poiting to us.
