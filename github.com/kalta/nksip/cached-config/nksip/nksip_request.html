field(App::term() | app_id() (see module nksip), ReqId::id() (see module nksip), Field::field()) -> term() | error

Gets specific information from the Request. The available fields are:

FieldTypeDescription
app_id nksip:app_id() SipApp this request belongs to
method nksip:method() Method
ruri binary() Request-Uri
ruri_scheme nksip:scheme() Request-Uri Scheme
ruri_user binary() Request-Uri User
ruri_domain binary() Request-Uri Domain
parsed_ruri nksip:uri() Request-Uri
aor nksip:aor() Address-Of-Record of the Request-Uri
call_id nksip:call_id() Call-ID Header
vias [binary()] Via Headers
parsed_vias [nksip:via()] Via Headers
from binary() From Header
from_scheme nksip:scheme() From Scheme
from_user binary() From User
from_domain binary() From Domain
parsed_from nksip:uri() From Header
to binary() To Header
to_scheme nksip:scheme() To Scheme
to_user binary() To User
to_domain binary() To Domain
parsed_to nksip:uri() To Header
cseq binary() CSeq Header
parsed_cseq {integer(),nksip:method()} CSeq Header
forwards integer() Forwards
routes [binary()] Route Headers
parsed_routes [nksip:uri()] Route Headers
contacts [binary()] Contact Headers
parsed_contacts [nksip:uri()] Contact Headers
content_type binary() Content-Type Header
parsed_content_type [nksip:token()] Content-Type Header
require binary() Require Header
parsed_require [nksip:token()] Require Header
expires binary() Expires Header
parsed_expires undefined | integer() Expires Header
supported binary() Supported Header
parsed_supported [nksip:token()] Supported Header
event</td> <td>`undefined | binary() Event Header
parsed_event undefined | nksip:token()' Event Header
all_headers [{binary(), binary()}] All headers in the request
body nksip:body() Parsed Body
dialog_id nksip_dialog:id() Dialog's Id (if the request has To Tag)
local {nksip:protocol(), inet:ip_address(), inet:port_number()} Local transport protocol, ip and port of a request
remote {nksip:protocol(), inet:ip_address(), inet:port_number()} Remote transport protocol, ip and port of a request
binary() {binary(), [binary()]} If you use a binary as a field name, NkSIP will return all the values of this header, or [] if it is not present

is_local_route(App::term() | app_id() (see module nksip), ReqId::id() (see module nksip)) -> boolean()

Checks if this request would be sent to a local address in case of beeing proxied. It will return true if the first Route header points to a local address or the Request-Uri if there is no Route headers.
