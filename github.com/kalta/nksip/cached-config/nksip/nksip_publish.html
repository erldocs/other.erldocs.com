NkSIP Event State Compositor.

NkSIP Event State Compositor

This module implements a Event State Compositor, according to RFC3903 By default, it uses the RAM-only built-in store, but any SipApp can implement nksip_sipapp:publisher_store/3 callback to use any external database.
