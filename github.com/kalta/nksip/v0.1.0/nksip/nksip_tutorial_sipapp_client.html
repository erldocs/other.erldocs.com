SipApp Tutorial client callback module implementation.

SipApp Tutorial client callback module implementation. This modules implements a client callback module for NkSIP Tutorial.

Functions

init(X1) -> term()

SipApp intialization.

invite(DialogId, ReqId, From, State) -> term()

Called when an INVITE is received. If the request has a SDP body, reply 180 Ringing, wait 2 seconds and reply 200 Ok with the same body (spawns a new process to avoid blocking the process). If not, reply 488 Not Acceptable with a Warning header.

options(ReqId, From, State) -> term()

Called when an OPTIONS is received. Reply 200 Ok with a custom header and some options.
