nksip_request

User Request management functions.

User Request management functions.

Types

field() = local | remote | method | ruri | parsed_ruri | aor | call_id | vias | parsed_vias | from | parsed_from | to | parsed_to | cseq | parsed_cseq | cseq_num | cseq_method | forwards | routes | parsed_routes | contacts | parsed_contacts | content_type | parsed_content_type | headers | body | dialog_id | sipapp_id

id() = {req, pid()}

Functions

field(Field::field(), Input::id() | request() (see module nksip)) -> any()

Gets specific information from the Request. The available fields are:

FieldTypeDescription
sipapp_id nksip:sipapp_id() SipApp's Id
method nksip:method() Method
ruri binary() Request-Uri
parsed_ruri nksip:uri() Request-Uri
aor nksip:aor() Address-Of-Record of the Request-Uri
call_id nksip:call_id() Call-ID Header
vias [binary()] Via Headers
parsed_vias [nksip:via()] Via Headers
from binary() From Header
parsed_from nksip:uri() From Header
to binary() To Header
parsed_to nksip:uri() To Header
cseq binary() CSeq Header
parsed_cseq {integer(),nksip:method()} CSeq Header
forwards integer() Forwards
routes [binary()] Route Headers
parsed_routes [nksip:uri()] Route Headers
contacts [binary()] Contact Headers
parsed_contacts [nksip:uri()] Contact Headers
content_type binary() Content-Type Header
parsed_content_type [nksip_lib:token()] Content-Type Header
headers [{binary(), binary()}] User headers (not listed above)
body nksip:body() Parsed Body
dialog_id nksip_dialog:id() Dialog's Id (if the request has To Tag)
local {nksip:protocol(), inet:ip_address(), inet:port_number()} Local transport protocol, ip and port of a request
remote {nksip:protocol(), inet:ip_address(), inet:port_number()} Remote transport protocol, ip and port of a request

fields(Fields::[field()], Input::id() | request() (see module nksip)) -> [any()]

Gets a number of fields from the Request as described in field/2.

headers(Name::string() | binary(), Input::id() | request() (see module nksip)) -> [binary()]

Gets all Name headers from the request.

method(Request::id() | request() (see module nksip)) -> method() (see module nksip)

Gets the method of a Request.

body(Request::id() | request() (see module nksip)) -> body() (see module nksip)

Gets the body of a Request.

provisional_reply(Request::id(), SipReply::sipreply() (see module nksip)) -> ok | {error, Error}

  • Error = unknown_request | invalid_response | network_error | invalid_request

Sends a provisional response to a request.

is_local_route(Request::id() | request() (see module nksip)) -> boolean()

Checks if this request would be sent to a local address in case of beeing proxied. It will return true if the first Route header points to a local address or the Request-Uri if there is no Route headers.
