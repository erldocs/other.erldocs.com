Functions

field(Field::field(), Input::id() | response() (see module nksip)) -> any()

Gets specific information from the Response. The available fields are the same as nksip_request:field/2, and also:

FieldTypeDescription
code nksip:response_code() Response Code
response binary() Reason Phrase

fields(Fields::[field()], Input::id() | response() (see module nksip)) -> [any()]

Gets a number of fields from the Response as described in field/2.

headers(Name::string() | binary(), Input::id() | response() (see module nksip)) -> [binary()]

Gets all Name headers from the response.

code(Input::id() | response() (see module nksip)) -> response_code() (see module nksip)

Gets the response code of a Response.

reason(Input::id() | response() (see module nksip)) -> binary

Gets the reason of a Response.

dialog_id(Input::id() | response() (see module nksip)) -> id() (see module nksip_dialog)

Gets the dialog id of a Response.

body(Input::id() | response() (see module nksip)) -> body() (see module nksip)

Gets the body of a Response.

wait_491() -> ok

Sleeps a random time between 2.1 and 4 secs. It should be called after receiving a 491 response and before trying the request again.
