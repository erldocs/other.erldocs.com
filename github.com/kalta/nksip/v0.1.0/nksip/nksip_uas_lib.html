preprocess(Sipmsg::request() (see module nksip)) -> {ok, request() (see module nksip)} | ignore

Preprocess an incoming request. Returns ignore if the request is an ACK directed to us, or {ok, nksip:request()} in any other case. If performs the following actions:

Adds rport, received and transport options to Via.

Generates a To Tag candidate.

Extracts the authorized digest users.

Performs strict routing processing.

Updates the request if maddr is present.

Removes first route if it is poiting to us.
