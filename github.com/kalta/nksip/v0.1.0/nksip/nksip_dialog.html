nksip_dialog

User dialog management module.

User dialog management module. This module implements several utility functions related to dialogs.

id(Dialog::spec()) -> id()

Calculates a Dialog's Id from a nksip_request:id(), nksip_response:id(), nksip:request() or nksip:response(). Dialog Ids are calculated as a hash over Call-ID, From tag and To Tag. If From Tag and To Tag are swapped the resulting Id is the same.

find_callid(AppId::sipapp_id() (see module nksip), CallId::call_id() (see module nksip)) -> [id()]

Finds all existing dialogs' processes having a Call-ID

stop(DialogSpec::spec()) -> ok | {error, unknown_dialog}

Stops an existing dialog (remove it from memory).

bye_all() -> ok

Sends an in-dialog BYE to all existing dialogs.

stop_all() -> ok

Stops all current dialogs.

is_authorized(Request::request() (see module nksip)) -> boolean()

Checks if a request is part of an already authenticated dialog, and it comes from the same ip and port.
