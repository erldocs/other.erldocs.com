nksip_config

NkSIP Config Server.

NkSIP Config Server.

This module implements a simple ETS-based config store.

NkSIP has few global configuration options. Config values are set using standard nksip application's erlang environmet (file nksip.app). Their default values are defined in the following table.

NameDefaultComments
timer_t1500Standar SIP T1 timer (msecs).
timer_t24000Standar SIP T2 timer (msecs).
timer_t45000Standar SIP T4 timer (msecs).
timer_c180000Standar SIP C timer (msecs).
transaction_timeout900000INVITE transaction maximum time (secs).
dialog_timeout900000 Time to destroy dialog if no message has been received (msecs).
tcp_timeout180000 Time to disconnect TCP/SSL connection if no message has been received (msecs).
sctp_timeout180000 Time to disconnect SCTP associations if no message has been received (msecs).
nonce_timeout30000 Time a new nonce in an authenticate header will be usable (msecs, only for ACK or requests coming from the same ip and port).
max_calls100000Maximum number of allowed calls. Each different Call-ID counts as a call.
max_connections1024 Maximum number of simultaneous TCP/TLS connections NkSIP will accept in each transport belonging to each SipApp.
registrar_default_time3600 Registrar default time (secs).
registrar_min_time60 Registrar minimum allowed time (secs).
registrar_max_time86400 Registrar maximum allowed time (secs).
dns_cache_ttl3600 DNS cache TTL (See nksip_dns) (secs).

cseq() -> cseq() (see module nksip)

Gets a new CSeq. After booting, CSeq's counter is set using nksip_lib:cseq/0. Then each call to this function increments the counter by one.
