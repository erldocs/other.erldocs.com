Functions

field(AppId::app_id() (see module nksip), RespId::id(), Field::field()) -> term() | error

Gets specific information from the Response. The available fields are the same as nksip_request:field/2, and also:

FieldTypeDescription
code nksip:response_code() Response Code
reason binary() Reason Phrase

fields(AppId::app_id() (see module nksip), RespId::id(), Fields::[field()]) -> [{atom(), term()}] | error

Get some fields from a response.

header(AppId::app_id() (see module nksip), RespId::id(), Name::binary()) -> [binary()] | error

Get header values from a response.

code(AppId::app_id() (see module nksip), RespId::id()) -> response_code() (see module nksip) | error

Gets the response code of a response.

reason(AppId::app_id() (see module nksip), RespId::id()) -> binary() | error

Gets the reason of a response.

body(AppId::app_id() (see module nksip), RespId::id()) -> body() (see module nksip) | error

Gets the body of a response.

dialog_id(AppId::app_id() (see module nksip), ReqId::id()) -> id() (see module nksip_dialog) | error

Gets the dialog_id of a request.

call_id(X1::id()) -> call_id() (see module nksip)

Gets the calls's id of a response id

wait_491() -> ok

Sleeps a random time between 2.1 and 4 secs. It should be called after receiving a 491 response and before trying the request again.
