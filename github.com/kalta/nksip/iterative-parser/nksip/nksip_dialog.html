nksip_dialog

User Dialog Management Module.

User Dialog Management Module. This module implements several utility functions related to dialogs.

Functions

field(AppId::app_id() (see module nksip), DialogSpec::spec(), Field::field()) -> term() | error

Gets specific information from the dialog indicated by DialogSpec. The available fields are:

FieldTypeDescription
app_id nksip:app_id() SipApp that created the dialog
call_id nksip:call_id() Call-ID
created nksip_lib:timestamp() Creation timestamp
updated nksip_lib:timestamp() Last update timestamp
answered nksip_lib:timestamp()|undefined Answer (first 2xx response) timestamp
status status() Current dialog state
expires integer() Seconds to expire current state
local_seq nksip:cseq() Local CSeq number
remote_seq nksip:cseq() Remote CSeq number
local_uri binary() Local URI
parsed_local_uri nksip:uri() Local URI
remote_uri binary() Remote URI
parsed_remote_uri nksip:uri() Remote URI
local_target binary() Local Target URI
parsed_local_target nksip:uri() Local Target URI
remote_target binary() Remote Target URI
parsed_remote_target nksip:uri() Remote Target URI
early boolean() Early dialog (no final response yet)
secure boolean() Secure (sips) dialog
route_set [binary()] Route Set
parsed_route_set [nksip:uri()] Route Set
local_sdp nksip:sdp()|undefined Current local SDP
remote_sdp nksip:sdp()|undefined Current remote SDP
stop_reason stop_reason() Dialog stop reason

class_id(Class::uac | uas, Sipmsg::request() (see module nksip) | response() (see module nksip)) -> id()

Calculates a dialog's id from a nksip:request() or nksip:response() and a endpoint class. Dialog ids are calculated as a hash over Call-ID, From tag and To Tag. Dialog ids with same From and To are different for different endpoint classes.

id(AppId::app_id() (see module nksip), DialogId::id() | id() (see module nksip_request) | id() (see module nksip_response)) -> id()

Calculates a dialog's id from a nksip_request:id(), nksip_response:id().
