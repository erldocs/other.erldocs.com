start a listener

    Ref = term()

    TransOpts = any()
    Protocol = any()
    ProtocolOpts = any()
    ListenerOpts - any(),

A Ref can be any Erlang term used to identify a listener. A listener is a gen_server that manage all acceptors workers and handle connections shutdown.

A protocol is the protocol used to handle a connection. It can be any module following the protocol behaviour. Barrel offers to handle the TCP ans SSL/TLS protocol for now.

A protocol is what will be used to handle the data coming from the socket. You can pass to it some options (ProtocolOpts).

Optionnaly you can pass custom options to the listener. This is where you pass the SSL options for example.
