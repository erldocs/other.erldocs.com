fold(View, Fun) -> term()

Calls Fun on successive keys and values of View Results. Fun must return a new accumulator which is passed to the next call. [] is returned if the list is empty. The evaluation order is undefined.

foreach(View, Fun) -> term()

Calls Fun(Elem) for each element Row in view results. This function is used for its side effects and the evaluation order is defined to be the same as the order of the elements in the results.
