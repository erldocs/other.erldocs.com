add(Attr::ejson_attr(), Value::ejson_val()) -> string()

Creates a JSON string, adds attribute 'Attr' with value 'Value'.

Example: 

  > Attr = attr2.
  > Value = <<"value2">>.
  > ejson:add(Attr, Value).
  "{\"attr2\":\"value2\",\"attr1\":\"value1\"}"

Example: 

  > Attr = "attr2.attr3".
  > Value = <<"value2">>.
  > ejson:add(Attr, Value).
  "{\"attr2\":{\"attr3\":\"value2\"},\"attr1\":\"value1\"}"

add(Json::string(), Attr::ejson_attr(), Value::ejson_val()) -> string()

Adds attribute 'Attr' with value 'Value' to a JSON string.

Example: 

  > Json = "{\"attr1\":\"value1\"}".
  > Attr = attr2.
  > Value = <<"value2">>.
  > ejson:add(Json, Attr, Value).
  "{\"attr2\":\"value2\",\"attr1\":\"value1\"}"

Example: 

  > Json = "{\"attr1\":\"value1\"}".
  > Attr = "attr2.attr3".
  > Value = <<"value2">>.
  > ejson:add(Json, Attr, Value).
  "{\"attr2\":{\"attr3\":\"value2\"},\"attr1\":\"value1\"}"
