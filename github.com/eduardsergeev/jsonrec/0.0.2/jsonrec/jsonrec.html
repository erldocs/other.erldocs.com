atom_to_pascal(Atom::atom()) -> string()

Converts erlang_atom into PascalStyle string

Convert atom() is a "standard" Erlang form (_ separated) into string() capitalized in "Pascal"-style

This function can be specified in jsonrec_encode:name_handler() or jsonrec_decode:name_handler() to force generated JSON to use PascalStyle field names

Examples:

atom -> "Atom"

another_atom -> "AnotherAtom"

very_verbose_atom -> "VeryVerboseAtom"
