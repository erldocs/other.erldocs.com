Generates JSON "decode" function body for JSON de-serialization using type annotation.

Note: This is meta function and normally it should not be used directly. Use ?decode_gen/2 and ?decode_gen/3 macros instead.

Arg is meta:quote() reference to generated function input parameter

Type is meta:quote() of the type being JSON decoded

Info is meta:info() structure returned by meta:reify/0

Options is meta:quote() of a list of options decode_options()

Typical usage: 

     decode(Bin) ->
       ?decode_gen(#my_record{}, Bin).

Upon compilation this code will generate parsing code which populates #my_record{} record if parsing succeeds.
