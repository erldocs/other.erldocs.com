ns_connection_pool

Connection manager for the more or less arbitrary protocol tcp sockets.

Connection manager for the more or less arbitrary protocol tcp sockets. This gen_server is responsible for keeping track of persistent connections to HTTP servers. The only interesting API is connection_count/0 and connection_count/1.
