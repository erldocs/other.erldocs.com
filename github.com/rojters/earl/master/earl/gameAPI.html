gameAPI

An API of functions needed when designing a game.

An API of functions needed when designing a game.

Functions

init(Game, Players, GameRoomPid) -> ok

Starts a game instance for the players in the Playerlist.

getInput(Pid) -> Input

Asks a player for an input. If that player has died that players Pid is returned.

getNumber(Pid) -> Integer | {error}

Sends a request to the specified Pid for a numeric input. The function returns either the read input or an error if an invalid character was specified. If that player has died that players Pid is returned.

print(Output, Players) -> ok

Sends a message to be printed to all players in the list.

getPlayer(Int, Players::Playerlist) -> Player

Gets the nth player from the list.
