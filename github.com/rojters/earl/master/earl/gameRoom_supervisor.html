gameRoom_supervisor

A supervisor that restarts a game room if it dies and updates the database.

A supervisor that restarts a game room if it dies and updates the database.

Functions

init(Gamelist::GameList, DbPid) -> GameList

Initiates the supervisor to trap and handle exits and starts the spawnRooms function.

linkRooms(Tail::GameList, ResultingList, DbPid) -> ok

Links all gameRooms in GameList to this supervising process.

trap(GameList, DbPid) -> ok

Traps exits and restarts any dying gameroom and updates the database on DbPid with a new GameList for any restarted gameRoom.

restart_room(Pid, GameList) -> GameList

restarts the gameRoom with Pid and updates its corresponding place in the Gamelist.
