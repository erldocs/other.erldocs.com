tree_length(Tree::mtree()) -> bin()

get highest Bin of the leaf node, which indicates the number of elements in the tree or the tree breadth/size. NOTE: Since the ETS table is an ordered storage the last element will always be a leaf node. So, generated bin will get the last entry in the ETS table and return it which can used to calculate the ROOT of the tree
