connect(Relay::relay(), Opts::options()) -> connection()

Connects to a given STMP relay and returns an SMTP connection. Note, that a relay workers are reused between {link connect/2} calls.

check(X1::connection()) -> {ok, binary()} | {error, error()}

Checks that a conection is valid, i. e. that we can establish an SMTP session and authenticate using configure credentials (if any).

send(X1::connection(), Email::#email{sender=undefined | address() (see module pat), recipients=undefined | [address() (see module pat)], subject=binary(), message=undefined | binary(), headers=[{binary(), binary()}]}) -> {ok, receipt()} | {error, error()}

Sends an email via a given SMTP relay connection and returns a receipt, if everything worked or an appropriate error atom.
