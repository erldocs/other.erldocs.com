xhttpc_cookielib

Cookie management library.

Cookie management library. Implement cookie header parsing/serialization and cookie policy rules. RFC6265 http://tools.ietf.org/html/rfc6265 See example usage in middleware/cookie_middleware.erl

lookup_key(X1::url()) -> lookup_key()

You may lookup all cookies, that may be allowed for this URL, using this key as prefixes, eg:

Pseudocode 

  {HostKey, PathKey} = lookup_key(Url)
  SELECT * FROM cookies
  WHERE $HostKey LIKE (cookies.key_host || '%')
        AND $PathKey LIKE (cookies.key_path || '%')

Dict as storage 

  {HostUrlKey, PathUrlKey} = lookup_key(Url),
  AllowedCookies = dict:fold(
      fun({HostCookieKey, PathCookieKey, _Name}, C, Acc) ->
         Allow = (lists:prefix(HostCookieKey, HostUrlKey)
                  and lists:prefix(PathCookieKey, PathUrlKey))
         if Allow -> [C | Acc];
         true -> Acc
         end
      end, [], Cookies).

dict:store(cookie_key(Cookie), Cookie, Storage)
