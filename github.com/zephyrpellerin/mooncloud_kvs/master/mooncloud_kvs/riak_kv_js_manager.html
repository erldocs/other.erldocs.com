reload(Names::[string() | atom()]) -> ok

Reload the Javascript VMs in the named pools. If no pool names are given, all pools in the JSPOOL_LIST (defined in riak_kv_js_pools.hrl) are reloaded. Pool names may be given as either list-strings (as they will be when this function is invoked via 'riak-admin js_reload') or as atoms (as they are defined in riak_kv_js_pools.hrl).
