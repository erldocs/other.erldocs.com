start(Name::atom(), URL::string(), ClientOptions::list(), HTTPOptions::list()) -> {ok, pid()} | start_error()

Start a new XML-RPC client. The name of the client is used as the inets client profile. ClientOptions is passed after the inets HTTP client initialization to httpc:set_options/2 and HTTPOptions at each remote procedure call to httpc:request/5.

start_link(Name::atom(), URL::string(), ClientOptions::list(), HTTPOptions::list()) -> {ok, pid()} | start_error()

Start a new XML-RPC client. The client is linked to the calling process.

See also: start/4.
