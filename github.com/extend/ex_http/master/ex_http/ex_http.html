token_k(Bin::binary(), Start::pos(), Cont::({binary(), pos()}) -> T) -> T | badarg

Extract a token from a binary and pass it to the given continuation, along with the position of the end of the token. If no token char is found at the start position, badarg is directly returned.

lc_token_k(Bin::binary(), I::pos(), Cont::({binary(), pos()}) -> T) -> T | badarg

Same as token_k/3 except that the token given to the continuation is passed with all characters lowercase.

lws_k(Bin::binary(), I::pos(), Cont::(pos()) -> T) -> T

Skip any linear whitespace at a given position in a binary and call the continuation with the new position.
