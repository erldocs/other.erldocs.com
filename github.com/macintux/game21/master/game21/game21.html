game21

Solving simple game problems from the classic MIT 6.001 class.

Solving simple game problems from the classic MIT 6.001 class. Simplified blackjack with limited rules, strictly numeric cards.

Types

action() = draw | done | quit | retry

card() = integer() >= 0

endresult() = user_quit
                    | user_busted
                    | house_busted
                    | house_won
                    | user_won

hand() = integer() >= 0

strategyfun() = fun((hand(), card()) -> action())

Functions

start(HouseStrategy::strategyfun(), PlayerStrategy::strategyfun()) -> endresult()

test_strategy(Player::strategyfun(), House::strategyfun(), TryCount::non_neg_integer()) -> non_neg_integer()

Trigger TryCount games of 21. Take two strategies, the first for the player, the second for the house, and run them TryCount times to see how often the player wins.

louis(Tally::hand(), OpponentUp::card()) -> action()

Louis' 21 strategy: draw if the current hand is < 12, hold if > 16, and otherwise decide what to do based on the house's up card.

both(S1::strategyfun(), S2::strategyfun()) -> strategyfun()

A strategy function generator that takes 2 strategies and returns a new one that will only draw if both say to draw.

watch_player(F::strategyfun()) -> strategyfun()

Take a strategy and return a wrapper function that will display the inputs and outputs.
