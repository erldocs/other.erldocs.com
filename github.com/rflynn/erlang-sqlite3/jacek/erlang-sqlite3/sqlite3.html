start_link(Db) -> {ok, Pid} | ignore | {error, Error}

  • Db = atom()

Opens a sqlite3 dbase creating one if necessary. The dbase must be called Db.db in the current path. start_link/1 can be use with stop/0, sql_exec/1, create_table/2, list_tables/0, table_info/1, write/2, read/2, delete/2 and drop_table/1. There can be only one start_link call per node.

To open multiple dbases on the same node use open/1 or open/2.

  • Db = atom()

DbFile must be the full path to the sqlite3 db file. start_link/1 can be use with stop/0, sql_exec/1, create_table/2, list_tables/0, table_info/1, write/2, read/2, delete/2 and drop_table/1. There can be only one start_link call per node.
