dirHandler(Directory::string()) -> {ok, DirList} | {error, Reason} | {error_eval, Bin}

Returns a sorted list over all files in Directory (with all directories placed before all files). If Directory does not exist, {error, enoent} is returned. If Directory is a file {error, eisfile} is returned.

fixPath(Path::string()) -> FixedPath

Returns a modified version of Path that both begins and ends with the $/. character. If both $/. are already present, Path is returned unchanged.

uploadFile(WebPath, FileContents) -> term()

getFile(FileName::string()) -> {ok, FileData, FileInfo} | {error, eisdir} | {error, enoent} | {error_eval, Bin}

Returns {ok, FileHandle} if file found. If FileName does not exist, returns {error, enoent} If FileName is a directory, returns {error, eisdir} For any other error returns {error_eval, Bin}

Use FileHandle in all file information functions in this module.

getContents(FileHandle::{FileData, FileInfo}) -> List

Returns a string containing the entire file

getInfoAll(Filehandle::{FileData, FileInfo}) -> List

Returns a list containing all {Infokey, Infovalue} pairs from FileHandle

getInfo(Filehandle::{FileData, FileInfo}, Info::List) -> List

Returns the corresponding file info in filehandle FileHandle to the atom Info, or false if it is not found. Example: getInfo("hej.txt", contenttype)
