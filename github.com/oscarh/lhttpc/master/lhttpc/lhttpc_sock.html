lhttpc_sock

This module implements wrappers for socket operations.

This module implements wrappers for socket operations. Makes it possible to have the same interface to ssl and tcp sockets.

Functions

connect(Host, Port, Options, Timeout, X5::SslFlag) -> {ok, Socket} | {error, Reason}

  • Host = string() | ip_address()
  • Port = integer()
  • Options = [{atom(), term()} | atom()]
  • Timeout = infinity | integer()
  • SslFlag = boolean()
  • Socket = socket()
  • Reason = atom()

Connects to Host and Port. Will use the ssl module if SslFlag is true and gen_tcp otherwise. Options are the normal gen_tcp or ssl Options.

recv(Socket, X2::SslFlag) -> {ok, Data} | {error, Reason}

  • Socket = socket()
  • Length = integer()
  • SslFlag = boolean()
  • Data = term()
  • Reason = atom()

Reads available bytes from Socket. Will block untill data is available on the socket and return the first packet.

Receives Length bytes from Socket. Will block untill Length bytes is available.

send(Socket, Request::Data, X3::SslFlag) -> ok | {error, Reason}

  • Socket = socket()
  • Data = iolist()
  • SslFlag = boolean()
  • Reason = atom()

Sends data on a socket. Will use the ssl module if SslFlag is set to true, otherwise the gen_tcp module.
