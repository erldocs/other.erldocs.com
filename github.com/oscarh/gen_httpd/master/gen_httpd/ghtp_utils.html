header_values(Name, Headers) -> Values

  • Name = string()
  • Headers = [header()]
  • Values = [string()]

Returns all values for the header named Name. Name is expected to be a lowercase string. Returns a list of all values found in the list of headers. Useful for for instance "Transfer-Encoding", of which several can be applied.

  • URIEncodedString = string()

Encodes a string according to RFC2396. All reserver or excluded characters are turned in to a "%" HEX HEX notation.

  • String = string()

Decodes a string according to RFC2396. All "%" HEX HEX sequences are turned in to the character represented by the HEX HEX number.
