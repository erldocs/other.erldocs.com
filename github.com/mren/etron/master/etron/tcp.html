Functions

startServer(Port, ReceivingPid) -> term()

start a tcp server listening on Port. the server will send messages to ReceivingProcess:

{messageFrom, Client::socket(), Message::term()} when a message from a connected client was received

{connected, Client::socket()} when a client has connected

{disconnected, Client::socket()} when a client has closed his connection

send(Socket, Message) -> term()
