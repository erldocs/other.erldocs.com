Functions

start(Mod, Pid) -> term()

start expects a callback module implementing the following functions

init(Width, Height) -> State initialize a canvas of the given size returns the initial state beforeDraw(State) -> State called before a frame is drawn afterDraw(State) -> State called after a frame is drawn setSize(State, {Width, Height}) -> State setColor(State, Red, Green, Blue) -> State set a color to be used to draw objects fillRect(State, X, Y, Width, Height) -> State draw a rectangle filled with the last set color setKeyUpCallback(State, CallbackFunction) -> State CallbackFunction(Key) -> ok. Key = left | right | up | down callback function must be called whenever a key is pressed on the canvas close(State) close the canvas

state is some internal state used by the callback module which gets passed around and into each callback
