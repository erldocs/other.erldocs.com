server_state

interface to an abstract data type ServerState which maintains mappings from client sockes to colors colors are used as player ids.

add_client(Client, X2::ServerState) -> {Color, ServerState} | false

remove_client(Client, X2::ServerState) -> {Color, ServerState} | false

removes a Client association with a color and returns the color it was associated with returns false if there was no Client previously added
