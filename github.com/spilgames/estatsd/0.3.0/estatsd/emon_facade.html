increment(Key::string(), Amount::pos_integer(), SampleRate::float()) -> ok

Increments the value of he given Key in the given Amount. Note that SampleRate smaller than 1 allows you to state that you are only measuring a porcentage of the actual calls and you want your amounts to be updated as if you where measuring them all. If in doubt on how to use this parameter, set it to 1.
