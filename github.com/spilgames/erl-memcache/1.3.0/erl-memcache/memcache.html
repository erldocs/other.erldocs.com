start_pool(Poolname::pool_name(), Host::pool_host(), Port::pool_port(), MemcachedOpts::memcached_opts(), Size::pool_size(), MaxOverflow::pool_max_overflow(), StartServer::boolean()) -> ok | {error, term()}

Starts a cache pool. The StartServer variable decides whether this application takes care of starting the memcached server as well. If it was already started, just returns ok. Note that the start server functionality will only work for starting local memcached instances, so Host should be a valid name for localhost or an IP address for one of the local network interfaces.

start_pool(Poolname, Host, Port, Size, MaxOverflow, StartServer) -> term()

stop_pool(Poolname::pool_name()) -> ok | {error, pool_not_found | term()}

Stops the given pool. If the memcached server for the pool had been started by this application, it is stopped as well.
