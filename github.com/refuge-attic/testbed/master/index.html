etap
etap is a TAP testing module for Erlang components and applications.

etap_application
Provide test functionality to the application and related behaviors.

etap_exception
Adds exception based testing to the etap suite.

etap_process
Adds process/pid testing to the etap suite.

etap_report
A module for creating nice looking code coverage reports.

etap_request
Provides test functionality against a specific web request.

etap_string
Provide testing functionality for strings.

etap_web
Provide testing functionality for web requests.
