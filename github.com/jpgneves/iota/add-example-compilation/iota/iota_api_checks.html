internal_consistency(Data::iota_info_item() (see module iota_scanner), Results::iota_result() (see module iota_result)) -> iota_result() (see module iota_result)

Perform internal consistency checks on a module. Checks if the declared API is a subset of the exported functions for a module and emits a warning if Exported - API = {}, and an error if API - Exported = {}.

external_calls(Data::iota_info_item() (see module iota_scanner), Results::iota_result() (see module iota_result)) -> iota_result() (see module iota_result)

Check external calls between applications using xref. Emits errors if an application calls another application's module which is not declared as an API module, or if it calls a function that is not declared as part of the API for that application through the module.
