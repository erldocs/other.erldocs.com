kraken_memcached

Synchronous TCP Client Interface based on the Memcached protocol so that we can use existing Memcached clients.

Synchronous TCP Client Interface based on the Memcached protocol so that we can use existing Memcached clients. This protocol may seem a little counter intuitive but that is because we want it to map cleanly to the Memcached protocol. For example, our own protocol would have single line SUBSCRIBE and UNSUBSCRIBE operations such that we wouldn't need to include the number of bytes.
