iso8601

This application allows to parse and format date, time, intervals and recurring intervals as described in ISO8601 standard.

parse_date(Date::binary() | nonempty_string()) -> {Date::date() (see module calendar), Format::date_format()}

Parses dates autodetecting one of formats specified in standard

This function Detects format of the date as specified in ISO 8601 standard section 4.1. The autodetection is based on length, and position of separators ( like "-" ) or special characters ( like "W" )
