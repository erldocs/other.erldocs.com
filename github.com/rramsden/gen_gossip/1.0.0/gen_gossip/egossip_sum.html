egossip_sum

Implements a simple aggregation-based summation protocol.

Implements a simple aggregation-based summation protocol. cycles/1 defines how long it takes to converge on the answer. We calculate the sum of the cluster by taking the average of value and multiplying it by the number of nodes in the conversation.

Usage:

(a@machine1)> egossip_sum:start_link(25). (b@machine1)> egossip_sum:start_link(25). (b@machine1)> net_adm:ping('a@machine1').
