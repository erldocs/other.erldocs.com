combinations(M, N) -> R

  • M = int()
  • N = int()
  • R = error | int()

Computes the combinations of M elements taken by N without repetitions.

This functions exits if (M < 0), (N < 0) or (M < N).

See also: div_factorial/2, factorial/1.
