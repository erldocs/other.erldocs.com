verify_token(Token::token(), TokenID::string(), Period::integer()) -> ok | {error, wrong_token} | {error, expired}

Verifies a confirmation token (not supplied by the user) against the confirmation string supplied by the user and an expiry period in seconds. Elapsed expiry time is now() - the token issue time.

expired(T::token(), Period::integer()) -> bool()

True if the token has expired (the time between the token issue time and now() is greater than Period seconds).
