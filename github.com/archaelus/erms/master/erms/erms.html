erms

This module contains functions intended to be run by humans from a console node.

This module contains functions intended to be run by humans from a console node. This module also contains various utility code that should migrate to more appropriate modules in the fullness of time.

Loads a single shortcode from the file named File and writes the definition to the database. You'll need to erms_shortcode:reconfigure(ShortcodeName) afterwards.
