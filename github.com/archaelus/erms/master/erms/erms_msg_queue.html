queue(Msg_status::#msg_status{}, Msg::#msg{}) -> ok | {error, Reason::term()}

Submits the Message Status/Message pair for processing and delivery. Message submission entails writing the status/message records to mnesia - this is essentially a fire and forget interface. If an {error,Reason} result is returned, the message can't be queued for some reason (most likely system overload).

This function is called by a gen_server when it is about to terminate. It should be the opposite of Module:init/1 and do any necessary cleaning up. When it returns, the gen_server terminates with Reason. The return value is ignored.
