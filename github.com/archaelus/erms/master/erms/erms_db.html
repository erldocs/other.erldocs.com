init_node() -> ok

Initalises a new ERMS node by creating an mnesia schema (with disk copies on the local node) and then recreating the erms database from priv/text/db.erl

See also: recreate_db/0.

init_node(Filename::string()) -> ok

Initialises a new ERMS node from the mnesia text dump in Filename. Uses recreate_db[{file, Filename}].

See also: recreate_db/1.

recreate_db() -> ok

Recreates the ERMS db with the standard options. This should enter the data in priv/text/db.erl into database tables (creating such if necessary).

recreate_db(Options::[Option]) -> ok

  • Option = drop | {drop, boolean()} | {file, Filename}

Recreates the ERMS database given the list of Options. The drop ({drop, true}) option drops existing ERMS tables before recreating and reentering data. The {file, Filename} option specifies the mnesia text dumpfile to use when creating the database and entering data. The default dumpfile is "priv/text/db.erl".
