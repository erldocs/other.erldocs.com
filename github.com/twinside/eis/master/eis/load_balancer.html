load_balancer

Module in charge of the dynamic balancing of different ressources.

Module in charge of the dynamic balancing of different ressources. can start a given number of thread to handle the charge.

  • Module = atom()
  • Function = atom()

Call to start the balancer.

Module/Function is what to start, MaxRessource is the upper bound of balanced ressources.
