irc_log

This module represent the logging thread which store all the diferents elements which happen across the server.

Functions

basic_init() -> term()

Start a basic logger which output everything on the standard output.

init(Function, FuncArg) -> Result

  • Function = func()
  • FuncArg = term()
  • Result = {ok, Pid} | {error, Reason}

Start a logger. It's a more advanced one, every logging events are passed through a given user function.

logDebug(Txt) -> nil

  • Txt = string()

Used to log debug information, should not be used in the final release.

logVerbose(Txt) -> nil

  • Txt = string()

Log marginaly informative messages, used for debug, or to display inner information about the server.

logInfo(Txt) -> Result

  • Txt = string()
  • Result = none

Log some milestones usefull for administration eg : begin/end of initialisation.

logEvent(Txt) -> nil

  • Txt = string()

Log information like "a client has connected" and information needed by administration (conf reloading is a good exemple).

logError(Txt) -> nil

  • Txt = string()

If an event is abnormal and cause a thread crash, it should be this log level.

logFatal(Txt) -> nil

  • Txt = string()

Log information issued before crashing the the server, should be followed by a call to the halt() function.
