ebi_mc2_cluster_sup

Supervisor governing processes related to single cluster connection.

Supervisor governing processes related to single cluster connection. They should be considered fragile. The server dicsonnects the connections from time to time. The ssh_channel sometimes hangs till first call, and then fails after some timeout.
