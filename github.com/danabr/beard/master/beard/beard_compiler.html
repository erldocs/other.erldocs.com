Functions

file(FilePath) -> term()

Parses the file at FilePath as a beard template.

beam(TemplatePath, LogicPath, OutDir) -> term()

Compiles a template and a logic module into a Erlang module, which will be placed in OutDir. The module itself will be named after the logic module. If the compilation fails, the Erlang source will be placed as LogicPath ++ ".erl" for debugging purposes. Otherwise, the source code will be removed.
