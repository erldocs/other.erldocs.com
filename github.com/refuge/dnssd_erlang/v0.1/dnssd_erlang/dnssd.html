dnssd

This module is the main interface to the application.

This module is the main interface to the application.

If successful, the enumerate, browse, resolve and register functions all return {ok, Ref, InitialResults}. InitialResults will be a list (usually empty), subsequent results will be sent to the process that started the operation via a message of the form {dnssd, Ref, Result}. To stop the operation pass Ref to stop/1. The operation will also be stopped if the calling process exists.
