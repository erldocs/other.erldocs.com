resource_discovery

Client API for resource discovery.

Client API for resource discovery. Services local to a node can use this API to discover and add resources for different hosts.

Types

resource() = #resource{}

Functions

insert(Host::string(), Resources::resource() | [resource()]) -> ok

Add resources for host.

If the host doesn't exist then create the host otherwise add the resources.

lookup(Host::string()) -> {ok, [resource()]} | {error, not_found}

Lookup the resources for a host.

create(Host::string()) -> ok | {error, exists}

Creates a new host for holding resources if it doesn't already exist.

delete_host(Host::string()) -> ok

deletes a host and all resources. If host doesn't exist just ignores it.

all() -> [{string(), [resource()] | []}] | []

Get a list of host resources in the form [{Host, [resources]}].

delete_resources_for(Host::string(), Resources::[resource()] | []) -> ok

Delete a list of resources from the specified host.

update_resources_for(Host::string(), Resources::[resource()] | []) -> ok

Update a list of resources from the specified host.
