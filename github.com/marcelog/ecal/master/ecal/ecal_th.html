Types

threshold() =
            {threshold_type(),
             threshold_start(),
             threshold_end(),
             threshold_length()}

threshold_end() = ecal_time:timespec()

threshold_length() = integer()

threshold_start() = ecal_time:timespec()

threshold_type() = absolute | sec_of_day | day_of_week

Functions

new_absolute(Start::timespec() (see module ecal_time), Length::integer()) -> threshold()

Returns an absolute time threshold, starting at the given timespec, and lasting n seconds.

new_second_of_day(Start::integer(), Length::integer()) -> threshold()

This one repeats every day, from second 0 to 86399 inclusive.

new(Type::threshold_type(), Start::timespec() (see module ecal_time), Length::integer()) -> threshold()

Constructs a new time threshold.

intersect(Th1::threshold(), Th2::threshold()) -> false | threshold()

Intersects 2 thresholds, returning false if they dont intersect, or a new threshold if they do, with their intersection information.
