Functions

now() -> timespec()

Returns the number of seconds since year 1 up to now.

datetime_to_secs(DateTime::datetime() (see module calendar)) -> timespec()

Returns the number of seconds since year 1 that the given datetime represents.

month_of_time(Timespec::timespec()) -> {timespec(), month()}

Returns the timespec for the start, and the numeric month, of the current ongoing year according to the given timestamp, starting at 0 (january) and up to 11 (december).

beginning_of_month(Timespec::timespec()) -> timespec()

Returns the 00:00:00 hours of the 1st day of the current month according to the given timestamp.

end_of_month(Timespec::timespec()) -> timespec()

Returns the 23:59:59 hours of the last day of the current month according to the given timestamp.

day_of_month(Timespec::timespec()) -> integer()

Returns the day of the month, starting at 0.

plus_month(Timespec::timespec(), YearIsLeap::boolean(), Month::month()) -> timespec()

Adds the given month to timespec.

minus_month(Timespec::timespec(), YearIsLeap::boolean(), Month::month()) -> timespec()

Substracts the given month to timespec.

beginning_of_year(Timespec::timespec()) -> timespec()

Returns a timespec equal to the first second of the current ongoing year according to the given timespec (backwards in time).

end_of_year(Timespec::timespec()) -> timespec()

Returns a timespec equal to the last second of the current ongoing year according to the given timespec (forward in time).

day_of_week(Timespec::timespec()) -> day()

Returns the number of the current day of the week according to the timespec, starting at saturday(0) up to friday(6).

days_in(Length::integer()) -> integer()

Returns the number of complete days found in Length (given in seconds).

beginning_of_week(Timespec::timespec()) -> timespec()

Returns the 00:00:00hrs of the first day of the week (saturday).

end_of_week(Timespec::timespec()) -> timespec()

Returns the 23:59:59hrs of the last day of the week (friday).

year_of_time(Timespec::timespec()) -> integer()

Returns the number of the year for the given Timespec.

timespec_for_year(Year::integer()) -> timespec()

Returns the Jan 1st, 00:00:00 for the given year.

plus_years(Timespec::timespec(), Years::integer()) -> timespec()

Adds the given number of years to the specified time.

minus_years(Timespec::timespec(), Years::integer()) -> timespec()

Substracts the given number of years to the specified time.

inc_year(Year::integer(), Timespec::timespec()) -> timespec()

Increments a year, based on if Year is a leap year.

dec_year(Year::integer(), Timespec::timespec()) -> timespec()

Decrements a year, based on if Year is a leap year. This one will actually substract the PREVIOUS year TO the GIVEN year. So the timespec resulting from substracting a year to 1999, is 1999 minus the length of 1998, so, 1/1/1998.

plus_leap_year(Timespec::timespec()) -> timespec()

Adds a complete leap year to the given timespec.

minus_leap_year(Timespec::timespec()) -> timespec()

Substracts a complete leap year to the given timespec.

plus_year(Timespec::timespec()) -> timespec()

Adds a complete year to the given timespec.

minus_year(Timespec::timespec()) -> timespec()

Substracts a complete year to the given timespec.

day_of_time(Timespec::timespec()) -> integer()

Returns the number of complete days in the given timestamp.

second_of_day(Timespec::timespec()) -> integer()

Returns the current second of the current day of the given timestamp (0 - 86400).

minute_of_day(Timespec::timespec()) -> integer()

Returns the current minute of the current day of the given timestamp (0 - 1440).

hour_of_day(Timespec::timespec()) -> integer()

Returns the current hour of the current day of the given timestamp (0 - 23).

tomorrow(Timespec::timespec()) -> timespec()

Returns a timespec equal to the first second of the next day according to the given timespec (forward in time).

yesterday(Timespec::timespec()) -> timespec()

Returns a timespec equal to the first second of the previous day according to the given timespec (backwards in time).

beginning_of_day(Timespec::timespec()) -> timespec()

Returns a timespec equal to the first second of the current ongoing day according to the given timespec (backwards in time).

end_of_day(Timespec::timespec()) -> timespec()

Returns a timespec equal to the last second of the current ongoing day according to the given timespec (forward in time).

beginning_of_minute(Timespec::timespec()) -> timespec()

Returns a timespec equal to the first second of the current ongoing minute according to the given timespec (backwards in time).

end_of_minute(Timespec::timespec()) -> timespec()

Returns a timespec equal to the last second of the current ongoing minute according to the given timespec (forward in time).

beginning_of_hour(Timespec::timespec()) -> timespec()

Returns a timespec equal to the first second of the current ongoing hour according to the given timespec (backwards in time).

end_of_hour(Timespec::timespec()) -> timespec()

Returns a timespec equal to the last second of the current ongoing hour according to the given timespec (forward in time).

plus_weeks(Timespec::timespec(), Weeks::integer()) -> timespec()

minus_weeks(Timespec::timespec(), Weeks::integer()) -> timespec()

plus_days(Timespec::timespec(), Hours::integer()) -> timespec()

Adds the given number of days to the specified time.

minus_days(Timespec::timespec(), Hours::integer()) -> timespec()

Substracts the given number of days to the specified time.

plus_hours(Timespec::timespec(), Hours::integer()) -> timespec()

Adds the given number of hours to the specified time.

minus_hours(Timespec::timespec(), Hours::integer()) -> timespec()

minus_minutes(Timespec::timespec(), Minutes::integer()) -> timespec()

Adds the given number of seconds to the specified time.

minus_seconds(Timespec::timespec(), Seconds::integer()) -> timespec()

is_leapyear(Year::integer()) -> boolean()

Returns true if the given year is a leap year.
