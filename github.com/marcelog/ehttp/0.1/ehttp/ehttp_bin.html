Functions

lc(Bin::binary()) -> binary()

Lowercases all characters between A-Z found in the binary.

trim(Bin::binary()) -> binary()

Removes " " and "\t" from left and right of the binary.

trim_left(Bin::binary()) -> binary()

Removes " " and "\t" from the left of the binary.

trim_right(Bin::binary()) -> binary()

Removes " " and "\t" from the right of the binary.

split(Bin::binary()) -> [binary()]

Splits a binary by " " and "\t".

split_by_char(Bin::binary(), Char::binary(), StopAtFirst::boolean()) -> [binary()]

Splits a binary by the given char.

reverse(List::[term()]) -> [term()]

Reverses a list.

decode(Bin::binary()) -> integer()

Decodes an encoded character in the form %NN where N is an hex digit.

parse_key_equal_value(Bin::binary()) -> {binary(), binary()}

Given a binary in the form <<"key=value">>, will return {<<"key">>, <<"value">>}.

parse_key_values_by(Bin::binary(), Sep::binary()) -> variables() (see module ehttp_variable)

Given a binary with key/value variables, will return the variables found. When called with <<"a=b&c=d">> (and Sep=&) will return a variable set with both variables set.

encode(Char::binary()) -> binary()

Encodes a string so it can be sent by wire.
