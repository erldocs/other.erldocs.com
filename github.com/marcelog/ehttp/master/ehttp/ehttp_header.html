Types

header() = {header_key(), header_value()}

header_key() = ehttp_variable:variable_key()

header_value() = ehttp_variable:variable_value()

header_values() = [header_value()]

headers() = [header()]

Functions

new_header_list() -> headers()

Returns a new header list.

marshall(Headers::headers()) -> binary()

Serializes all headers present in list, ready to be sent by wire.

unmarshall(Header::binary()) -> {header()}

Parses binaries exactly like: "Key: Value" and returns a header, with its key lower-cased.

set(Headers::headers(), Key::header_key(), Value::header_value()) -> headers()

Sets a header value. If the header does not exist, it will be added. If the header already exists, the value will be replaced.

get(Headers::headers(), Key::header_key()) -> header_values() | notfound

Returns a header value. Will stop traversing the header list when found.

has(Headers::headers(), Key::header_key()) -> boolean()

true if the given header list contains the given header.

get_single(Headers::headers(), Key::header_value()) -> header_value()

Returns the first value found for the given key.
