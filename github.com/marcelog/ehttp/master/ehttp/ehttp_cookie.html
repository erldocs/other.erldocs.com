Types

cookie() =
            {{name, cookie_name()},
             {value, cookie_value()},
             {params, cookie_params()}}

cookie_name() = ehttp_variable:variable_key()

cookie_params() = ehttp_variable:variables()

cookie_value() = ehttp_variable:variable_value()

cookies() = ehttp_variable:variables()

Functions

new_cookies() -> cookies()

Returns a new cookie jar.

new_cookie(Name::cookie_name(), Value::cookie_value(), Params::cookie_params()) -> cookie()

Creates a new cookie.

new_cookie(Name::cookie_name(), Value::cookie_value()) -> cookie()

Creates a new cookie with empty params.

marshall(Cookie::cookie() | cookies(), HeaderName::header_name() (see module ehttp_header)) -> binary()

Marshalls a cookie.

unmarshall(Bin::binary()) -> cookie()

Unmarshalls a cookie from a binary.

set(Cookies::cookies(), Name::cookie_name(), Value::cookie_value()) -> cookies()

Sets a cookie value. If the cookie does not exist, it will be added. If the cookie already exists, the value will be replaced.

set(Cookies::cookies(), Cookie::cookie()) -> cookies()

Adds a cookie to the given cookie list.

get(Cookies::cookies(), Key::cookie_name()) -> cookie_value() | notfound

Returns a cookie value. Will stop traversing the cookie list when found.

has(Cookies::cookies(), Key::cookie_name()) -> boolean()

true if the given cookie list contains the given cookie.
