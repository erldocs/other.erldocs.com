ehttp_variable

This generically represents a variable and a set of variables.

This generically represents a variable and a set of variables. Pretty much like dict, but only with binaries for keys and values.

Types

variable() = {variable_key(), variable_value()}

variable_key() = binary()

variable_value() = binary()

variable_values() = [variable_value()]

variables() = [variable()]

Functions

new_variables() -> variables()

Creates a new set of variables.

set(Variables::variables(), Key::variable_key(), Value::variable_value() | variable_values()) -> variables()

Sets a variable value. If the variable does not exist, it will be added. If the variable already exists, the value will be replaced.

get(Variables::variables(), Key::variable_key()) -> variable_value() | variable_values() | notfound

Returns a variable value. Will stop traversing the variable list when found.

has(Variables::variables(), Key::variable_key()) -> boolean()

true if the given variable list contains the given variable.
