Functions

from_file(FileName::string()) -> {[string()], [binary()]}

Read in a bag-of-words dataset from the MXM project.

Returns a pair where the first component is a list of words, and the second component is a list of binaries, one for each track. Use parse_track to parse a binary track.

parse_track(Track::binary()) -> {binary(), binary(), [{integer(), integer()}]}

Parse a track from a binary on the form: track_id, mxm_track_id, <word idx>:<cnt>, <word idx>:<cnt>, ...

Return a tuple where the first component is track_id, second is mxm_track_id, and the third is a list of pairs: word index and count. Remember word index starts at 1 (not zero). See comment in the start of the bag-of-words dataset for more detailed information.
