poll(Socket, Events) -> ok | {error, Reason}

  • Events = [Event]
  • Event = pollin | pollout | pollerr

Poll for socket events. Events are received in the caller's mailbox as {zmq, Socket, REvents}. Where REvents is a list containing 'pollin' and/or 'pollout' or 'pollerr'.
