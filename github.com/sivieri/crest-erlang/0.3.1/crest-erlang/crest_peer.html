crest_peer

Server module of a CREST peer.

Server module of a CREST peer. It offers all the standard server start, stop and response methods, plus the two specific CREST operations: spawn (split into the installation and the execution parts) and remote.

Functions

start() -> ok

Start this peer

stop() -> ok

Stop this peer

spawn_install(Params::[{string(), any()}]) -> {reply, string(), dictionary()}

Install a new operation on this peer, using the given parameters.

spawn_exec(Key::[Key], Params::[{atom(), any()}]) -> {reply, {ok, any()}, dictionary()} | {reply, {error}, dictionary()}

Execute an already installed operation, specified by the given unique key.

remote(Params::[{string(), any()}]) -> {reply, {ok, any()}, dictionary()} | {reply, {error}, dictionary()}

Install and execute a new operation on this peer, and then delete it.

add_child(Key::string(), Pid::pid()) -> {noreply, dictionary()}

Add a new process to the internal server list, with the associated key.
