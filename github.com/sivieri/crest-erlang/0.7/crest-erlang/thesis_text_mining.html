thesis_text_mining

Utility functions for calculating text mining weights: word frequency, term frequency - inverse document frequency, cosine similarity.

Utility functions for calculating text mining weights: word frequency, term frequency - inverse document frequency, cosine similarity. This module is used for the thesis demo.

Functions

Get the word count as a dictionary from the specified file.

get_word_counts(Filename::string(), Dict::dictionary()) -> dictionary()

Get the word count as a dictionary from the specified file, using the specified dictionary to add words to.

print_dict(Dict::dictionary()) -> ok

Print the dictionary.

tf_idf(DictList::[{string(), dictionary()}]) -> [{string(), dictionary()}]

This function takes a list of dictionaries, each of them containing words with their frequency, and calculates the term frequency, the inverse document frequency and then it combines them into the tf-idf weight. It returns again a list of the same dictionaries as before, with the weight instead of the word count.

cosine_documents(ListOfDict::[{string(), dictionary()}]) -> [{string(), string(), float()}]

This function takes a list of dictionaries, each of them containing words with their frequency, and calculates the cosine similarity between all pairs of dictionaries (that are documents).
