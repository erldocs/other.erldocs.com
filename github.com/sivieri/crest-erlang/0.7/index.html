CREST application behaviour: it starts and stop the application.

crest_deps
Ensure that the relatively-installed dependencies are on the code loading path, and locate resources relative to this application's path.

crest_json
A module for destructuring JSON Erlang objects (coming from mochijson2:decode()) and selecting inner parts with a Javascript-like syntax.

crest_manager
Manager module: it invokes all the currently spawned processes, searching for some parameters, then returns all data to the caller in JSON format.
