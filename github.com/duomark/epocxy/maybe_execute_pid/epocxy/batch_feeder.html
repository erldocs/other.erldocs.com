batch_feeder

Batches are often used as a way to increase throughput or to handle a large amount of data in bite-size chunks.

Batches are often used as a way to increase throughput or to handle a large amount of data in bite-size chunks. The key features of this module are the ability to control the size and pace of the batches and to adapatively modify the amount of processing in each step. A common pattern is to pair a batch_feeder with a cxy_fount or cxy_ctl to incrementally process a large set of data using concurrency in a controlled way to avoid overloading the CPU and memory.
