Decodes a raw IPMB response according the standard format: rqSA, netFn/rqLUN, chk1, rsSA, rqSeq/rsLUN, cmd, completion, <data>, chk2 Calling this directly is useful for e.g. responses for bridged requests. Refer to chapter 6.13, BMC Message Bridging in the IPMI specification.
