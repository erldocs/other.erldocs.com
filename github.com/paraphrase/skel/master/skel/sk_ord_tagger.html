sk_ord_tagger

This module contains 'ord' skeleton tagger logic.

This module contains 'ord' skeleton tagger logic.

The 'ord' skeleton can reorder outputs from its inner skeletons such that they have the same order coming out the ord skeleton as they had going into it.

The tagger takes each input and adds information as to its place in the input stream of the inner skeleton.

Functions

start(NextPid::pid()) -> eos

Starts the tagger, labelling each input so that the order of all inputs is recorded.
