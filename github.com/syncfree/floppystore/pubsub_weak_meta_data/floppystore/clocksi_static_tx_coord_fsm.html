clocksi_static_tx_coord_fsm

The coordinator for a given Clock SI static transaction.

The coordinator for a given Clock SI static transaction. It handles the state of the tx and executes the operations sequentially by sending each operation to the responsible clockSI_vnode of the involved key. When a tx is finalized (committed or aborted), the fsm also finishes.
