make_key(OrgName::binary(), BatchSize::integer(), Start::integer(), Ids::[binary()], ReqPath::string(), Paths::[{binary(), [binary()]}]) -> {binary(), binary()}

Return the cache key for specified inputs that describe a search request. The key combines request attributes and solr results to identify unique search queries. The Ids are those returned from Solr. ReqPath should be the raw HTTP request path including query parameters. Path is the list of partial search paths or empty list. This function returns a tuple {OrgName, Digest} so that get/2 and put/2 have easy access to the org name to implement black lists for caching.
