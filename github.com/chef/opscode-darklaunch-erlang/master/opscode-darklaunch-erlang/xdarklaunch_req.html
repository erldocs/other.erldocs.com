parse_header(GetHeader::header_fun()) -> #xdarklaunch{orgname=undefined | string(), raw_header=undefined | string(), values=dict()}

Fetch and parse the darklaunch header

GetHeader is a fun taking the header name and returing a binary; e.g. chef_wm_util:get_header_fun

Expects a header in the format KEY1=VALUE1;KEY2=VALUE2... KEY is expected to be alphanumeric plus _ VALUE can be any non-whitespace string, but currently we only expect 0,1
