Sets the expectation that Function of Module will be called during a test with Args. Args should be a fun predicate that will return true or false whether or not the argument list matches. The argument list of the function is passed in as a tuple.

Ret is either a value to return or a fun of arity 2 to be evaluated in response to a proxied call. The first argument is the actual args from the call, the second is the call count starting with 1.
