e_fe_cluster

Module responsible for managing the frontend nodes of the Erlang Web cluster.

Module responsible for managing the frontend nodes of the Erlang Web cluster. All the functions should be called on the frontend server.

Functions

write_file(Filename::string(), Content::binary()) -> ok | {error, Reason::atom()}

Saves the passed Content under the Filename on the frontend node. The previous version of the file is erased.

write_file_chunk(Filename::string(), Chunk::binary()) -> ok

Appends the Content to the end of the file specified by Filename. If file does not exist, it is created.
