generator_request_initiator

This module generates requests.

This module generates requests. In charge of creating generator slaves which actually send the requests out. Each initiator is responsible for one request chain. Input: list of tuples in the following format: {Module, Function, Arguments}. An example of a tuple is {generator_slave,start,[RequestTuple1,RequestTuple2,RequestTuple3..,RequestTupleN]}.
