rebar_relx_plugin

This provides a trivial way to integrate the relx release builder into rebar.

This provides a trivial way to integrate the relx release builder into rebar.

It has two optional configuration values. Those are 1) relx_libdirs 2) relx_output

These have sane defaults but can be specified by the user.
