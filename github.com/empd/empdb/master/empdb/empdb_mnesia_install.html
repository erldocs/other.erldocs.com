empdb_mnesia_install

This module is called from EMPD.erl, and provides the callbacks devoted to database verification and installation.

This module is called from EMPD.erl, and provides the callbacks devoted to database verification and installation. It also makes sure all defaults are inserted into the database too.

install(Nodes::nonempty_nodelist(), Options::[tuple()]) -> possible_failure(ok)

Installs the EMP Mnesia backend. Uses the options provided by empdb application.

verify(Nodes::nonempty_nodelist(), Options::[tuple()]) -> ok | {install, nonempty_nodelist()} | {error, continue | shutdown, Reason::any()}

Verify that your database is installed on at least one of the nodes and can be communicated to by all of them. This verification can take some time, as this is only called once on EMPDB startup.
