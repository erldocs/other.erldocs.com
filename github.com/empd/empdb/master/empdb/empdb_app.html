empdb_app

The front end OTP Application for the EMP Database application.

The front end OTP Application for the EMP Database application. EMPDB is the state store and distributed cache for the Extensible Monitoring Platform.

EMPDB is completely extensible and can be swapped out for other storage options. However, the default uses MNesia, the erlang native database solution and provides fault-tolerance through replication. See the EMPDB module for more information on how to swap out other backends.

Functions

start(StartType, StartArgs) -> term()

Starts the EMP database application after verifying its installation.

prep_stop(State) -> term()

Prepares to shutdown.

stop(State) -> term()

Stops the EMPDB application
