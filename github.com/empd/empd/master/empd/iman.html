iman

The Interface manager is a supervisor for all the interface handlers.

The Interface manager is a supervisor for all the interface handlers. Each handler can monitor a different type of connection protocol, so we can have a TCPHandler, UDPHandler or even your own internal protocol you define to talk to EMP. If ever any of them go down, they will get restarted.

Functions

start_link() -> term()

Start the IMan server and all EMP interfaces.
