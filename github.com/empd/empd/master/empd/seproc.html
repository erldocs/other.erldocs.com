seproc

When EMP starts up, the root eman creates N System Plugin Event Processors (seproc), where N is the number of users.

When EMP starts up, the root eman creates N System Plugin Event Processors (seproc), where N is the number of users. Each seproc is responsible for sending its user's eman an event for every one it recieves where there is a user subscription for.
