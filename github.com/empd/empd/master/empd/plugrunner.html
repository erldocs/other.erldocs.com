plugrunner

Plugin runner is a server that communicates with a plugin on behalf of Emp.

Plugin runner is a server that communicates with a plugin on behalf of Emp. It has functions for the running of commands and setting the variables of a given plugin. This makes it easy for uman to talk to plugins.
