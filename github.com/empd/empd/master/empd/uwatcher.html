uwatcher

User watcher is a supervisor that generates and runs all user nodes as their own processes.

User watcher is a supervisor that generates and runs all user nodes as their own processes. This will monitor if any of the user's crash, if this happens then it will restart them. This is to help improve individual reliability of the user's data/events as well as protect other users from downtime another user may cause.

Functions

start_link() -> {ok, pid()} | {error, any()}

Starts the User Watcher.
