Promises.

Promises. API inspired by Clojure: promise -> new deliver -> set deref -> get realized? -> is_set

Types

Functions

eval(Thunk::function(), Timeout::timeout()) -> promise()

Return a promise for the result of Thunk, which gets Timeout ms to run.

new(Timeout::timeout()) -> promise()

Return a fresh promise. The promise will GC itself after Timeout milliseconds without any of the API functions being called on it.

is_promise(P::term()) -> boolean()

Return true iff X is a promise.

is_set(P::promise()) -> boolean()

Return true iff Promise has a value, otherwise false.

get(P::promise()) -> maybe(term(), term())

Get Promise's value. Blocks if Promise hasn't been set.

get(P::promise(), T::timeout()) -> maybe(term(), term())

Get Promise's value. Blocks for T ms if Promise hasn't been set.

set(P::promise(), Val::term()) -> maybe(term(), term())

Set Promise's value to Val. May only be called once.
