txn(Txn::txn_fun(), Reads::txn_decl(), Writes::txn_decl()) -> txn_ret()

Run a transaction (a generated flow analogous to Fun). Reads/Writes describe the contexts which F consumes/produces. These are similar to service contracts, but store annotations are implicit, and there aren't any namespace restrictions.
