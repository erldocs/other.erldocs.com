tulib_sh

Safe and convenient procedures for interacting with Unix.

Safe and convenient procedures for interacting with Unix. All functions assert success internally and aim to return something that's useful in a pipeline.

Types

maybe(A, B) = {ok, A} | {error, B}

Evaluate Cmd via os:cmd/1. Return Cmd's output or Cmd's exit code along with it's output, if the exit code is =/= 0.

mkdir_p(Dir::string()) -> string()

mktemp() -> string()

Create a file with a name that is probably unique. Return the filename. The exclusive option should be supplied when opening this filename. The two optional arguments are Prefix: the stem of the returned filename (defaults to mktemp) Dir: the directory in which to create the file (defaults to /tmp)

mktemp(Prefix) -> term()

mktemp(Prefix, Dir) -> term()

mktemp_u() -> string()

Unsafe mktemp. Returns a filename which is probably unique, but does not actually create the file.

mktemp_u(Prefix) -> term()

mktemp_u(Prefix, Dir) -> term()

mktemp_d() -> string()

Create a unique directory and return its name.

mktemp_d(Prefix, Dir) -> term()
