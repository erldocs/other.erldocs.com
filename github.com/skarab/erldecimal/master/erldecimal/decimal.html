Decimal operations based on the General Decimal Arithmetic Specification:.

Decimal operations based on the General Decimal Arithmetic Specification:

http://www2.hursley.ibm.com/decimal/decarith.html

and also IEEE standard 854-1987:

http://www.cs.berkeley.edu/~ejr/projects/754/private/drafts/854-1987/dir.html

This should support arithmetic using basic rules, avoiding a good deal of the tricky representation issues associated with binary floating point. The obvious use is for representing money where 1.00 rem 0.1 gives 0.09999999999999995 instead of the expected 0.00 when dealing with normal decimal operations.
