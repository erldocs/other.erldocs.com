linc_us4_port

Module to represent OpenFlow port.

Module to represent OpenFlow port. It abstracts out underlying logic of either hardware network stack or virtual TAP stack. It provides Open Flow ports represented as gen_server processes with port configuration and statistics according to OpenFlow specification. It allows to create and attach queues to given ports and supports queue statistics as well. OpenFlow ports can be programatically started and stopped by utilizing API provided by this module.
