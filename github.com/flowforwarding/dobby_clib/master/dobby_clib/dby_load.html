dby_load

Load data from files.

Load data from files. Data files are Erlang terms.

Functions

load(Publisher::binary(), Filename::string()) -> ok | {error, term()}

Equivalent to load(Publisher, Filename, infinity).

load(Publisher::binary(), Filename::string(), BatchSize::inifinity | non_neg_integer()) -> ok | {error, term()}

load/3 reads Filename and calls dby:publish/3 with the contents of the file using Publisher as te publisher. The maximum batch size is BatchSize. If BatchSize is infinity the contents of the file are not batched.

dirload(Publisher::binary(), Dirname::string()) -> ok | {error, term()}

Equivalent to dirload(Publisher, Dirname, infinity).

dirload(Publisher::binary(), Dirname::string(), BatchSize::infinity | non_neg_integer()) -> ok | {error, term()}

dirload/3 finds all the files in Dirname with the .dobby extension and calls load/2 on each one. Processing aborts if there is an error. Each file is loaded as a separate transaction, so an abort will result in a partial load of the data. The maximum batch size is BatchSize. If BatchSize is infinity the contents of the file are not batched.
