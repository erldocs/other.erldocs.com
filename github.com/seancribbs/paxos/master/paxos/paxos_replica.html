paxos_replica

The Replica process and behaviour.

The Replica process and behaviour. To implement a Replica, create a module that implements the perform(Operation, State) function and start the replica like so:

paxos_replica:start_link(Module, InitialState, LeaderPids)

where Module is the module implementing this behaviour, InitialState is the state upon which the callback will operate, and LeaderPids is a list of pids which are the leaders.

When the Replica commits an operation, it will call Module:perform/2 to retrieve the modified state and result of the operation.
