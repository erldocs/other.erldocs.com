fn

Functions

arity(Fun::function()) -> non_neg_integer()

Return the arity of the function argument

identity(A) -> A

Return the input

constantly(Val::any()) -> function()

Return a zero arity function that returns Val.

constantly(Arity::non_neg_integer(), Val::any()) -> function()

Return a function that ignores it's arguments and always returns Val. The returned function has arity Arity.

complement(Fun::function()) -> function()

Return a function that calls Fun but returns the opposite boolean value. Works with functions of any arity.

comp(Funs::[function()]) -> function()

Compose a list of functions together, right-to-left. Returns a function with arity equal to the last function.

comp(FunA::function(), FunB::function()) -> function()

Compose two functions FunA and FunB together. Returns a function that has the same arity as FunB. The resulting function calls FunB and then FunA with the result of calling FunB.

partial(Fun::function(), Arg::term()) -> function()

Return a function with one-less arity than Fun with Arg implicitly as the first argument.

flip(Fun::(A, B) -> C) -> (B, A) -> C

Flip the arguments of a binary function.
