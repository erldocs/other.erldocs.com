flexihash

Functions

replicas(Replicas::non_neg_integer()) -> ok

Change the number of replicas created when adding targets to a table. The initial value is 64. Implemented via flexihash_server.

newtable(TableName::atom()) -> ok

Create a new empty table. If the name refers to an existing table it will be replaced. Implemented via flexihash_server.

deltable(TableName::atom()) -> ok

Delete a table. Succeeds quietly if the table does not exist. Implemented via flexihash_server.

addtarget(TableName::atom(), Target::string() | binary()) -> ok | {error, notable}

Add one Target (with current number of replicas) to the specified table, which must exist. Implemented via flexihash_server.

addtargetlist(TableName::atom(), TargetList::[string() | binary()]) -> ok | {error, notable}

Equivalent to invoking addtarget multiple times. Implemented via flexihash_server.

deltarget(TableName::atom(), Target::string() | binary()) -> ok | {error, notable}

Delete all replicas of the specified target from the named table, which must exist. Succeeds quietly if the target was not present. Implemented via flexihash_server.

lookup(TableName::atom(), Value::string() | binary()) -> ok | {error, notable} | {error, empty}

Determine which target stored in the named table, which must exist, is appropriate to the requested Value. Error if the table is empty. Implemented via flexihash_server.

start() -> ok | {error, Reason::term()}

Start the application.
