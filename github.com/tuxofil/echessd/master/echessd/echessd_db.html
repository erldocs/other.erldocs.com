init() -> ok

Create mnesia database from scratch. All previous data if will be wiped off. Normally the function is called only once before the first server start.

wait() -> ok

Wait until all database tables become available or fail if a timeout of 10 seconds elapsed. Called at the server start from the application supervisor init/0.
