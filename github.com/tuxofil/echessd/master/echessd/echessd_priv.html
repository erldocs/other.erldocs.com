echessd_priv

Proxy process for 'priv' directory contents.

Proxy process for 'priv' directory contents.

The process give access to the Echessd files stored in so called "priv" directory. When Echessd is running as normal Erlang application within Erlang node, all read_file_info/1 and read_file/1 calls are mapped to the file:read_file_info/1 and file:read_file/1 calls respectively. When Echessd is started as packaged Erlang script (escript), contents of the archive are deflated and stored in ETS table, so read_file_info/1 and read_file/1 calls will fetch deflated data directly from the ETS.
