webdrv_wire

WebDriver Wire Protocol.

WebDriver Wire Protocol

The individual functions are not documented. They have type annotations, for more information on the particular functions, we refer to (The WebDriver Wire Protocol).

The errors reported are slightly more advanced that the basic error reporting prescribed in the protocol description; see the type request_error().

Types

capability() = #capability{} | null | jsonobj()

frame_id() = jsonstr() | number() | null

json() = jsonobj()
               | jsonlist()
               | jsonnum()
               | jsonstr()
               | true
               | false
               | null

jsonkey() = string()

jsonlist() = [json()]

jsonnum() = integer() | float()

jsonobj() = {obj, [{jsonkey(), json()}]}

jsonstr() = binary()

request_error() = {html_error, string()}
                        | {json_error, string()}
                        | {cmd_error, json()}
                        | {wire_error,
                           {atom(), string()},
                           session_id()}
                        | {type_error, atom(), term()}

request_ok() = {ok, session_id(), jsonobj()}

request_res() = request_ok() | request_error()

session_id() = string() | null
