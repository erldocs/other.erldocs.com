Functions

start(StartType, StartArgs) -> {ok, pid()} | {error, startlink_err()}

  • StartType = term()
  • StartArgs = term()

Called when the OTP system wants to start the application, it must perform the actual startup and return the process id of the root supervisor. We can do any other startup tasks here as well, such as reading a configuration file, initialize ETS tables a.s.o.

stop(State) -> ok

  • State = term()

Called when the OTO system wants to shutdown the application, nothing special to do so we ignore the param and return ok.
