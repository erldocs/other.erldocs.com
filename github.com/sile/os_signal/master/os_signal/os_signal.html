Functions

kill(OsPid::os_pid(), Signal::signal()) -> ok | {error, Reason::posix() (see module file)}

Send signal to an OS process.

Example: 

  > Port = open_port({spawn_executable, "/bin/sleep"}, [{args, ["100"]}, exit_status]).
  #Port<0.595>
  > {os_pid, OsPid} = erlang:port_info(Port, os_pid).
  > os_signal:kill(OsPid, sigkill).
  > flush().
  Shell got {#Port<0.595>,{exit_status,137}}
  ok

See the man page of kill(2) for futher details.
