Stream Processing Functions.

Stream Processing Functions.

This module contains functions for binary string (called stream) processing.

Functions

len(Stream) -> Length

Returns the number of characters in the stream.

Types:

Stream = binary()
Length = integer()

equal(Stream::Stream1, Stream::Stream2) -> bool()

Tests whether two streams are equal. Returns true if they are, otherwise false.

Types:

Stream1 = Stream2 = binary()

concat(Stream1, Stream2) -> Stream3

Concatenates two streams to form a new stream. Returns the new stream.

Types:

Stream1 = Stream2 = Stream3 = binary()

chr(Stream, Character) -> Index

Returns the index of the first occurrence of Character in Stream. 0 is returned if Character does not occur.

Types:

Stream = binary()
Character = char()
Index = integer()

rchr(Stream, Character) -> Index

Returns the index of the last occurrence of Character in Stream. 0 is returned if Character does not occur.

Types:

Stream = binary()
Character = char()
Index = integer()

str(Stream, SubStream) -> Index

Returns the position where the first occurrence of SubStream begins in Stream. 0 is returned if Substream does not exist in Stream.

Types:

Stream = SubStream = binary()
Index = integer()

For example: 

  > stream:str(<<" Hello Hello World World ">>, <<"Hello World">>).
  8

rstr(Stream, SubStream) -> Index

Returns the position where the last occurrence of SubStream begins in Stream. 0 is returned if SubStream does not exist in Stream.

Types:

Stream = SubStream = binary()
Index = integer()

For example: 

  > stream:rstr(<<" Hello Hello World World ">>, <<"World">>).
  20

index(Stream, SubStream) -> Index

This function is deprecated: See str/2 for details.

Equivalent to str(Stream, SubStream).

span(Stream, Chars) -> Length

Returns the length of the maximum initial segment of Stream, which consists entirely of bytes from Chars.

Types:

Stream = Chars = binary()
Length = integer()

For example: 

  > stream:span(<<"\t    abcdef">>, <<" \t">>).
  5

cspan(Stream, Chars) -> Length

Returns the length of the maximum initial segment of Stream, which consists entirely of bytes not from Chars.

Types:

Stream = Chars = binary()
Length = integer()

For example: 

  > stream:cspan(<<"\t    abcdef">>, <<" \t">>).
  0

substr(Stream, Start) -> SubStream

See substr/3 for details.

substr(Stream, Start, Length) -> SubStream

Returns a substream of Stream, starting at the position Start, and ending at the end of the stream or at length Length.

Types:

Stream = SubStream = binary()
Start = Length = integer()

For example: 

  > stream:substr(<<"Hello World">>, 4, 5).

tokens(Stream, SeparatorList) -> Tokens

Returns a list of tokens in Stream, separated by the characters in SeparatorList.

Types:

Stream = Seps = binary()
Tockes = [binary()]

For example: 

  > stream:tokens(<<"abc defxxghix jkl">>, <<"x ">>).
  [<<"abc">>, <<"def">>, <<"ghi">>, <<"jkl">>]

strip(Stream) -> Stripped

See strip/3 for details.

strip(Stream, X2::Direction) -> Stripped

See strip/3 for details.

strip(S::Stream, X2::Direction, Char::Character) -> Stripped

Returns a stream, where leading and/or trailing blanks or a number of Character have been removed. Direction can be left, right, or both and indicates from which direction blanks are to be removed. The function strip/1 is equivalent to strip(String, both).

Types:

Stream = Stripped = binary()
Direction = left | right | both
Character = char()

For example: 

  > stream:strip(<<"...Hello.....">>, both, $.).
  <<"Hello">>

left(Stream, Number) -> Left

See left/3 for details.

left(Stream, Number, Character) -> Left

Returns the Stream with the length adjusted in accordance with Number. The left margin is fixed. If the length(Stream) < Number, Stream is padded with blanks or Characters.

Types:

Stream = Left = binary()
Number = integer()
Character = char()

For example: 

  > stream:left(<<"Hello">>, 10, $.).
  <<"Hello.....">>

right(Stream, Number, Character) -> Right

Returns the Stream with the length adjusted in accordance with Number. The right margin is fixed. If the length(Stream) < Number, Stream is padded with blanks or Characters.

Types:

Stream = Right = binary()
Number = integer()
Character = char()

For example: 

  > stream:right(<<"Hello">>, 10, $.).
  <<".....Hello">>
