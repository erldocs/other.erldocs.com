open(Table::any(), Options::[Option]) -> {ok, Table} | {error, Reason}

  • Option = {base_path, Path} | {max_file_bytes, Bytes} | {max_total_bytes, Bytes} | {n_partitions, NPartitions}

Open the flatula table named Table. More than one process may open a given table, and a single process may open it more than once; the table is not closed until a matching close() is called for each open().

The following options are mandatory:

base_path: the base path for files in the table, of the form /PATH/BASENAME. flatula may create any files of the form /PATH/BASENAME.EXTENSION.

max_file_bytes: maximum bytes in one table file. When an entry is added that exceeds this limit, a new file is created.

max_total_bytes: maximum total bytes in the table. When an entry is added that exceeds this limit, the oldest file in the table is removed (along with all the entries in it).

The following options are optional:

n_partitions: number of server processes (default 1). Using more than one partition can increase throughput, but is usually unnecessary.

close(Table::any()) -> ok

Close the table Table. More than one process may open a given table, and a single process may open it more than once; the table is not closed until a matching close() is called for each open().
