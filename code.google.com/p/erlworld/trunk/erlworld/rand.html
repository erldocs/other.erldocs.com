rand

A random number generator server.

A random number generator server.

You create a new random server by calling the new function and then pass this into the random function when generating numbers.

This is only meant as a cheap and cheerful random number module. For anything more sophisticated use the random module supplied in the Erlang OTP.

Functions

new() -> Server::pid()

Spawns a new random number server and returns it's process id.

random(Rand::pid(), Max::number()) -> RandomNum::number()

Returns a number from 0 to Max.

random(Rand::pid(), Min::number(), Max::number()) -> RandomNum::number()

Returns a number from Min to Max.

destroy(Rand::pid()) -> true

Destroys the given random number generator process. This is the same as just killing it yourself using the exit function.
