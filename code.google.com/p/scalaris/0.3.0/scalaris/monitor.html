Functions

proc_init(Process::atom()) -> tableid() (see module pdb)

Initialises a process' monitor DB. Needs to be called prior to the other proc_* methods! Beware: there is only one monitor table available per process if pdb uses erlang:put/get!

proc_set_value(Table::tableid() (see module pdb), Key::key(), Value::term()) -> ok

Sets the value at Key.

proc_get_value(Table::tableid() (see module pdb), Key::key()) -> term() | undefined

Gets the value stored at Key.

proc_inc_value(Table::tableid() (see module pdb), Key::key()) -> ok

Increases the value stored at Key by 1. If there is no such value, 1 is stored.

proc_inc_value(Table::tableid() (see module pdb), Key::key(), Inc::number()) -> ok | unsupported

Increases the value stored at Key by Inc. If there is no such value, Inc is stored. Note: Inc may be negative in order to decrease values. If the stored value is no number, unsupported is returned and nothing is changed.

proc_report_to_my_monitor(Table::tableid() (see module pdb)) -> ok

Reports all collected Key/Value pairs to the process' monitor process and resets the values and the list of stored keys.
