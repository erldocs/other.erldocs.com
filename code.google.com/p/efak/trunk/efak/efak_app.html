Application module of the Erlang Financial Application Kit.

Application module of the Erlang Financial Application Kit.

The application is configured via an XML file containing <isopackagers>, <interfaces> and <lines> container tags.

The XML configuration file is handled by the config_server process that is present on each node of the efak cluster. Default configuration file is named efakconf.xml, to specify a different configuration file you can change the env parameter config_server in the efak.app file.
