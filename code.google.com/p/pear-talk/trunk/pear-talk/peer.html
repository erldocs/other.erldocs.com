Functions

start(G::integer()) -> list()

start(G) Starts the client and creates an empty ets called friends that is the default friend list, if G is set to 1 then the gui is activated.

send(Receiver_username::string(), Mode::atom(), Obj::any()) -> term()

send(Receiver, Mode, Obj) sends Mode and Obj to Receiver if Receiver is in the friend list, Mode specifies for Receiver what kind of data that is beeing sent.
When Mode is set to file, friendlist, ping or pong, Receiver will neglect Data and process Obj that will be supposed to contain respectively a file a friend list or an empty list in the case of a ping or a pong.

read_address(Address::string()) -> any()

read_address(Address) reads Address and if Address is a domain it returns Address if Address is an ip-address of the type "xx.yy.zz.ww" it returns a tuple of the type {xx,yy,zz,ww}

friends() -> any()

friends() shows the friend list.

sendFr(Receiver_username::string(), Receiver_address::term(), Receiver_listen_port::integer()) -> any()

sendFr(Receiver_username, Friend_public_ip, Friend_listen_port) Sends a friend request to your friend.

shut_down() -> term()

shut_down() stops the client

host_info() -> term()

host_info() gives information about your computer

get_status() -> term()

get_status() gives information about your status

my(Status_element::atom()) -> term()

my(Status_element) returns the value of Status_element

mess(Receiver_username::string(), String::string()) -> term()

mess(Receiver , String) sends String to Receiver if Receiver is in the friend list

ping(Receiver::string()) -> term()

ping(Receiver) pings Receiver if Receiver is in the friend list

send_file(Receiver::string(), Path::string(), Name::string()) -> term()

send_file(Receiver, Path, Name) sends the file Path ++ Name to Receiver if Receiver is in the friend list

pingon() -> term()

pingon() allows you to see who pings you.

pingoff() -> term()

pingoff() disallows you to see who pings you.
