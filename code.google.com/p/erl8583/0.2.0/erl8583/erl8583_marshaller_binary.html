erl8583_marshaller_binary

This module provides functions to marshal an iso8583message() into a list of bytes, to unmarshal a list of bytes into an iso8583message() and to marshal/unmarshal the MTI, bitmap and fields of an ISO 8583 message.

This module provides functions to marshal an iso8583message() into a list of bytes, to unmarshal a list of bytes into an iso8583message() and to marshal/unmarshal the MTI, bitmap and fields of an ISO 8583 message.

When functions are used to unmarshal the MTI, bitmap or a field of a message, the function assumes that the component to be unmarshalled is at the start of the list. The returned value is a 2-tuple containing the unmarshalled value and the remainder of the list that needs to be unmarshalled.

Marshals an iso8583message() to a list of bytes.

unmarshal(Marshalled::[byte()]) -> iso8583message()

Unmarshals a list of bytes to an iso8583message().

marshal_bitmap(FieldIds::[integer()]) -> [byte()]

Constructs a list of bytes representation of the bitmap for an iso8583message().

unmarshal_bitmap(BinaryMessage::[byte()]) -> {[integer()], [byte()]}

Extracts a list of field IDs from a list of bytes representation of an ISO 8583 message. The result is returned as a 2-tuple: a list of field IDs and the remainder of the message excluding the bit map.

marshal_field(FieldId::integer(), FieldValue::iso8583field_value(), EncodingRules::module()) -> [byte()]

Marshals a field value into a byte list using a specified encoding rules module.

unmarshal_field(FieldId::integer(), BinaryFields::[byte()], EncodingRules::module()) -> {iso8583field_value(), [byte()]}

Extracts a field value from the start of a byte list. The field value and the rest of the unmarshalled byte list is returned as a 2-tuple. A module that specifies how the field is encoded must be passed as an argument.

marshal_mti(Mti::string()) -> [byte()]

Marshals the MTI into a byte list.

unmarshal_mti(Marshalled::[byte()]) -> {string(), [byte()]}

Extracts the MTI from the start of a byte list. The MTI and the rest of the unmarshalled list is returned as a 2-tuple.
