erl8583_convert
This module provides a number of functions for converting between representations of data.

erl8583_fields
This module provides a function to get how a field is encoded according to the 1987 version of the ISO 8583 specification.

erl8583_marshaller_ascii
This module marshals an iso8583message() into an ASCII string or unmarshals an ASCII string into an iso8583message().

erl8583_marshaller_ascii_field
This module marshals an iso8583message() field into an ASCII string or unmarshals an ASCII string into an iso8583message() field.

erl8583_marshaller_binary
This module marshalls an iso8583message() into a binary.

erl8583_marshaller_binary_field
This module marshals a field of an iso8583message() into a binary representation or unmarshals a binary value into a field of an iso8583message().

erl8583_marshaller_xml
This module marshals an iso8583message() into an XML element and can unmarshal an XML element into an iso8583message().

erl8583_marshaller_xml_field
This module marshals an iso8583message() field into an XML element or unmarshals an XML element into an iso8583message() field.

erl8583_message
Functions for constructing and reading ISO 8583 messages.
