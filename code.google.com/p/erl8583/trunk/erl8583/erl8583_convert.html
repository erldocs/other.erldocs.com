erl8583_convert

This module provides a number of functions for converting between representations of data common in ISO 8583.

This module provides a number of functions for converting between representations of data common in ISO 8583. Particular data representations are ASCII and EBCDIC strings, BCD encoding and the encoding used for track 2 data.
