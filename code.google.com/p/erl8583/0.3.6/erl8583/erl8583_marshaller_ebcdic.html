erl8583_marshaller_ebcdic

This module provides functions to marshal an iso8583message() into an EBCDIC string, to unmarshal an EBCDIC string into an iso8583message().

This module provides functions to marshal an iso8583message() into an EBCDIC string, to unmarshal an EBCDIC string into an iso8583message(). There are also functions to marshal/unmarshal the MTI, bitmap and fields of an ISO 8583 message.

The functions to unmarshal the MTI, bitmap or a field of a message assume that the component to be unmarshalled is at the start of the string. The returned value is a 2-tuple containing the unmarshalled value and the remainder of the string that needs to be unmarshalled.

Functions

marshal(Message::iso8583message()) -> [byte()]

Constructs an EBCDIC string representation of an iso8583message().

unmarshal(Marshalled::[byte()]) -> iso8583message()

Constructs an iso8583message() from an EBCDIC string marshalling of the message.

marshal_field(FieldId::integer(), FieldValue::iso8583field_value(), EncodingRules::module()) -> [byte()]

Marshals a field value into an EBCDIC string using a specified encoding rules module.

unmarshal_field(FieldId::integer(), Marshalled::[byte()], EncodingRules::module()) -> {iso8583field_value(), [byte()]}

Extracts a field value from the start of an EBCDIC string. The field value and the rest of the unmarshalled string is returned as a 2-tuple. A module that specifies how the field is encoded must be passed as an argument.

marshal_mti(Mti::string()) -> [byte()]

Marshals the MTI into an EBCDIC string.

unmarshal_mti(Marshalled::[byte()]) -> {string(), [byte()]}

Extracts the MTI from the start of an EBCDIC string. The MTI and the rest of the unmarshalled string is returned as a 2-tuple.

marshal_bitmap(Message::[integer()]) -> [byte()]

Constructs an EBCDIC string representation of the bitmap for an iso8583message().

unmarshal_bitmap(Marshalled::[byte()]) -> {[integer()], string()}

Extracts a list of field IDs from an EBCDIC string representation of an ISO 8583 message. The result is returned as a 2-tuple of the field IDs and the remainder of the the message (encoding the field values but not the bit map).

marshal_end(Message::iso8583message(), Marshalled::[byte()]) -> [byte()]

Completes the marshalling of a message and returns the marshalled form.

unmarshal_end(Message::iso8583message(), Marshalled::[byte()]) -> iso8583message()

Completes the unmarshalling of a message and returns the message.
