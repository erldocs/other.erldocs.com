erl8583_marshaller_xml

This module marshals an iso8583message() into an XML document and can unmarshal an XML document into an iso8583message().

This module marshals an iso8583message() into an XML document and can unmarshal an XML document into an iso8583message(). This module also exposes functions for explicitly marshalling/unmarshalling the MTI, bitmap and fields of a message. This is to conform to the design of other marshallers and the contract required by the erl8583_marshaller.

The structure of the XML messages is intended to be compatible with that used by jPOS.

marshal_end(Message::iso8583message(), Marshalled::string()) -> string()

Wraps XML elements in an <isomsg> XML element and returns the resultant XML document. The attributes of the original message are attributes of the <isomsg> XML element.

unmarshal_end(Message::iso8583message(), Marshalled::string()) -> iso8583message()

Finishes the unmarshalling of a message and returns the message.

marshal_init(Message::iso8583message()) -> {string(), iso8583message()}

Starts the marshalling of a message and returns the initial marshalled data and message as a 2-tuple.

unmarshal_init(Message::string(), Marshalled::string()) -> {iso8583message(), string()}

Creates an ISO 8583 message by extracting the attributes of an <isomsg> XML element and returns the message and the XML document as a 2-tuple. No fields of the message are populated by this function; only the attributes of the message are set.

marshal_bitmap(Message::[integer()]) -> string()

Returns an empty string.

unmarshal_bitmap(Marshalled::string()) -> {[integer()], string()}

Extracts a list of field IDs from an XML representation of an ISO 8583 message. The result is returned as a 2-tuple of the field IDs and the XML document.

marshal_mti(Mti::string()) -> string()

Marshals the MTI into an XML <field> element.

unmarshal_mti(Marshalled::string()) -> {string(), string()}

Extracts the MTI from an XML document. The MTI and the XML document are returned as a 2-tuple.
